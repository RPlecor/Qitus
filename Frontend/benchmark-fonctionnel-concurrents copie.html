En resume

Pennylane couvre 10/10 etapes. C'est la reference du marche, pas le concurrent direct.

Indy couvre 9/10 etapes. C'est le concurrent le plus proche par la cible (freelances + societes a l'IS). Point faible : opacite IA, positionnement ambigu sur l'EC.

Tiime ne fait que de la pre-comptabilite (3/10). Son perimetre s'arrete avant les ecritures.

Paperasse couvre 2-3/10 solidement, 5/10 partiellement. Differenciateurs : transparence des regles, architecture low-cost, positionnement EC-first. Les 3 livrables Phase 8.5 (TVA, liasse, partage EC) sont non negociables pour la beta.

Ce qui differencie Paperasse

Categorisation : boite noire vs triple couche

Critere Pennylane Indy Paperasse
Approche IA + regles (details non publics) IA (details non publics) Regles deterministes ~85%, IA residuelle ~15%
Transparence Faible boite noire Faible boite noire Forte regles auditables, IA tracee
Apprentissage Corrections utilisateur Corrections utilisateur CorrectionRules + VendorMapping
Cout par transaction Eleve (IA sur chaque tx) Eleve (IA sur chaque tx) Faible (~15% seulement via IA)
Differenciateur cle : quand Pennylane ou Indy categorisent une transaction, l'utilisateur ne sait pas pourquoi. Quand Paperasse le fait via une regle deterministe, le "pourquoi" est tracable. C'est un argument de confiance fort pour le segment expert-comptable.

Gap analysis

Les 3 trous critiques de Paperasse

Par ordre de priorite, aligne avec la Phase 8.5 de la roadmap :

# Trou Impact Concurrents Statut Paperasse
1 TVA dans les ecritures Sans TVA, produit limite aux franchises de TVA. Le regime reel simplifie (cible SASU/EURL) est bloque. Pennylane et Indy gerent TVA nativement Phase 8.5 livrable 1
2 Liasse fiscale structuree Indy et Pennylane generent la liasse automatiquement. Sans liasse, pas de cloture complete. 2033/2065 auto chez les deux Phase 8.5 livrable 2
3 Lien de partage expert-comptable C'est la promesse "triple couche". Sans partage EC, le positionnement ne tient pas. Pennylane : espace EC. Indy : EC optionnel Phase 8.5 livrable 3

Positionnement prix : pour etre credible face a Indy a 49€/mois (societes), Paperasse doit etre significativement en dessous. L'architecture rules-first (pas d'IA couteuse sur chaque transaction) rend ce positionnement economiquement viable.

Bottom line

Le vrai risque n'est pas de manquer des features (facturation, OCR, paie, teledeclaration). C'est de ne pas livrer les 3 fondamentaux comptables qui rendent le dossier utilisable par un expert-comptable au regime reel simplifie.

La Phase 8.5 (TVA, liasse, partage EC) est le seul prerequis. Tout le reste est du scope post-PMF.

Paperasse ne battra jamais Pennylane ni Indy sur le perimetre fonctionnel. Le differenciateur est ailleurs : transparence, auditabilite, cout, et un positionnement EC-first que personne n'occupe.
