# Où trouver mon code de confirmation Paypal ?

Pour relier votre compte pro Indy à votre compte Paypal, ces derniers vous créditent temporairement d'une somme de 0,01€ sur votre compte, avec un libellé spécifique contenant le code à 4 chiffres à renseigner sur Paypal.

Ce libellé n'apparaît pas directement sur votre interface Indy, **pour y accéder il convient tout d'abord de trouver la transaction sur votre compte Indy et de cliquer sur celle-ci** pour en afficher le détail :  
​  
​

[Image]

  
​  
👇 C'est dans l'en-tête de la **Ventilation de la transaction** que s'affiche le libellé de la transaction, le code Paypal est situé au début sous le **format PP.xxxx.PP 👇**  
​

[Image]

Vous devez catégoriser la transaction de **0,01 €** de **Paypal** en "**Autres gains divers"** _colonne **Revenus**_ ✅

  
Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite 😉

À très vite ! 👋
