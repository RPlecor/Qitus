# Quelle est la version de mon logiciel comptable Indy ?

Sur Indy, vous n'avez pas de mise à jour à faire vous même 🎉

Le logiciel est une application en ligne et se met à jour seul plusieurs fois par semaine, vous avez toujours la dernière version 🎁

## L'avantage ?

Si vous ou un autre utilisateur nous remonte un incident, celui-ci sera résolu dans les plus brefs délais et l'application sera mise à jour sans que vous ayez à faire quoi que ce soit de votre côté 🤓  
​

## Renseignements demandés

Le nom du logiciel : Indy.fr  
Fournisseur : Indy.fr SAS

  
Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite 😊
