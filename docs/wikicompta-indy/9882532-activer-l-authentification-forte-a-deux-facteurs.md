# Activer l'authentification forte à deux facteurs

Voici comment l'activer sur votre compte :

1️⃣ Rendez-vous dans vos **[Paramètres](https://app.indy.fr/compte/abonnement)**, partie **Sécurité.** Activez la coche **Authentification à 2 facteurs** 👇

[Image]

Il est nécessaire que votre adresse email ait été préalablement vérifiée.

Si ce n'est pas le cas, cliquez sur **Vérifier mon adresse mail** 👇

[Image]

2️⃣ Une fois que l'option a été configurée, il faudra **confirmer votre mot de passe** dans la modale qui s'ouvre pour des raisons de sécurité 👇

[Image]

3️⃣ Un QR code va ensuite s'afficher avec des instructions. Ouvrez Google Authenticator, scanner le QR code et saisissez le code à 6 chiffres👇

[Image]

💡 Vous pouvez désactiver l'option à tout moment dans vos **[Paramètres](https://app.indy.fr/compte/abonnement)**, partie **Sécurité 👇**

[Image]

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite !

À très vite 👋
