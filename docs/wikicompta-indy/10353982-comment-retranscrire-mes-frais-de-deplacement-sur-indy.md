## Restaurant et repas d'affaires

Les frais de restauration correspondent aux repas pris par le dirigeant.

Ainsi, vous devez déduire votre repas via cette catégorie si vous déjeunez seul, dans le cadre d'un déplacement professionnel par exemple 🍽️

## Et si j'ai un doute ?

Si vous hésitez dans le choix de la catégorie de votre transaction, vous pouvez cliquer sur **À catégoriser** sur la transaction concernée et taper dans le moteur de recherche.

_Par exemple : si vous ne savez pas comment catégoriser la transaction de vos frais de parking, vous pouvez écrire "parking" et Indy vous indiquera que cette dépense peut se placer dans **Frais de déplacement**._
