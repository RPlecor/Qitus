# Déduire les frais de véhicules

Si vous utilisez un véhicule pour vos déplacements professionnels, vous pouvez déduire ces frais de deux façons :

## Où faire le choix sur Indy ?

[Image]

## Je n’ai pas de frais de véhicule à retraiter

Si vous n’avez **aucun frais de véhicule à retraiter**, vous pouvez simplement répondre `Non` à la question _"Souhaitez-vous vous verser des indemnités kilométriques ?"_ puis passer à l’étape suivante 🙂

## Déduire vos frais via les indemnités kilométriques

Si vous utilisez un véhicule personnel pour vos trajets professionnels et que vous souhaitez utiliser le barème kilométrique, vous devez répondre `Oui` à la question _"Souhaitez-vous vous verser des indemnités kilométriques ?"_ 👇

[Image]

## Déduire vos frais au réel

Si vous préférez comptabiliser les dépenses réellement supportées (carburant, assurance, entretien etc.), vous devez répondre `Non` à la question _"Souhaitez-vous vous verser des indemnités kilométriques ?"_ 👇

[Image]

👉 Un message vous indique alors que les frais seront pris en compte au réel, à partir des dépenses enregistrées.

Dans ce cas, les dépenses liées au véhicule doivent être catégorisées en **Véhicule et carburant** afin d’être intégrées dans votre comptabilité ✅

## Cas particulier : LOA / LLD

Dans l’étape **Déduire les frais de véhicule**, Indy vous pose également la question suivante : _"Votre entreprise a-t-elle loué un véhicule en LLD ou LOA sur cet exercice ?"_ 👇

[Image]

[Image]

## Et les frais de parking / péage ?

Les **péages** et **frais de parking** ne sont pas inclus dans le barème kilométrique.

Vous pouvez alors les catégoriser en **Frais de déplacement** sur Indy ✅

## Le remboursement des frais de véhicule (si vous avez avancé des dépenses)

Si vous avez payé des dépenses liées au véhicule depuis votre **compte personnel**, vous pouvez vous **rembourser** via un virement depuis le **compte professionnel**.

Sur Indy, ce type de transaction doit être catégorisé en **Prélèvement personnel** ✅
