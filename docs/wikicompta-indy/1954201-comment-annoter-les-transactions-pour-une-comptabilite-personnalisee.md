# Comment annoter les transactions pour une comptabilité personnalisée ?

Vous pouvez personnaliser une transaction en ajoutant une annotation dessus.

## 🧩 Les annotations classiques

Il vous suffit de cliquer sur votre transaction afin d'inscrire ce que vous souhaitez.

_Par exemple, pour vos frais de déplacement, cela peut vous permettre de préciser la référence de votre voyage_ 👇

[Image]

Elles peuvent permettre d'ajouter des numéros de facture, des noms de fournisseurs...

Cela vous permet de **spécifier des transactions** et d'**analyser plus précisément** ces dernières.

**💡 À noter**

Les annotations sont retranscrites dans vos documents comptables (FEC et Grand Livre).

## 🔑 Les mots-clefs

Dans cette même logique, nous avons voulu ajouter des étiquettes aux transactions, c'est pourquoi vous pouvez à présent utiliser la fonctionnalité **#motclé**.

Cette fonctionnalité vous permet d'isoler certaines dépenses pour mieux les piloter.

Il suffit d'annoter les transactions avec un **#motclé** pour permettre ensuite de les suivre facilement avec le moteur de recherche.

Comme l'annotation mais avec l'ajout du signe dièse (hashtag) : **#**

Cela surlignera automatiquement le mot clé en orange 👇  
​

[Image]

_J'ai ajouté le mot-clé **#train** afin d'identifier rapidement les dépenses._

## 🔍 Les recherches sur Indy

Rendez-vous dans l'onglet **[Transactions](https://app.indy.fr/transactions)** pour retrouver les mots clés créés.

En haut de la page, vous pouvez effectuer une recherche. Toutes les transactions dans lesquelles vous avez ajouté l'annotation recherchée apparaitront 👇
