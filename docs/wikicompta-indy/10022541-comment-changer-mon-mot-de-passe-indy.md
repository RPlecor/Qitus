# Comment changer mon mot de passe Indy ?

Si vous avez besoin de modifier le mot de passe de votre compte Indy, rendez-vous dans **Paramètres > Onglet Sécurité.**

[Image]

Cliquez ensuite sur le bouton **"🔒 Modifier le mot de passe".**

Indy vous demande alors d'inscrire votre mot de passe actuel qui est vérifié 🔎

Vous devez ensuite inscrire votre nouveau mot de passe et le confirmer avant de pouvoir cliquer sur le bouton "**Modifier le mot de passe**".

  
Indy vous indique la force de votre nouveau mot de passe. Plus la barre se déplace vers la droite, plus votre mot de passe est fort 💪

👉 [Cliquez ici](https://wikicompta.indy.fr/fr/articles/9965796-pourquoi-votre-mot-de-passe-est-vulnerable) pour en savoir plus sur les bonnes pratiques pour créer un mot de passe non vulnérable.

Une fois votre mot de passe modifié, vous recevez un email de confirmation de la part d'Indy pour vous confirmer que votre mot de passe a été mis à jour ✅

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite ! 😊
