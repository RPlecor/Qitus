# Comment activer FranceConnect+ ? | Le wikicompta d'Indy

## À quoi sert FranceConnect+ ?

FranceConnect+ est une solution de l’État qui **renforce la sécurité** de certaines démarches en ligne (celles où l’on a besoin d’être “vraiment sûr” de qui vous êtes).  
Concrètement, FranceConnect+ s’appuie sur une **identité numérique** : c’est elle qui vous permet ensuite de vous connecter sur les sites compatibles.

## Dans quels cas j'ai besoin de FranceConnect+ ?

Vous pouvez croiser FranceConnect+ sur certaines démarches “sensibles”, par exemple sur le **Guichet unique (INPI)** pour le **dépôt des comptes annuels** (et plus largement certaines formalités).

## Étape 2 - Faire vérifier votre identité

C’est l’étape où l’on “valide” que vous êtes bien la bonne personne. Vous avez 2 possibilités 👇

### Option A : en bureau de poste

Vous vous présentez en bureau de poste avec votre **pièce d’identité**.  
À l’issue de la vérification, vous recevez un **code d’activation par SMS** (ou un code à saisir dans l’application) qui vous permettra de finaliser l’activation.

### Option B : à distance

Selon les cas, une vérification à distance peut être proposée.

👉 Si vous bloquez en ligne, le passage **en bureau de poste** est souvent la voie la plus simple.
