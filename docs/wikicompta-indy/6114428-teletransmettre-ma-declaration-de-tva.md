# Télétransmettre ma déclaration de TVA

Indy vous permet de télétransmettre votre déclaration de TVA directement depuis votre compte, sans passer par votre espace impots.gouv 🎉

Pour cela nous faisons directement le lien avec les impôts.

## Comment ça marche ?

Une étape dédiée figure dans votre activité de déclaration de TVA.

💡 Cette étape est également disponible pour les acomptes de TVA.

Elle s'appelle **Télétransmission de votre déclaration 👇**

[Image]

Pour cela, il vous suffira de cocher l'option que vous préférez👇

[Image]

⚠️ Si vous choissisez de déposer vous-même votre déclaration, n'oubliez pas d'aller reporter les montants directement sur votre espace professionnel du site des impôts !

Afin de télétransmettre directement votre déclaration aux impôts nous vous demanderons des informations supplémentaires, à savoir :

**💡 À savoir**

Ces informations sont à remplir uniquement lors de votre première télétransmission.

Nous les ré-utiliserons ensuite pour les déclarations suivantes 🤓

Une fois toutes les informations saisies vous pourrez **valider** l'étape.

Indy vous demandera d'effectuer une dernière vérification de votre déclaration de TVA, puis de **Télétransmettre** 🎉

Un mail de confirmation, avec l'accusé de réception joint, vous sera envoyé une fois la déclaration reçue par les impôts 👇

[Image]

Pour toutes questions supplémentaires, n'hésitez pas à nous écrire nous via le live-chat en bas à droite ! 😊
