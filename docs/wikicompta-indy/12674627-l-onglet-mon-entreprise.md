# L'onglet "Mon entreprise" | Le wikicompta d'Indy

Cet article vous présente l'onglet Mon entreprise et le rôle de chaque encart.

10 avril 2026

Table des matières

L'onglet **Mon entreprise** regroupe toutes les informations clés de votre entreprise pour que vous puissiez avoir un aperçu clair et rapide de celle-ci via deux encarts !

  
​

## **Encart principal**

Cet encart correspond à la carte d'identité de votre entreprise sur Indy. Passons en revue les différentes sections de celui-ci 👇

Vous y trouverez les informations suivantes

## **Activité**

Cette section indique l'**intitulé de votre profession** ainsi que le **code APE** (**A**ctivité **P**rincipale **E**xercée) qui y est associé. Ce code est fourni par l'INSEE et permet d'identifier la branche d'activité principale de votre entreprise.

## **Mode d'imposition**

C'est ici que l'on précise la manière dont vos bénéfices sont imposés. Étant donné que vous êtes soumis à l'impôt sur les sociétés, la mention suivante sera affichée : **IS - Impôt sur les sociétés ✅**
