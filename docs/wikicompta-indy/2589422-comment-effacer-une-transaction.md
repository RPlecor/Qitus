# Comment effacer une transaction ?

Grâce à la synchronisation bancaire, toutes les opérations présentes sur vos relevés bancaires sont présentes sur votre compte Indy.

🚨 Il n'est donc pas possible de **supprimer une transaction synchronisée**.

Cela fausserait votre comptabilité et pourrait être considéré comme de la fraude.

## 🧐 Que faire des transactions non professionnelles ?

## 👬 J'ai une transaction qui apparait en double

Il peut arriver que vous ayez des doublons dans vos transactions : cela fausse donc votre comptabilité. Dans ce cas, et **uniquement** dans ce cas, il faut la supprimer !

  
Cependant, vous devez nous contacter via le live-chat et nous fournir votre relevé bancaire prouvant que ce doublon n'apparaît pas et nous la supprimerons pour vous 👨‍💻

## ✍️ Les transactions ajoutées manuellement

Les transactions qui ne sont pas synchronisées avec votre compte bancaire et que vous ajoutez depuis l'onglet **Transactions** sont modifiables à tout moment.

Il vous suffit de cliquer sur la transaction manuelle déjà créée.

[Image]

Ensuite, cliquez sur la case **Supprimer** puis **Appliquer**.

[Image]

La transaction manuelle n'existe plus ✅

  
Pour savoir comment ajouter une transaction manuelle, vous pouvez consulter notre article **[ici](https://wikicompta.indy.fr/fr/articles/2028081-comment-ajouter-une-transaction-manuellement)**.
