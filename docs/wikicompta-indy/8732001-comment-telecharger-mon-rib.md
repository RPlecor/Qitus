# Comment télécharger mon RIB ?

## Depuis l'app mobile

Pour télécharger votre **RIB** au format PDF ou le partager à vos contacts, il vous suffit de vous rendre sur l'application Indy dans l'onglet **"Compte Pro"** et de cliquer sur **"RIB"** 👇

[Image]

Vous pouvez choisir de copier directement les informations du RIB (IBAN, BIC et titulaire du compte), ou bien cliquer sur **"Partager le RIB"** pour l'obtenir en PDF 👇

[Image]

## Depuis l'espace web

Il vous suffit de vous rendre sur l'onglet "**Compte pro"** et de cliquer sur **"RIB"** 👇  
​

[Image]

Vous pouvez choisir de copier directement les informations du RIB (IBAN, BIC et titulaire du compte), ou bien cliquer sur **"Télécharger le RIB"** pour l'obtenir en PDF 👇

[Image]

👉 Un onglet supplémentaire s'ouvre et vous permet de **télécharger le fichier.**

[Image]

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite 😉

À très vite ! 👋
