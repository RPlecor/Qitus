# Comment exporter mes documents comptables depuis Indy ?

## 📚 Quels documents sont disponibles sur Indy ?

Indy vous permet d'obtenir tous les documents dont vous avez besoin pour votre comptabilité.

Voici la liste des documents disponibles :

Tous ces documents forment votre **liasse fiscale 🤓**

Ils sont envoyés à votre AGA si vous en avez renseigné une sur Indy.

À cela, on peut ajouter les déclarations de TVA (format pdf), si vous êtes concerné·e 😊

C'est très simple ! Rendez-vous dans l'onglet **[Documents](https://app.indy.fr/documents).**

Vous pouvez visualiser d'un coup d'oeil l'ensemble des documents 👇

[Image]

👁️ Ils sont classés en :

Si vous souhaitez les télécharger, rien de plus simple, il suffit de cliquer sur l'icône de téléchargement à droite du document souhaité 👇

[Image]

**💡 À savoir**

L'ensemble des documents (sauf l'attestation de conformité qui est valable pour l'année) sont dynamiques. Ainsi toute modification effectuée depuis l'onglet **[Transactions](https://app.indy.fr/transactions)** les impactera directement.

📆 Par défaut, les documents proposés sont ceux de l'**année fiscale en cours**.

  
Si vous souhaitez accéder aux documents des années précédentes, il vous suffit de cliquer sur le bouton dédié en haut à droite de votre écran 👇

[Image]

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite 😉
