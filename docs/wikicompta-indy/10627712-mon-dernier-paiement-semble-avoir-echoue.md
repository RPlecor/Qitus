# Mon dernier paiement semble avoir échoué

En vous connectant à votre compte Indy, vous pouvez apercevoir ce type de notification 👇

[Image]

Plusieurs raisons existent dans le cas où le paiement de votre abonnement Indy ne fonctionne pas :

Dans le cas d'un prélèvement SEPA, il se peut que le paiement de votre abonnement soit bloqué pour des raisons de sécurité.

Dans ce cas de figure, nous vous recommandons de passer sur un **paiement par carte bancaire**.

Si le paiement de votre abonnement ne fonctionne pas, une nouvelle tentative de paiement aura lieu automatiquement sous quelques jours et ce, jusqu'à ce que le paiement fonctionne.

ℹ️ Dans certaines situations, votre facture peut entrer en **litige**. Cela peut se produire si vous ou votre banque refusez ou contestez le paiement.

Dans ce cas, vous ne pourrez plus utiliser Indy tant que la situation n'est pas régularisée.

Si vous avez opté pour le prélèvement SEPA, vous serez automatiquement redirigé vers le paiement par carte bancaire et une facture de recouvrement sera créee.

N'hésitez pas à revenir vers nous via le live-chat en bas à droite si cet article n'a pas répondu à toutes vos questions !
