# Comment clôturer mon compte pro Indy ?

Vous pouvez clôturer votre compte pro Indy en autonomie depuis **l'app mobile** ou la **[version Web d'Indy](https://app.indy.fr/)**.

Une fois que vos éventuels sous-comptes sont en statut "**En fermeture**" et que le solde du compte principal est à 0 :

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite 😉

À très vite ! 👋
