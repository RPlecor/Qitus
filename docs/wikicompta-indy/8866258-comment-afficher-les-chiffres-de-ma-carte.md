# Comment afficher les chiffres de ma carte ?

Si vous souhaitez ajouter vos coordonnées de carte sur un site pour précéder à un paiement, rien de plus simple !

## Depuis votre app Indy

  
1️⃣ Rendez-vous sur l'onglet "**Compte pro**" puis cliquez sur la carte concernée :  
​

[Image]

2️⃣ Cliquez sur **"Voir les détails" 👇**

[Image]

3️⃣ **Réalisez l'authentification** forte en saisissant votre code de sécurité à 6 chiffres.  
​

4️⃣ Un écran vous affichera ensuite vos informations de carte. Vous pourrez ainsi **copier ces informations** **sur le site commerçant** pour payer avec votre carte.

## **Depuis la [version web d'Indy](https://app.indy.fr/)**

1️⃣ Sous l'onglet "**Compte pro"** dans la section "**Cartes"**, cliquez sur la carte concernée 👇  
​

[Image]

2️⃣ Cliquez sur "**Voir les détails**" 👇  
​

[Image]

  
​

3️⃣ Faites l'**authentification forte 🔏 depuis le lien reçu par SMS depuis votre smartphone** :  
​

[Image]

  
​

4️⃣ Un écran vous affichera ensuite vos informations de carte. Vous pourrez ainsi **copier ces informations** **sur le site commerçant** pour payer avec votre carte :  
​

[Image]
