# Comment comptabiliser les pourboires que je verse et ceux que je reçois ?

  
Les pourboires sont des sommes d’argent supplémentaires payées par les clients, lorsqu'ils sont satisfaits d'un service.

💡 Les pourboires ne sont pas soumis à la TVA.

## **Comment comptabiliser ces pourboires ?**

## 👉 **Pour les pourboires versés**

Dans le cas où vous versez vous-même un pourboire, la catégorie à utiliser sur Indy est **Publicité**.

## 👉 **Pour les pourboires reçus**

Si vous recevez un pourboire de la part de vos clients, il est nécessaire de renseigner ces pourboires dans votre activité **Bilan de commerce**.

[Image]

Voyons ensemble comment les comptabiliser dans votre bilan de commerce.

## **Comment comptabiliser les pourboires reçus ?**

Lorsque vous réalisez votre bilan de commerce du mois, vous vous basez sur votre **Z de caisse** afin d'enregistrer vos ventes mensuelles.

Il est alors nécessaire **les enregistrer comme une vente** dont la nature sera `Pourboires`.

Le **taux de TVA n’est pas modifiable** car il doit être de **0%**👇

[Image]

Les pourboires entrent dans le **chiffre d’affaires imposable**. Vous pouvez retrouver leur montant dans votre compte de résultat à la ligne _Produits divers de gestion courante_ 👇

[Image]

## **Et si ce pourboire est reversé sous forme de rémunération ?**

Pour les salariés munis d’un contrat de travail, les pourboires ne doivent pas être inclus dans le calcul des charges sociales.

ℹ️ Les gérants de SARL/EURL ou les présidents de SAS/SASU ne sont pas concernés par cette exonération de charges sociales.

👉 **Si vous êtes en SAS ou SASU**, le montant du pourboire est à ajouter sur la fiche de paie dans la rémunération brute et sera soumis aux cotisations sociales.

👉 **Si vous êtes en SARL ou en EURL** : la transaction de re-versement du pourboire sur votre compte personnel devra être catégorisé en **\[Gérant\] Rémunération**.
