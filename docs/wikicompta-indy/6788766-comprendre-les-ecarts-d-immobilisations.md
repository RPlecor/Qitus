# Comprendre les écarts d'immobilisations | Le wikicompta d'Indy

Sur Indy, lorsque vous ajoutez une immobilisation achetée sur l'exercice en cours, il faut absolument que les montants soient les mêmes entre :

S'il y a une différence entre ces deux sommes, vous ne pourrez pas fermer l'étape.

👉 Cela nous permet de vérifier qu'aucune immobilisation n'a été oubliée dans le registre.

⚠️ Cette vérification est faite sur les immobilisations mais également sur les cessions.

## **⚖️ Résoudre un écart dans le registre**

Il existe deux cas possibles.

### 1️⃣ J'ai oublié des immobilisations dans mon registre

Si j'ai catégorisé des transactions en **Immobilisation** mais oublié de les inscrire dans l'étape, Indy me le signalera avec ce message 👇  
​

[Image]

**La solution :** vérifiez toutes vos transactions catégorisées en **Immobilisation**.

Pour cela, vous pouvez cliquer sur Voir les transactions directement dans l'étape 👇

[Image]

Le montant total doit être renseigné dans le registre.

⚠️ Si vous êtes assujetti·e à la TVA, pensez bien à vérifier les taux appliqués !

### 2️⃣ J'ai oublié de catégoriser mes transactions en Immobilisation

Si j'ai saisi des immobilisations dans mon registre, mais que je n'ai pas ajouté ou catégorisé la transaction correspondante en **Immobilisation**, Indy me le signalera avec ce message 👇

[Image]

**La solution** : vérifiez votre registre et comparez-le avec la liste des transactions catégorisées en **Immobilisation** sur votre exercice. Si certains achats ont été faits via votre compte personnel, il faudra les ajouter manuellement 👉 [article ici](https://wikicompta.indy.fr/fr/articles/2028081-comment-ajouter-une-transaction-manuellement) 🤓

## **💡 Modifier une immobilisation**

Pour modifier une immobilisation, il faut d'abord la supprimer avec la petite poubelle, puis la saisir à nouveau 👇

[Image]

Cette modification implique la **[réouverture d'un exercice](https://wikicompta.indy.fr/fr/articles/4507700-reouverture-et-changement-d-exercice-fiscal-annee-comment-faire)** si vous aviez fait une erreur en N-1 ou N-2 par exemple.
