# Étape de clôture des Revenus perçus en avance

Cette étape va vous permettre de gérer les revenus que vous avez perçu en avance en date de clôture, ce qu’on appelle aussi les produits constatés d'avance 🤓

Vous aurez la possibilité dans cette étape d'**ajouter** un nouveau revenu perçu en avance au cours de l'exercice 💰 Ils seront alors directement affectés à l'exercice suivant.

## 🔍 Les 4 types de commandes en cours

## ➕ Ajouter un revenu perçu en avance

Pour cela, il vous suffit de cliquer sur **\+ Ajouter** dans l'étape de clôture ✍️

[Image]

Puis de sélectionner le type du revenu que vous avez perçu en avance ✔️

En fonction du type sélectionné, vous devrez ensuite compléter chaque champ 👇

1️⃣ Dans le cas d'un loyer

2️⃣ Dans le cas d'un autre type de revenu

Il faut systématiquement ajouter une ligne par revenu perçu en avance ⚠️

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite !
