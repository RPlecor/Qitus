# Le registre des immobilisations : acquisitions, amortissements et cessions

## Qu’est-ce qu’une immobilisation ?

Le registre des immobilisations recense l’ensemble des biens durables acquis par votre entreprise : matériel informatique, véhicule, mobilier, logiciel, parts sociales…  
Ces éléments sont inscrits à l’actif du bilan et font l’objet d’un **amortissement** comptable sur plusieurs exercices.

Pour rappel, tout achat qui répond à la définition d’une immobilisation, à savoir 👇

doit obligatoirement être immobilisé dans votre comptabilité ✅

## Enregistrer une immobilisation

Cette étape se déroule en deux temps 👇

## 1️⃣ Catégoriser la dépense dans les Transactions

Rendez-vous dans votre onglet **Transactions** et classez la dépense concernée dans la catégorie d’immobilisation appropriée :

|     |     |     |
| --- | --- | --- |
| Type d’immobilisation | Description | Exemple |
| **Immobilisation corporelle** | Bien physique | Matériel informatique, véhicule, mobilier |
| **Autre immo. incorporelle** | Bien immatériel durable | Logiciel, fonds commercial, droit au bail |
| **Autre immobilisation financière** | Actif financier | Parts de SCM, dépôt de garantie |

## 2️⃣ Ajouter une entrée dans le registre des immobilisations

Une fois la transaction classée, allez dans **À faire > Clôture comptable et fiscale > Compléter le registre des immobilisations**, puis cliquez sur **\+ Ajouter** 👇

## Les immobilisations acquises lors d’un exercice précédent

👉 Si votre **premier exercice sur Indy** est aussi votre premier exercice d’activité, vous pouvez saisir directement toutes vos immobilisations depuis l’étape de clôture.

👉 Si vous avez **déjà clôturé un exercice**, les immobilisations antérieures sont automatiquement reprises lors de l’initialisation de votre bilan sur Indy.  
Elles apparaîtront dans votre registre mais **ne sont pas modifiables**, car elles sont déjà intégrées dans vos écritures passées.

Indy calculera ensuite automatiquement les prochains amortissements à passer lors de la clôture de l’exercice en cours 🤓

## Télécharger le registre des immobilisations

Vous avez besoin de récupérer votre registre (par exemple pour le transmettre à un organisme tiers) ?  
Vous pouvez désormais **télécharger le registre des immobilisations directement depuis Indy** 🎉

👉 Pour cela, rendez-vous dans **Documents > Comptabilité**, puis téléchargez le **registre des immobilisations**.

## Impossible d’oublier une entrée dans le registre

S’il existe un écart entre le **total des transactions** classées en immobilisation et le **total des entrées du registre**, Indy bloque la validation de l’étape pour éviter tout oubli 💪

## Céder une immobilisation

Lorsqu’un bien immobilisé est vendu, il faut enregistrer une **cession d’immobilisation** pour le sortir du patrimoine professionnel.

Pour cela :

Indy calcule automatiquement la **plus-value** ou **moins-value**, à court ou long terme, selon la durée de détention.

## Choisir le type d’exonération

Lors de la cession, Indy vous propose de choisir entre deux régimes 👇

Il suffit ensuite de cliquer sur **Céder** pour valider la sortie du bien du registre 🎉

## Exemple complet

Vous achetez un véhicule à 14 000 € HT le 1er mars 2025.  
Vous le catégorisez en **Immobilisation corporelle** dans vos **Transactions**, puis l’ajoutez dans le registre.

Après vérification, vous constatez que vous avez saisi 13 000 € au lieu de 14 000 €.  
👉 Vous pouvez désormais cliquer sur **Modifier**, corriger le montant et enregistrer.

Indy recalculera automatiquement les amortissements à partir du nouveau montant, sans devoir supprimer et recréer l’immobilisation 👌
