# Tout savoir sur les virements internes

## À quoi sert la catégorie “Virement interne” ?

La catégorie **Virement interne** est à utiliser **lorsqu’un virement est effectué entre deux comptes professionnels synchronisés sur Indy**. Il s’agit donc d’un simple transfert d’argent d’un de vos comptes à un autre, sans impact sur votre résultat comptable.

## Le compte comptable “Virement interne” doit toujours être équilibré

Ces deux mouvements sont enregistrés dans le compte comptable `580000 – Virement interne`, qui a pour vocation d’être **parfaitement équilibré**.  
👉 **Son solde doit toujours être égal à 0 €**.

Un solde différent de 0 signifie qu’il manque un des deux mouvements, ou qu’une erreur de catégorisation s’est glissée dans votre comptabilité.
