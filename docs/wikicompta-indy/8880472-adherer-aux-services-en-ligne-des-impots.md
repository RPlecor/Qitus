# Adhérer aux services en ligne des impôts

La création d'un espace professionnel sur impots.gouv est nécessaire pour déclarer et payer vos impôts (la TVA, l'impôt sur les sociétés, etc).

On vous explique comment faire 😊

Cela se fait en 2 temps :

1️⃣ Créer votre espace professionnel

2️⃣ Adhérer aux services nécessaires

## **1️⃣ Créer votre espace professionnel**

[Image]

Vous allez privilégier **le mode simplifié** : celui-ci vous permettra de créer un espace unique pour votre entreprise.

[Image]

## Valider votre adresse e-mail

Vous allez recevoir un e-mail de confirmation de la part des impôts, avec un lien.

[Image]

Il faudra **cliquer sur ce lien dans les 72h** afin de valider la procédure.

Cela permet aux impôts de vérifier votre adresse e-mail 🤓

Quelques semaines après cette étape, vous recevrez un **courrier papier avec un code d'activation,** afin de finaliser la création de votre espace impots.gouv.

Vous aurez alors 60 jours pour activer votre espace professionnel.

Passé 60 jours, il faudra **re-faire une demande d'adhésion en ligne** pour obtenir un nouveau code d'activation et donc finaliser la procédure.

## Activer votre espace professionnel

Une fois le courrier reçu, vous pourrez suivre [ce lien](https://cfspro-idp.impots.gouv.fr/oauth2/authorize?response_type=code&redirect_uri=https%3A%2F%2Fcfspro.impots.gouv.fr%2F%3Fopenidconnectcallback%3D1&nonce=1644999717_3551&client_id=id_cfspro_prod&display=&state=1644999717_28927&scope=openid%20profile%20email%20info) pour activer votre espace 👇

## 2️⃣ Adhérer aux services nécessaires

Sélectionner **_Gérer mes services_** dans votre espace professionnel 👇

[Image]

Dans le cadre services en ligne, cliquez sur **_Adhérer aux services en ligne_** 👇

[Image]

Saisir le **SIREN** de l’entreprise puis cliquer sur Valider.

Cocher ensuite le ou les services auxquels vous souhaitez adhérer : **_Consulter le compte fiscal_** par exemple, dans le cadre **_Les services auxquels vous pouvez adhérer._**
