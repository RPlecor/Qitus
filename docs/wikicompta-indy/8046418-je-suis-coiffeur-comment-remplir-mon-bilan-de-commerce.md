# Je suis coiffeur : comment remplir mon bilan de commerce ?

En tant que coiffeur ou coiffeuse, vous êtes concerné par le remplissage du bilan de commerce sur Indy ✅

Pour aller plus loin, venez découvrir quelle situation vous correspond le mieux :

Cet article détaille également **toutes les étapes à suivre** pour remplir correctement votre bilan mensuel.

Le bilan de commerce est une activité à remplir tous les mois dans l'onglet **À faire** sur Indy : il permet de récupérer toutes les informations nécessaires sur votre activité pour que votre comptabilité soit correcte.

Le bilan de commerce doit être activé manuellement dans **Paramètres > Options 👇**

[Image]

💬 Si vous ne voyez pas cette option, votre compte n’est pas peut-être pas correctement paramétré : contactez-nous sur le live-chat via la bulle rose en bas à droite de votre écran.

Voyons ensemble comment remplir les étapes du bilan mensuel commerçant 🤓

## Récupérer les bonnes informations

Avant de remplir votre bilan mensuel, il est nécessaire de récupérer l'ensemble des informations sur les ventes du mois.

👀 Ces éléments sont disponibles sur le **ticket Z de votre logiciel de caisse**. Aussi, si vous en avez ce mois-ci, les commissions de votre TPE sont à récupérer sur la facture de votre prestataire TPE.

Voyons ensemble un exemple d'activité sur un mois dans un salon de coiffure 👇

## Facture

En ce qui concerne la facture éditée pour le théâtre de 2 400€ TTC pour la prestation sur-mesure, celle-ci doit être saisie dans l'onglet **Facturation** de Indy comme ceci 👇

[Image]

Ainsi, le chiffre d'affaires de 2 000€ HT a déjà été comptabilisé et vous ne devez pas ajouter ce montant dans le bilan de commerce. Autrement, le chiffre d'affaires serait comptabilisé en double.

## Types de ventes

Dans cette première étape, il est nécessaire d'indiquer le type de ventes qui vous concerne :

Dans l'exemple d'un salon de coiffure, seule l'option `Ventes en boutique` devra être cochée ✅

[Image]

Une fois le type de vente renseigné, vous pouvez cliquer sur le bouton **Valider et passer à l'étape suivante.**

## Remboursements

Cette étape permet de déclarer des remboursements, annulations ou impayés.

🧐 Afin de pouvoir déclarer l'un de ces évènements, **le chiffre d'affaires doit être déclaré** dans l'un des bilans mensuels.

Dans le même exemple, j'ai réalisé un remboursement ce mois-ci d'un montant de 24€ TTC en espèces. Je l'ajoute en cliquant sur **+Ajouter une ligne**.

Si vous éditez des factures via l'onglet **Facturation** sur Indy, cet onglet présentera les factures qui sont actuellement en attente de paiement ainsi que les transactions bancaires qui ne sont pas encore reliées à une facture.

## Valider le bilan

Une fois les quatre étapes remplies, un récapitulatif vous permet de vérifier si les données renseignées sont conformes à votre logiciel de caisse.

[Image]

Une fois le bilan validé, les données seront prises en compte dans l'onglet **Pilotage** 🎉

Enfin, tous les bilans mensuels sont disponibles dans l'onglet **Facturation > Bilans mensuels.** Si besoin de corriger l'un deux, vous avez la possibilité de les rouvrir en cliquant sur les trois petits points puis sur le bouton **Modifier**.

[Image]
