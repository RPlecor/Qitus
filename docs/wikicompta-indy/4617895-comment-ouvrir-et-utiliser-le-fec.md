# Comment ouvrir et utiliser le FEC ?

## À quoi sert le FEC ?

Le fichier FEC est un fichier technique, édité dans un format standardisé qui **sert à l'administration fiscale en cas de contrôle.**

Le format étant spécifique, si vous n'avez pas de lecteur spécial, vous ne pourrez pas lire correctement ce document. Cela vous semblera illisible.

Pas d'inquiétude, vous n'avez pas besoin de le lire : il servira uniquement en cas de demande de l'administration fiscale.

Bonne nouvelle : Indy vous permet d'obtenir ce fichier en quelques clics 🖱️

Pour cela, rendez-vous sur votre compte Indy, onglet **Documents**.

Vous y trouverez votre **FEC** que vous pouvez télécharger en cliquant sur le bouton de téléchargement 👇

[Image]

Et voilà ! Vous pouvez désormais l'envoyer à l'**administration fiscale**. Celle-ci s'occupera de l'analyser.

💡 Pas d'inquiétude si vous ne parvenez pas à ouvrir ce fichier : celui-ci est lisible exclusivement grâce à un logiciel adapté et vous pouvez d'ailleurs le faire passer au **[testeur de la DGFiP](http://www.economie.gouv.fr/dgfip/outil-test-des-fichiers-des-ecritures-comptables-fec)** pour être certain de sa **conformité**.

Si vous êtes curieux de savoir **comment faire** pour ouvrir ce document, laissez-vous guider par les étapes ci-dessous 👇

**1 - Les pré-requis à connaître** 🚨

Il est bon de savoir que le logiciel fonctionne **uniquement** sur les ordinateurs équipés de Windows XP ou d’une version ultérieure de Windows.

Si vous souhaitez ouvrir ce fichier, pensez bien également à installer un **lecteur PDF** sur votre ordinateur.

  
​**2 - L'installation du logiciel** 🤓

👉 Pour commencer, **rendez-vous** sur **[cette page internet](https://www.economie.gouv.fr/dgfip/outil-test-des-fichiers-des-ecritures-comptables-fec)** de la DGFiP et cliquez sur ce lien pour télécharger le logiciel **Test Compta Démat** :

[Image]

👉 Choisissez ensuite la **version compatible** avec le système d’exploitation de l’ordinateur (système 32 bits ou 64 bits) :

[Image]

**👉 Lancez l’installation** en double-cliquant sur le fichier (.exe), puis en cliquant sur **Exécuter**,

👉 Après **installation** sur votre poste, vous n'aurez plus qu'à **importer** votre fichier FEC et le tour sera joué 🎉

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite 😉

À très vite !
