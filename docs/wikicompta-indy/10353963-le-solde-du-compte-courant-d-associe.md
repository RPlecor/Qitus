## Une petite précision sur l'onglet Transactions

L’onglet **Transactions** suit une autre logique, car il calcule uniquement la différence entre vos **entrées** et vos **sorties d’argent**.

Par conséquent, les montants affichés dans **Pilotage** peuvent parfois différer de ceux visibles dans l'onglet **Transactions** 🤓
