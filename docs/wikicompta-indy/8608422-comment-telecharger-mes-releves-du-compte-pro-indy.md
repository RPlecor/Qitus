# Comment télécharger mes relevés du compte pro Indy ?

Vos relevés de compte sont **disponibles à partir du 1er jour du mois**. Vous pouvez les télécharger depuis votre **app mobile Indy** ou depuis la **version Web**.

## **Depuis votre app Indy**

Il vous suffit d'aller dans l'onglet **Compte Pro** > **Documents** puis de cliquer sur le relevé souhaité 👇

[Image] [Image]

👉 Un nouvel écran s'affichera avec le **relevé en PDF à télécharger**.

[Image]

## **Depuis le web**

Il vous suffit d'aller dans l'onglet **Compte pro** > **Relevés et frais bancaires** 👇

[Image]

  
Vous pouvez ensuite sur **Visualiser** à droite du relevé souhaité.

[Image]

👉 Un nouvel écran s'affichera avec le **relevé en PDF à télécharger**.

[Image]
