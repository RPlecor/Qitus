# Comment intégrer les frais que j’ai engagés avant de créer ma SCI ?

Dans le cadre de la création de votre entreprise, il est possible que vous ayez avancé des frais depuis votre compte bancaire personnel 💰

Parmi les dépenses les plus courantes, nous retrouvons :

## Est-ce que je peux les intégrer dans la comptabilité de ma SCI ?

Oui, si ces frais respectent les 3 conditions suivantes :

De préférence, vous devez être en mesure de présenter les factures originales libellées **au nom de la société en cours d'immatriculation**. À défaut, celles-ci devront être établies au nom du gérant ou d’un associé de la SCI.

_Les factures établies à votre nom seront plus difficiles à justifier_ _en cas de contrôle auprès de l’administration fiscale_ 👮

## Comment intégrer ces frais sur Indy ?

Vous pouvez intégrer ces dépenses dans votre onglet **[Transactions](https://app.indy.fr/transactions)**, via le bouton **\+ Ajouter une transaction > Note de frais** puis sélectionner la catégorie correspondante :

_Si vous avez besoin d'un rappel sur le principe des notes de frais, c'est par [ici](https://wikicompta.indy.fr/fr/articles/6611638-comment-ajouter-une-transaction-qui-n-apparait-pas-sur-votre-compte) !_

Ces transactions vont ainsi créditer le solde de votre compte courant d'associé.

Vous pourrez donc vous rembourser les frais avancés via un virement de votre compte bancaire professionnel vers votre compte bancaire personnel qui devra être catégorisé en **Prélèvement personnel** sur Indy 🤓

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via la bulle rose en bas à droite ! 😊
