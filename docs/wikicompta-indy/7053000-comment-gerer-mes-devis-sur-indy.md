# Comment gérer mes devis sur Indy ?

## Créer un nouveau devis

## Sur ordinateur

Depuis l’onglet **Facturation**, cliquez sur **Nouveau**, puis **Créer un devis** 👇

[Image]

  
Le formulaire de création s’ouvre : vous pouvez alors remplir les champs nécessaires.

## Sur l’app mobile

Allez dans **Facturation**, puis cliquez sur **Devis**.  
Vous accédez à une interface dédiée, avec un bouton **Créer un nouveau devis** 👇

[Image]

Suivez les étapes de création : elles sont très similaires à celles sur ordinateur 🙂

Que vous soyez sur ordinateur ou sur l'app mobile, les informations à renseigner sont les suivantes 👇

Une fois le formulaire complété, vous pouvez valider votre devis et choisir un **numéro personnalisé** si besoin (ex : DEV-2025-001) ✔

## Ajouter des options à vos devis

Sur la version **web** d’Indy, vous pouvez désormais marquer certains articles comme **optionnels**.

## Visualiser et suivre vos devis

Tous les devis créés sont accessibles depuis l’onglet **Devis**, que ce soit sur ordinateur ou sur mobile.  
Ils sont automatiquement classés par statut :

## Modifier un devis

Vous pouvez modifier un devis tant qu’il n’a pas été :

Dans ce cas, le devis repassera automatiquement au statut **Brouillon**.

➡️ Pour cela, cliquez sur les **trois petits points** à droite de la ligne (sur web ou mobile), puis sur **Modifier le devis**. Vous pourrez ajuster n’importe quelle information : article, client, dates, etc 👇

[Image]

## Supprimer un devis

Tant que le devis **n’a pas été signé**, vous pouvez le supprimer à tout moment.

➡️ Pour cela, cliquez sur les **trois petits points** à droite de la ligne du devis, puis sélectionnez **Supprimer** 👇

[Image]

## Créer la facture associée à votre devis

Tout d'abord, il faudra marquer votre devis qui est **En attente de signature**, comme étant **signé** 👇

[Image]

Une fois que vous marquez un devis comme **signé**, Indy vous proposera automatiquement de créer la **facture associée** 👇

[Image]

Une fois cela fait :

📆 Vous pouvez aussi choisir de **créer la facture plus tard** si besoin.

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite 😉
