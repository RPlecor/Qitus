# Comment faire ma déclaration de TVA ?

Il existe plusieurs régimes de TVA (franchise en base, réel simplifié, réel normal) en fonction du chiffre d’affaires réalisé et du montant de TVA dû aux impôts chaque année.

👉 Vous trouverez plus d’informations à ce sujet dans **[cet article](https://wikicompta.indy.fr/fr/articles/6670923-quel-regime-de-tva-choisir)**.

**Indy vous accompagne et vous aide à éditer vos déclarations de TVA 🎉**

Cet article reprend les différentes étapes pour les éditer et les finaliser sur Indy.

## Vous êtes au régime réel normal

Vous faites donc une **déclaration mensuelle ou trimestrielle**, appelée CA3.

Indy vous ouvre cette déclaration **dès le 3 du mois** suivant la période concernée.

Afin d'éditer votre déclaration de TVA, il faut finaliser les étapes suivantes 👇

[Image]

## 0️⃣ Ajouter un crédit reportable à cette déclaration

Comme vous n'avez pas fait votre précédente déclaration de TVA avec nous, nous ne connaissons pas votre historique. Vous avez donc la possibilité ici d'indiquer si vous aviez (ou non) un **crédit de TVA à reporter** 🤓

Pour connaitre cette information, munissez-vous de votre dernière déclaration de TVA, le montant du crédit de TVA à reporter est situé :

Inscrivez ensuite le montant dans l'étape sur Indy ✔️

[Image]

Si vous n'avez pas de crédit de TVA sur votre précédente déclaration de TVA, vous pouvez simplement cocher **Non** et valider l'étape 😇

## 1️⃣ Catégoriser toutes les transactions

Il sera nécessaire de catégoriser toutes vos transactions de la période de déclaration concernée. Si ce n'est pas le cas, vous avez la possibilité de cliquer sur **Voir les transactions** et le faire directement dans l'étape 👇

[Image]

Si vous avez déjà catégorisé toutes vos transactions, c'est parfait !

## 2️⃣ Vérifier les taux de TVA

Il est également important d'avoir vérifié tous les taux de TVA renseignés pour vos transactions de la période de déclaration concernée puisque ce sont ces taux qui seront pris en compte et déclarés auprès des impôts 🔍

Indy vous affiche ainsi un message de rappel dans cette étape 👇

[Image]

Vérifiez tous les taux de TVA depuis votre onglet **Transactions** et validez l'étape **🎉**

## **3️⃣ Acompte de TVA payés  
​**

**💡Cette étape est disponible uniquement dans le cas où vous étiez en CA12 en cours d'exercice. Elle vous permet d'intégrer les acomptes déjà payés.**

  
Dans cette étape, vous devrez sélectionner dans vos transaction le ou les acomptes payés sur l'année.

[Image]

  
​  
Si une transaction catégorisée en **Acompte de TVA** n’est pas sélectionnée à ce moment, elle sera catégorisée automatiquement en **TVA payée**

Une fois les transactions sélectionnées, le montant de l’acompte est placé en **ligne 2C** de la CA3  
​

[Image]

Si vous étiez en **CA12** et que vous devez déclarer un acompte mais que cette section n'apparaît pas, n'hésitez pas à contacter notre service client en bas à droite.

## 4️⃣Demande de remboursement de crédit de TVA

Si vous avez déduit plus de TVA que vous n'en avez collecté, Indy va calculer automatiquement votre **crédit de TVA** 🧮 Sous certaines conditions, vous pouvez faire une demande de remboursement de ce crédit de TVA auprès des impôts.

Si vous êtes éligible à une demande de remboursement, vous pourrez renseigner les 5 plus grosses dépenses qui ont permis d'obtenir ce crédit de TVA.

[Image]

_Retrouvez comment réaliser votre demande de remboursement de crédit de TVA dans [cet article](https://wikicompta.indy.fr/fr/articles/4319003-credit-et-remboursement-de-credit-de-tva) 👈_

Si vous n'avez pas de crédit remboursable, ou si vous n'y êtes pas éligible, vous pouvez simplement choisir de **reporter votre crédit sur votre prochaine déclaration ✅**

## 5️⃣ Télétransmission de votre déclaration

La dernière étape permet de préparer la télétransmission de votre déclaration directement aux impôts 🚀

Si vous souhaitez activer la télétransmission, activez la coche 👇

[Image]

Remplissez ensuite les informations demandées afin qu'Indy s'occupe de la télétransmission, soit les informations de votre société et vos coordonnées bancaires.

## 6️⃣ Récapitulatif de votre déclaration

Afin d'avoir une vision plus claire sur votre TVA due ou votre crédit de TVA, Indy vous propose un tableau récapitulatif dans cette étape. Vous avez également la possibilité de voir la déclaration de TVA dans son intégralité en cliquant sur **Voir la déclaration** 👇

[Image]

## 7️⃣ Finaliser la déclaration

Vous venez de terminer toutes les étapes de votre déclaration de TVA 🥳

Il faut maintenant la déclarer aux impôts en suivant les informations données par Indy, qui diffèrent selon votre choix (télétransmission ou télédéclaration).

→ Vous avez opté pour qu'elle soit **télétransmise** 👇

[Image]

→ Vous avez opté pour qu'elle soit **télédéclarée** 👇

[Image]

## Vous êtes au régime simplifié

Vous faites donc une **déclaration annuelle**, appelée CA12.

Elle se décompose en **deux acomptes** - un en juillet et un en décembre de l’année concernée, puis une déclaration de solde finale, en mai de l'année suivante.

Afin d'éditer votre déclaration de TVA, il faut finaliser les étapes suivantes 👇

[Image]

Plusieurs étapes sont similaires avec la déclaration mensuelle ou trimestrielle (CA3) :

Vous pouvez donc vous **reporter à la partie du dessus** pour finaliser ces étapes ☝️

Nous allons ici regarder les étapes supplémentaires.

## ➡️ Acomptes de TVA

Cette étape correspond à la vérification des **Acomptes de TVA** que vous avez payé durant l'exercice 🤓 Pour tout savoir sur les acomptes, consultez notre article **[juste ici](https://wikicompta.indy.fr/fr/articles/5339005-tout-savoir-sur-les-acomptes-de-tva)** !

Si vous avez versé des acomptes de TVA pendant votre exercice, vous avez donc dû les catégoriser en **TVA payée** dans votre onglet **Transactions**. Automatiquement, Indy les remonte ensuite dans cette étape.

Il faudra dans le tableau, renseigner le montant de chaque acompte de TVA payé (ou restant à payer), et sélectionner les transactions correspondantes aux acomptes de TVA payés afin de les déduire du solde final de TVA 👇

[Image]

Si vous avez payé vos acomptes de TVA comme attendu, vous n'avez rien à compléter dans la partie **Montant restant à payer** 👉 vous devez renseigner un montant dans cette partie si vous avez demandé à étaler le paiement de vos acomptes par exemple.

💡 Si vous n'avez versé aucun acompte de TVA pendant votre exercice, vous n'avez donc rien à sélectionner et vous pouvez simplement **Valider** l'étape.

## ➡️ TVA sur vos opérations à l'étranger (UE et hors UE)

Cette étape a un but **informatif**.

Indy remonte ainsi les transactions ayant un taux de TVA **UE** dans cette étape.

Vous devez indiquer si ces transactions concernent des achats de biens physiques dans un pays membre de l'UE ou non👇

[Image]

Si **Non**, alors rien de plus à faire, vous pouvez valider l'étape ✅

Si **Oui**, alors il faudra contacter votre SIE avant de finaliser votre déclaration de TVA.

Votre SIE vous indiquera la marche à suivre.
