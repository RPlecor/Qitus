# Comment catégoriser une transaction ?

Comment catégoriser les transactions bancaires de votre SCI ?

12 janvier 2026

Table des matières

Avec Indy, **90% de vos transactions bancaires sont catégorisées automatiquement 🙌**

Certaines transactions parmi les plus délicates peuvent échapper au premier niveau de catégorisation c'est alors à vous de trouver la bonne catégorie ✨

## **Transactions à catégoriser**

Vous pouvez facilement voir le nombre de transactions qui n'ont pas encore de catégorie attribuée sur Indy ; depuis votre onglet **[Transactions](https://app.indy.fr/transactions)**, il vous suffit de cliquer sur **Catégoriser ces transactions** pour accéder à la liste des écritures **À catégoriser 👇**

[Image]

  
​

## **Rechercher une catégorie**

Si vous ne savez pas quelle catégorie appliquer à une transaction, vous pouvez utiliser la **barre de recherche** en tapant le **mot-clé** auquel se réfère votre transaction et vous devriez trouver la catégorie adaptée👇

[Image]

Si vous avez un doute sur une catégorie, vous pouvez passer votre curseur sur chacune des catégories pour obtenir une description plus précise de son contenu.

## **Pour vous aider, quelques exemples**

|     |     |     |
| --- | --- | --- |
| **Types de transaction** | **Catégories à appliquer**<br><br>**sur Indy** | **Remarques** |
| Je perçois des loyers tous les mois. | **Loyer encaissé** | Si le loyer contient une provision pour charges il faut [subdiviser](https://wikicompta.indy.fr/fr/articles/6603878-comment-subdiviser-une-transaction) la transaction en ventilant la part des provisions en **Provisions pour charges locatives**. |
| J'ai réglé le paiement de la taxe foncière. | **Taxe foncière** | La taxe foncière peut être mensualisée. |
| Tous les mois, je paie des mensualités de remboursement d'emprunt à la banque. | **Remboursement d'emprunt** | Au moment de la clôture, vous aurez [une étape pour ventiler ces transactions](https://wikicompta.indy.fr/fr/articles/6990308-ventiler-vos-echeances-d-emprunt) en trois catégories :<br><br>*   **Remboursement d'emprunt** pour la part de capital<br>    <br>*   **Intérêts & frais d'emprunt** pour les intérêts<br>    <br>*   **Assurance emprunt** |
| J'ai régulièrement des frais bancaires (agios pour découvert, frais de dossier, frais de comptes...). | **Frais bancaires** |     |
| Je fais régulièrement des transferts d'argent entre le compte de ma SCI et mon compte bancaire personnel. | Si c'est un apport pour votre SCI 👉 **Apport personnel**<br><br>Si vous achetez quelque chose pour votre SCI avec votre argent personnel et que vous souhaitez vous rembourser 👉 **Prélèvement personnel**<br><br>Si vous souhaitez retirer de l'argent du compte de la SCI vers votre compte bancaire personnel 👉 **Prélèvement personnel** | Pour l'intégration de dépenses faites avec votre compte bancaire personnel, pensez à ajouter une [note de frais](https://wikicompta.indy.fr/fr/articles/6611638-comment-ajouter-une-transaction-qui-n-apparait-pas-sur-mon-compte) catégorisée selon la nature de la dépense. |
| Je paie des honoraires à une agence immobilière qui gère mes biens immobiliers et mes locataires. | **Honoraires divers** |     |
| Je loue un système de surveillance (caméra, alarme, etc.). | **Location de matériel** |     |
| J'ai acheté un système de surveillance (caméra, alarme, etc.). | **Autre immobilisation corporelle** ou **Matériel et outillage** |     |
