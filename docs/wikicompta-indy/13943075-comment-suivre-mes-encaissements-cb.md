# Comment suivre mes encaissements CB ?

Vous pouvez consulter la liste de vos derniers encaissements CB et leurs statuts depuis l'app mobile ou la version web.

## Depuis l'app mobile Indy

Il vous suffit d'aller dans l'onglet **Compte pro > Encaissement CB**.

## Depuis la version web

Dans l'onglet **Compte pro** > **Encaissements**, il vous suffit de cliquer sur **Paramètres** dans l'encadré **Encaissement CB**.

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite 😉

À très vite !
