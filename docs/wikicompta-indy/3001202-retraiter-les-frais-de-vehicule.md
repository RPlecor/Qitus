# Retraiter les frais de véhicule

Vous pouvez comptabiliser vos déplacements professionnels de deux façons : avec le barème kilométrique ou bien avec les frais réels.

2 mars 2026

Table des matières

Dans cette étape, vous pourrez faire votre choix entre **le barème km** et **les frais réels**.

## Barème kilométrique 🚗

Si vous faites ce choix, il faudra cocher **Barème kilométrique** dans **[l'étape dédiée](https://app.indy.fr/activities/d-2035-activity/car-fees)** 👇

[Image]

**💡 À noter**

Les deux sont compatibles entre eux 😊

_Par exemple_

_Dans le cadre de votre activité vous utilisez un véhicule de location ainsi que votre moto personnelle ponctuellement. Vous pouvez appliquer le barème BIC sur le véhicule de location ainsi que le barème BNC sur la moto._

Vous pourrez ensuite vous rendre dans l'étape 👉 **[Barème kilométrique forfaitaire](https://app.indy.fr/activities/d-2035-activity/kilometer-option)** afin de saisir les informations nécessaires au calcul de votre indemnité.

L'indemnité sera calculée pour chaque véhicule renseigné dans l'étape 👇

[Image]

Une fois l'ensemble des véhicules ajoutés, vous pourrez **Valider et passer à l'étape suivante.** Indy va créer une ligne comptable (une opération diverse) au **31 décembre** de l'exercice en cours 🤖

[Image]

Cette indemnité sera visible en ligne 23 👉 **Frais de véhicule** de votre 2035.

## Frais réels 📝

Si vous choisissez d'être aux **frais réels**, il faudra le renseigner dans **[l'étape dédiée](https://app.indy.fr/activities/d-2035-activity/car-fees)** 👇

[Image]

Ensuite il faudra saisir le pourcentage d'usage **personnel** de votre véhicule.

Indy fera une ventilation de toutes les transactions catégorisées en **Véhicule et carburant** en fonction du % renseigné 😊

_Par exemple_

_Si vous faites un usage personnel de votre voiture le week-end, vous pouvez saisir 25% dans l'étape. Indy ventilera automatiquement 25% de vos transactions de véhicule en **Dépense personnelle**, ainsi cette partie ne sera pas déduite dans votre 2035._

Le total déductible sera visible en ligne 23 👉 **Frais de véhicule** de votre 2035.

Les frais déductibles ne seront pas les mêmes si votre véhicule est immobilisé ou non.

### 🚗 Véhicule de location

Si vous avez un véhicule en LLD ou LOA vous devrez le renseigner aussi dans l'étape **[Retraiter les frais de véhicule](https://app.indy.fr/activities/d-2035-activity/car-fees)** et renseigner le montant des amortissements non déductibles sur vos loyers.

  
Ce montant est indiqué sur votre contrat de location et doit être transmis chaque année par l'entreprise qui vous loue le véhicule.
