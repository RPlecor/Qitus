# Financer l'achat de mon immobilisation

Lors de l'acquisition d'une immobilisation, vous pouvez l'acheter comptant, ou bien faire appel à d'autres modes de financement.

## 1️⃣ Le financement par un emprunt bancaire

Vous pouvez contracter un emprunt bancaire pour financer votre immobilisation.

Dans ce cas, plusieurs transactions apparaissent sur votre espace Indy 👇

Renseignez ensuite votre immobilisation dans le **registre des immobilisations** à la date de son acquisition ➕

Pour cela, rendez-vous dans l'étape de clôture **Compléter le registre d’immobilisations** disponible dans l'onglet **[À faire](https://app.indy.fr/activites)** de votre compte Indy ✅

## 2️⃣ Le financement par un crédit-bail avec option d'achat

Si vous avez financé votre matériel via un crédit-bail avec option d'achat, l'immobilisation sera tout de même à enregistrer dans le registre de vos immobilisations mais **uniquement** **lorsque le financement sera complet et que vous aurez activé votre option d'achat** 🚨

Deux types d'immobilisations peuvent être acquises en crédit-bail :

Une fois que le véhicule ou le matériel est entièrement financé et que vous souhaitez en faire l'acquisition, il faudra alors catégoriser le prix de rachat en **Immobilisation corporelle**, et ajouter cette immobilisation dans votre registre des immobilisations à la date de l'option d'achat 🙂

À ce moment là, Indy vous demandera la **valeur HT initiale avant le début du contrat** 👇
