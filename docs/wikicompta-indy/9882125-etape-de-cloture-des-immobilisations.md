# Étape de clôture des Immobilisations

Cette étape va vous permettre de gérer vos immobilisations autres que les biens immobiliers et les travaux : immobilisations corporelles, incorporelles et financières ✏️

Vous aurez la possibilité dans cette étape👇

[Image]

## **Ajouter une immobilisation**

Pour ajouter une immobilisation sur Indy, il y a 2 étapes ✌

**1️⃣ Catégoriser la transaction concernée**

La première étape est de catégoriser la transaction d’achat de votre immobilisation dans l’onglet **Transactions** 👇

[Image]

**2️⃣ Ajouter l’immobilisation dans le registre**

La deuxième étape est de l’ajouter dans votre registre des immobilisations, situé au sein de l’activité **Clôture comptable et fiscale** > **Vérifier les immobilisations**.

Pour visualiser toutes les immobilisations à ajouter en un coup d’oeil, vous pouvez cliquer sur **Voir la (ou les) transaction** puis sur **\+ Ajouter** pour l'ajouter 👇

[![image___qnkka9nygl96tufk.png](https://indy-5e7b6340ff8c.intercom-attachments-1.com/i/o/1181649766/7b15762672a5a6713d4562e8/image___qnkka9nygl96tufk.png?expires=1779643800&signature=8f69e7836a4c0a8ecebe0bfac06bad9185102b63a21f17fa6d77eb6e7b863dd3&req=dSEvF896lIZZX%2FMW1HO4zbNEnasMFuZRacp4vLX2rFS6WdSmrIQvlEj0Hx%2Bu%0A6d7pMjnywlri6oNJ5T8%3D%0A)](https://indy-5e7b6340ff8c.intercom-attachments-1.com/i/o/1181649766/7b15762672a5a6713d4562e8/image___qnkka9nygl96tufk.png?expires=1779643800&signature=8f69e7836a4c0a8ecebe0bfac06bad9185102b63a21f17fa6d77eb6e7b863dd3&req=dSEvF896lIZZX%2FMW1HO4zbNEnasMFuZRacp4vLX2rFS6WdSmrIQvlEj0Hx%2Bu%0A6d7pMjnywlri6oNJ5T8%3D%0A)

Il faudra ajouter chaque immobilisation catégorisée comme telle dans le registre !

Chaque champ doit ensuite être complété ✍

→ Puis vous pouvez valider l’étape et passer à l'étape de clôture suivante ✅

## **Modifier ou supprimer une immobilisation**

Pour modifier une immobilisation acquise sur l'exercice, il est nécessaire de **la supprimer puis de la rajouter** à nouveau avec les bonnes informations 👇

[Image]

Dans le cas où vous avez des immobilisations financières et si ces dernières ont pris ou perdu de la valeur, il est nécessaire de l'indiquer dans l'étape en cliquant sur le crayon à côté de chaque immobilisation financière 👇

[Image]

Une nouvelle fenêtre s'ouvre vous permettant d'indiquer la **nouvelle valeur de l'immobilisation**.

## Céder une immobilisation

Si une de vos immobilisations a été vendue sur l'exercice, il est nécessaire de **la céder** dans votre registre des immobilisations, pour cela cliquez sur **\+ Céder 👇**

[Image]

Chaque champ doit ensuite être complété ✍

## Le registre des immobilisations

Votre registre des immobilisations sera accessible directement dans votre onglet **Documents**, puis **Registre des immobilisations** 👇

[Image]

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite !
