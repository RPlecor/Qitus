# Comptabiliser le cashback | Le wikicompta d'Indy

Cet article vous explique ce qu’est le cashback et comment comptabiliser correctement ce type d’opération sur Indy 👇

## Qu’est-ce que le cashback ?

Un site de cashback est une plateforme de récompense qui reverse à ses membres un pourcentage du montant de leurs achats lorsqu’ils passent commande via ses liens d’affiliation 🔗

👉 En d’autres termes, une partie de la dépense effectuée est récupérée sous forme de recette.
