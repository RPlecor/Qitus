# Comment faire sa cessation d'activité sur Indy ?

En cessation d'activité, il existe deux cas de figure pour déclarer vos revenus :

## **👀 Pré-requis**

Tout d'abord, vous devez procéder à la déclaration de cessation d'activité sur le site du **[guichet des formalités des entreprises](https://entreprendre.service-public.fr/vosdroits/R61572)** dans les **30 jours qui suivent la cessation d'activité**.

Les organismes obligatoires (INSEE, caisses de protections sociales...), seront automatiquement informés afin de procéder au calcul de vos dernières cotisations.

## **💻 Comment faire sur Indy ?**

Sur Indy, il faudra suivre différentes étapes afin de préparer votre dernière **clôture comptable micro-BIC**.

**1ère étape : contactez-nous sur le live-chat**

Si les étapes de clôture ne sont pas ouvertes, contactez-nous sur le live-chat afin que nous puissions les ouvrir pour vous de façon anticipée 🤗

**2ème étape : suppression des transactions & validation du solde final**

Toute transaction après la date de votre cessation devra être supprimée.

En revanche, certaines de ces transactions doivent bien apparaitre dans votre comptabilité (prélèvement Urssaf, abonnement Indy...).

Il faut donc **[ajouter manuellement](https://wikicompta.indy.fr/fr/articles/2028081-comment-ajouter-une-transaction-manuellement)** ces transactions **à la date de cessation d’activité** pour qu’elles soient bien prises en compte dans votre comptabilité ✅

➡️ Prévenez-nous dès que ce sera fait, afin que l'on supprime les transactions après votre date de cessation.

Pour cela, envoyez-nous **vos derniers relevés de banque** au format PDF, afin que nous validions votre solde final dans l'étape **[4 - Vérifier les soldes bancaires](https://app.indy.fr/activities/d-2035-activity/initialize-financial-accounts)** 📮  
​

Pour le reste, il vous suffira de faire toutes vos étapes de clôture de façon habituelle.

Vous pourrez ensuite envoyer valider votre exercice et accéder à votre déclaration de revenus 👀

## **👋 Désabonnement Indy**

À noter : en cessation, il se peut que certains organismes vous contraignent d’établir une **déclaration 2042 C-PRO rectificative.** C'est le cas, par exemple, lorsque vous avez des régularisations auprès de l'Urssaf.

  
Nous vous recommandons de **rester abonné jusqu’à la fin du processus de cessation d’activité**, afin de pouvoir **modifier votre déclaration si besoin** 🙏

  
Pour vous simplifier la vie :

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite ! 😊
