# Immobilisations financières ou titres de participation : quelle différence ?

## Pourquoi cette distinction est importante ?

Quand vous ajoutez une immobilisation sur Indy, vous devez préciser s’il s’agit d’une **immobilisation financière** ou de **titres de participation**.

1️⃣ Vous ajoutez votre immobilisation dans le registre 👇

[Image]

2️⃣ Vous devez sélectionner la bonne nature d'immobilisation 👇

[Image]

  
👉 Cette distinction a des conséquences comptables et fiscales, notamment en cas de **cession** ou de **distribution de dividendes**.

## Immobilisations financières

Ce sont des **placements financiers** détenus durablement, **sans volonté de contrôle**.

## Titres de participation

Ils correspondent à des **parts détenues dans une entreprise avec influence**.

## Différences clés

|     |     |     |
| --- | --- | --- |
|     | **Immobilisation financière** | **Titre de participation** |
| **Objectif** | Placement passif | Investissement actif |
| **Pouvoir dans la société émettrice** | Aucun | Influence ou contrôle |
| **Exemples** | Parts sociales d’une banque, obligations, fonds | \>10% du capital, parts de SCM |
| **Fiscalité en cas de cession** | Plus-value imposée normalement | Régime particulier des plus-values long terme (exonération partielle) |

## Et sur Indy ?

Lors de l’ajout dans le registre des immobilisations :

Dans les deux cas, vous pouvez ajouter une **provision pour dépréciation** si la valeur baisse 🤓

ℹ️ Assurez-vous également que la **transaction d'achat relative est également catégorisée de la même manière**.

En effet, vous trouverez deux catégories comptables distinctes : **Autre immobilisation financière** ou **Titres de participation** 👇
