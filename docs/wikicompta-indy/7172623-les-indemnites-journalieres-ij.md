# Les indemnités journalières (IJ) | Le wikicompta d'Indy

En cas d’arrêt de travail, vous pouvez être amenés à recevoir des indemnités journalières de la part de la **CPAM**, de votre **caisse de retraite** ou de votre **mutuelle/prévoyance Madelin**. Ces indemnités se comptabilisent et se déclarent différemment selon l'organisme qui vous les a versées.

Cet article a donc pour objectif de vous expliquer comment gérer ces revenus sur Indy.

## Le fonctionnement de l'étape

1️⃣ Pour commencer, les indemnités journalières reçues doivent être classées dans la catégorie **Indemnité journalière** pour le **montant net réellement perçu 👍**

2️⃣ Une fois que les transactions sont bien catégorisées, il faudra vous rendre dans l'onglet 👉 **[À faire](https://app.indy.fr/activites) > Déclaration 2035 > étape Indemnités journalières**.

À cet endroit, toutes les transactions préalablement catégorisées en **Indemnité journalière**, vont remonter dans un tableau.

Vous devrez ainsi compléter chaque indemnité une à une, afin de permettre à Indy de faire les retraitements comptables nécessaires, en indiquant :

3️⃣ Une fois l'étape validée, une opération diverse est générée 👇

## **Où retrouver le détail des indemnités journalières de la sécurité sociale (CPAM) ?**

Sur le site **ameli.fr** dans le menu, choisir 👉 **Mes paiements > Mes paiements**, puis entrer la date des indemnités journalières que vous souhaitez consulter.
