# Comment faire ma déclaration et mes acomptes de TVA au régime simplifié ?

## Ce que dit la loi

Lorsque vous êtes au régime simplifié, vous devez déposer (jusqu'à) 3 déclarations par an auprès des impôts.

Voici les déclarations attendues :

**_Par exemple :_**

Pour l'exercice 2025, vous allez déposer 2 acomptes en juillet puis décembre 2025.

Ensuite, vous avez jusqu'à début mai 2026 pour faire la déclaration finale.  
​

## Préparer ses acomptes avec Indy

Le calcul des acomptes est strictement défini par les impôts.

Vous trouverez une activité pour calculer ces acomptes dans l'onglet **[À faire](https://app.indy.fr/activities/vat-advance-payment-activity)** en temps voulu.

Cela vous permettra de calculer le montant dû pour chaque acompte.

Pour cela vous pourrez renseigner dans la 1ère étape le montant de la ligne 57 de votre déclaration précédente :

[Image]

💡 S'il s'agit de votre première déclaration d'acompte, les impôts attendent un acompte correspondant à 80% de la TVA due sur la période.

Indy calculera automatiquement le montant de l'acompte à verser dans l'étape 2.

Si vous choisissez de moduler cet acompte, vous pourrez changer ce montant 👇

[Image]

Enfin si vous avez un crédit à imputer il faudra le saisir dans la dernière étape :

[Image]

Une fois toutes ces étapes complétées, vous pourrez cliquer sur **Valider et passer à l'étape suivante** 👇

[Image]

La déclaration d'acompte est terminée 🎉

Vous pouvez ensuite opter pour la télétransmission de l'acompte de TVA aux impôts depuis votre compte Indy👇

[Image]

⚠️ Pour que la télétransmission fonctionne correctement, les coordonnées bancaires doivent être celles que vous avez enregistrées sur votre espace professionnel impots.gouv.  
​

**Ensuite vous pouvez valider pour que la télétransmission se fasse,** vous recevrez un mail de confirmation avec votre accusé de réception **🎉**

🙅 Si vous souhaitez déclarer l'acompte de TVA vous même c'est possible, il suffit de **refuser la télétransmission par Indy en décochant la case** et valider l'étape.

## Finaliser sa déclaration annuelle

Dès janvier de l'année suivante, une autre activité est disponible 👉 onglet **[À faire](https://app.indy.fr/activities/vat-activity)**.

Elle vous permet de préparer la CA12.

Pour la préparer, il vous suffit de suivre les différentes étapes proposées 👇

[Image]

Ces étapes vous permettent de :

**0️⃣ Ajouter un crédit reportable à cette déclaration**  
​

**💡 Cette étape est disponible uniquement lors de votre 1ère déclaration avec Indy.**  
​

Si vous venez d'arriver sur Indy et que vous n'avez pas fait votre précédente déclaration de TVA via notre application, nous avons besoin de savoir si vous aviez un crédit de TVA en cours.  
​  
Vous avez donc la possibilité ici d'indiquer si vous aviez (ou non) un **crédit de TVA à reporter** 🤓

Pour connaitre cette information, munissez-vous de votre dernière déclaration de TVA, le montant du crédit de TVA à reporter est situé :

[Image]

[Image]

Inscrivez ensuite le montant dans l'étape sur Indy :  
​

[Image]

1️⃣ **Catégoriser toutes les transactions**

À cette étape, Indy vous demandera de catégoriser toutes vos transactions sur la période déclarée. Cela permettra de s'assurer que :

[Image]

Vous ne pourrez passer à l’étape suivante que lorsque toutes vos transactions seront catégorisées.  
​

**2️⃣** ​**Ventiler les repas hors domicile**  
​  
Si vous avez appliqué de la TVA sur vos transactions de **Frais de repas hors domicile**, elle sera calculée uniquement sur la part déductible.  
​  
Pour en savoir plus sur la part déductible, vous pouvez **[cliquer ici](https://wikicompta.indy.fr/fr/articles/3064398-ventiler-mes-repas-hors-domicile) 👈**

  
​**3️⃣ Vérifier les taux de TVA**

  
Il est important d'avoir **vérifié tous les taux de TVA renseignés** pour vos transactions de la période de déclaration concernée puisque ce sont ces taux qui seront pris en compte et déclarés auprès des impôts 🔍

Indy vous affiche ainsi un message de rappel dans cette étape 👇  
​

[Image]

**4️⃣ Vérifier ou ajouter les acomptes payés**

**💡 Les montants seront pré-remplis dans cette étape vous avez fait vos acomptes de TVA sur Indy.**

Cette étape permet de saisir le montant des acomptes payés pour l’année en cours.  
Il faut les identifier dans vos transactions afin de valider l’étape 😊

Vos acomptes doivent être catégorisés en **TVA Payée** pour remonter dans l’étape.

Vous devez donc :

[Image]

💡 Le montant des acomptes de TVA doivent obligatoirement être saisies en nombre entier (pas de décimale).

[Image]

On peut reporter dans cette étape un crédit de TVA imputé sur les acomptes.

Cette information est à retrouver en ligne 2 de vos déclarations d’acomptes :

[Image]

Si cette ligne n’est pas remplie dans les déclarations d’acomptes il faut laisser la case vide sur Indy.  
​

**5️⃣ Vérifier si vous avez fait des achats avec un taux UE et hors UE**  
​

Cette étape a un but **informatif**.

Indy remonte ainsi les transactions ayant un taux de TVA étrangers dans cette étape.

  
Vous devez indiquer si ces transactions concernent des achats de biens physiques ou non👇  
​

[Image]

Si **Non**, alors rien de plus à faire, vous pouvez valider l'étape ✅  
​

Si **Oui**, alors il faudra contacter votre SIE avant de finaliser votre déclaration de TVA.

Votre SIE vous indiquera la marche à suivre.

6️⃣ **Faire une demande de remboursement de crédit si nécessaire**

  
Si vous avez déduit plus de TVA que vous n'en avez collecté, Indy va calculer automatiquement votre **crédit de TVA** 🧮  
​  
Sous certaines conditions, vous pouvez faire une demande de remboursement de ce crédit de TVA auprès des impôts 👉 plus d'infos dans [cet article](https://wikicompta.indy.fr/fr/articles/4319003-credit-et-remboursement-de-tva)

7️⃣ **Renseigner les informations relatives à votre entreprise**

  
La dernière étape permet de préparer la télétransmission de votre déclaration directement aux impôts 🚀

Si vous souhaitez activer la télétransmission, activez la coche 👇  
​

[Image]

Remplissez ensuite les informations demandées afin qu'Indy s'occupe de la télétransmission et du paiement, soit les informations de votre société et vos coordonnées bancaires.

🔍 Les informations renseignées ici doivent être strictement les mêmes que celles connues par les impôts pour que la télétransmission fonctionne.  
​  
Si vous choisissez de télétransmettre avec Indy, c'est Indy qui transmet l'ordre de paiement pour vous. Vous n'avez rien à faire de plus.  
​

💡 La télétransmission de la déclaration de TVA est possible, mais pas obligatoire.

8️⃣ Enfin, la dernière étape vous permet de voir un récapitulatif clair et synthétique de votre TVA due ou crédit de TVA.

Vous avez également la possibilité de voir la déclaration de TVA dans son intégralité en cliquant sur **Voir la déclaration** 👇  
​

[Image]

La déclaration est préparée à partir des taux de TVA que vous avez appliqué sur vos transactions et ce sur toute l'année, donc **il est essentiel de bien vérifier lesdites transactions ainsi que leurs taux de TVA 🤓**

Une fois toutes ces étapes complétées, vous pouvez **valider** la déclaration.

Si vous avez choisi la télétransmission vous recevrez un mail de confirmation **avec votre accusé de réception en pièce jointe.**

Autrement vous retrouverez la déclaration dans votre onglet **[Documents](https://app.indy.fr/documents)**.

Il faudra bien reporter les montants de votre déclaration sur le site des impôts via

👉 **impots.gouv > accès professionnel > déclarer > TVA.**

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite ! 😊
