# Peut-on modifier le code PIN ?

Le code PIN est attribué lors de la création de la carte physique. Il est généré automatiquement ou peut-être personnalisé par les clients **Plus** et **Premium** lors de la commande.

Une fois la carte créée, **ce code PIN ne peut** **pas être modifié**.

Si vous avez oublié votre code PIN, vous pouvez l'afficher depuis votre compte pro Indy.

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite 😉
