# Associer une transaction à un associé

Si votre société compte plusieurs associés, vous pouvez désormais préciser l’associé concerné pour chaque transaction impactant un compte courant d’associé directement depuis l’onglet Transactions.

16 mars 2026

Table des matières

Si vous avez plusieurs associés dans votre société, dès lors qu'une transaction impacte son compte courant d'associé, il est possible de préciser le nom de l'associé concerné directement dans votre onglet **[Transactions](https://app.indy.fr/transactions)**, au fil de l'eau 🙂

Il faudra ensuite sélectionner l'associé concerné par cette opération en cliquant sur le bouton `Compléter` 👇

[Image]

ℹ️ En cas d'erreur, vous pouvez à tout moment supprimer cette association en cliquant sur la petite croix, ou changer le nom de l'associé concerné en cliquant sur `Modifier` 👇

[Image]

La transaction sera alors **automatiquement attribuée** au bon associé dans l'étape de clôture **Comptes courants des associés**, venant mettre le solde du compte courant d'associé de l'associé concerné à jour 👇

[Image]

⚠️ Assurez-vous en amont d'avoir ajouté tous les associés de la société dans l'étape de clôture **Recenser les associés de la société** de votre **[Clôture comptable et fiscale](https://app.indy.fr/activities/bic-is-fiscal-declaration-activity)**.

Si ce n'est pas le cas, la liste des associés ne pourra pas s'afficher et Indy vous re-dirigera après avoir cliqué sur le bouton `Ajouter` 👇
