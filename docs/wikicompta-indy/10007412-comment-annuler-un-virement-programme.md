# Comment annuler un virement programmé ?

## Sur l'app :

Pour annuler un virement programmé, il suffit tout d'abord de se rendre sur l'onglet **Compte pro** puis cliquer sur **"Virements"** 👇

[Image]

Il suffit ensuite de cliquer sur "**Annuler le virement**" 👇

[Image]

Le virement disparaît alors des opérations à venir ✅

## Sur le web :

Pour annuler un virement programmé, il suffit tout d'abord de cliquer sur **"Virements"** 👇

[Image]

Cliquez ensuite sur le virement programmé que vous souhaitez annuler 👇  
​

[Image]

Cliquez sur "**Annuler le virement**", puis confirmer la suppression en cliquant sur "**Oui, annuler le virement**" 👇  
​

[Image]

[Image]

C'est tout bon, votre virement est annulé, il n'apparaît plus dans les virements à venir sur le menu "**Virements**"

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite ! 😊

À très vite 👋
