# À quelle heure sont débités mes prélèvements SEPA ?

Les prélèvements SEPA sont débités le jour où ils sont dus, **généralement vers 16h30.**

ℹ️ Si vous n'avez pas les fonds sur le compte à J-2, vous recevrez un email de notre part vous en informant, afin que vous puissiez alimenter votre compte dans les temps.

Attention, les rejets éventuels de prélèvements occasionneront des frais de notre partenaire Swan, ces frais seront débités 14 jours après le rejet de la transaction.

**Frais de rejet de prélèvement sortant pour solde insuffisant**

  
👉 10€ par prélèvement rejeté, dans la limite du montant du prélèvement rejeté
