# Étape de clôture d'Affectation du résultat N-1

Cette étape de clôture permet de préciser comment le résultat de l'exercice précédent a été affecté.

Plusieurs affichages sont possibles dans cette étape !

## 🥇 C'est le premier exercice

Bonne nouvelle, vous n'avez rien à faire dans cette étape puisque vous n'avez pas de résultat précédent à affecter ! 🎉

Vous pouvez **Valider** l'étape 👇

[Image]

## 🥈 Ce n'est pas le premier exercice mais le premier sur Indy

Lors de la création de votre compte Indy, vous avez effectué votre initialisation de comptabilité où vous avez indiqué le résultat de votre exercice précédent.

Indy récupère donc le résultat de votre exercice précédent de cette manière 🙂

Si vous êtes dans une situation de bénéfice, Indy vous propose différentes affectations possibles entre les associés et le report à nouveau 👇

[Image]

Si vous êtes dans une situation de perte, Indy affecte automatiquement la perte en report à nouveau débiteur 👇

[Image]

## 🥉 Ce n'est pas votre premier exercice sur Indy

Dans ce cas, Indy reprendra automatiquement les montants que vous avez affectés lors de la rédaction de [votre PV d'AG](https://wikicompta.indy.fr/fr/articles/10199667-approbation-des-comptes-et-redaction-du-pv-d-ag) 🎉

Vous avez juste à **Valider** l'étape.

Pour toute question supplémentaire, n'hésitez pas à nous contacter via la bulle rose en bas à droite 😉

À très vite ! 👋
