# Comment ouvrir un compte pro Indy ?

L'ouverture et la gestion de votre compte pro Indy sont 100% gratuits.

## Depuis l'interface web :

Rendez-vous sur l'onglet **Compte pro** et cliquez sur "**Créer mon compte pro gratuit**".

[Image]

  
​👉 Suivez les instructions, vous recevrez ensuite un lien par SMS pour finaliser la procédure de vérification de votre identité sur votre smartphone.
