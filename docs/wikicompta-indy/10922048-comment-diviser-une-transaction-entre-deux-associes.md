# Comment diviser une transaction entre deux associés ?

Il est fréquent que des associés possèdent un compte joint personnel et qu'ils l'utilisent pour faire des apports directement à la SCI ou régler certaines de ses dépenses.

Dans l'onglet **[Transactions](https://app.indy.fr/transactions)**, vous ne pouvez attribuer qu'un associé à chaque transaction qui affecte les comptes courants d'associés.
