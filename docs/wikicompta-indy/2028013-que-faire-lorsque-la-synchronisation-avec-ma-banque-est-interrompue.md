# Que faire lorsque la synchronisation avec ma banque est interrompue ?

Si depuis quelques jours les transactions effectuées avec votre compte professionnel n'apparaissent plus, c'est sûrement que la **connexion avec votre banque est coupée**.

Pour vous en assurer et comprendre la nature du problème, rendez-vous dans l'onglet **[Paramètres](https://app.indy.fr/compte/banques)** [> **Banques**](https://app.indy.fr/compte/banques).

  
Depuis cet onglet vous pourrez voir l'état de la connexion entre notre partenaire spécialisé dans la synchronisation bancaire et votre banque.

## **Les erreurs de connexion identifiables**

## Double Authentification

L'authentification forte est un moyen de sécuriser l'accès à vos informations bancaires.

Les interfaces bancaires réclament cette vérification tous les 90 jours (parfois plus fréquemment).

Pour vous reconnecter sur Indy, cliquez sur le bouton **Vérifier** et laissez vous guider par les indications de la fenêtre qui apparaîtra.

[Image]

Une fois dans l'onglet **[Banques](https://app.indy.fr/compte/banques)**, vous pouvez rapidement voir si la synchronisation de votre compte a été coupée.

Il faut alors valider un code reçu par SMS, une notification sur votre application bancaire, ou par le boîtier fourni par votre banque, pour prouver que c'est bien vous qui souhaitez accéder à vos données bancaires.

Ainsi tout le monde est serein 😌

⚠️ Veillez à ce que votre téléphone soit enregistré comme appareil de confiance sur votre espace bancaire pour réceptionner le SMS ou la notification

## Identifiants incorrects

S'il vous est précisé que vos identifiants sont incorrects, cliquez sur le bouton **Reconnecter.** Saisissez de nouveau vos identifiants en faisant attention aux majuscules et aux chiffres que vous saisissez 🙏

[Image]

  
​Si vous n'êtes pas certains de vos identifiants, n'hésitez pas à vous connecter sur votre banque en ligne afin de les tester.  
​  
Vous vérifierez par la même occasion que vous n’avez ni message à valider ni confirmation à opérer sur le site de votre banque.

Cela peut également bloquer votre connexion 😉

## Erreur interne

Pour rétablir la synchronisation bancaire, cliquez sur le bouton **Réessayer**.  
Si l'erreur subsiste : essayez à nouveau dans 2-3 jours.

Il y a parfois des mises à jour et les agrégateurs bancaires peuvent avoir besoin de quelques jours pour actualiser le connecteur.

[Image]

  
​Si le problème persiste après quelques jours, venez nous en parler afin que la situation soit résolue dans les plus brefs délais 🚀

##   
**La synchronisation est coupée depuis plusieurs mois**

Vous venez de vous connecter à votre compte Indy après plusieurs mois sans y être allé et vous vous apercevez que votre compte n'est plus synchronisé. Que faire ?

Dans un premier temps il faudra vous rendre dans l'onglet **[Paramètres](https://app.indy.fr/compte/banques)** [>](https://app.indy.fr/compte/banques) **[Banques](https://app.indy.fr/compte/banques)** pour suivre les instructions décrites au-dessus ☝️

Ensuite il sera nécessaire de venir sur le live-chat afin de nous envoyer vos e-relevés originaux car la synchronisation ne permet d'importer que les transactions des dernières semaines.

Grâce à vos relevés nous pourrons importer toutes les transactions manquantes 💪

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite ! 😊

À très vite 👋
