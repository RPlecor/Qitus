# Quel est le délai de réception d'un paiement en ligne de facture ?

Lorsqu'un client règle votre facture par carte via un lien de paiement en ligne, la transaction s'affiche dans un premier temps dans les **Transactions à venir** de votre onglet **Transactions.**

Les fonds sont ensuite **réceptionnés le jour ouvré suivant** **le paiement**.

Pour vous protéger contre les risques de fraude, les fonds sont **mis sous réserve pendant 5 jours ouvrés après réception.** Ils seront crédités dans votre solde disponible à l'issue de ce délai.
