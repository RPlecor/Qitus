# Ventiler les cotisations Urssaf des auto-entrepreneurs

## Catégoriser mes cotisations et mes remboursements

Les cotisations versées à l'Urssaf sont à catégoriser en **Cotisation sociale Urssaf**.

Les remboursements que vous recevez sont à catégoriser en **Cotisation sociale Urssaf** 🤓

## Ventiler mes cotisations

Une fois que toutes vos cotisations et tous vos remboursements sont bien catégorisés, vous pouvez vous rendre dans [notre étape dédiée](https://app.indy.fr/activities/d-2035-activity/urssaf) afin de ventiler vos cotisations.

Indy vous pose cette question pour commencer 👇

[Image]

2 situations sont possibles pour vous :

Si vous aviez le **statut auto-entrepreneur durant toute l'année fiscale,** cochez **Oui.**

L'étape sera validée automatiquement ✅

Si vous êtes dans le 2ème cas : **auto-entreprise avec changement de statut**, cochez **Non**.

Il faudra ensuite fonctionner en deux temps :

**1️⃣** Intégrer le versement libératoire de l'impôt

Si vous êtes soumis au **versement libératoire de l’impôt**, vos transactions sont à **subdiviser**. Cet article vous indique comment [subdiviser une transaction](https://wikicompta.indy.fr/fr/articles/451550-comment-subdiviser-une-transaction).

Subdivision 1 👉 Montant d’impôt à catégoriser en **Dépense personnelle**  
Subdivision 2 👉 Montant d’Urssaf à catégoriser en **Cotisation sociale Urssaf**

**2️⃣** Remplir l'étape Urssaf

Il faudra sélectionner la **saisie d’échéancier.** En tant qu'auto-entrepreneur, vous n’avez pas besoin de répartir les différentes cotisations 🤓

Reportez le montant de chaque transaction en **Autre cotisation obligatoire (+ Ajouter une cotisation) ✅**
