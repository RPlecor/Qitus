# Dans quelle rubrique catégoriser une transaction ?

Vous avez une transaction à classer sur Indy. Découvrez comment choisir la bonne rubrique et catégoriser correctement vos opérations comptables.

Mis à jour il y a plus de 2 semaines

Table des matières

Avec Indy, 90% de vos transactions sont catégorisées **automatiquement 🙌** Certaines d'entre-elles, plus délicates, ou dont l'intitulé n'est pas assez précis, peuvent échapper au premier niveau de catégorisation.

## Transactions à catégoriser

Vous pouvez facilement visualiser le nombre de transactions sans catégorie sur Indy.

Depuis votre onglet **Transactions**, cliquez sur **Classer ces transactions** pour accéder à la liste des écritures qu'il vous reste à catégoriser **👇**

[Image]

En cliquant sur **À catégoriser** pour l'une des transactions concernées, le tableau des catégories va s'afficher 👇

[Image]

Si vous trouvez la catégorie voulue, il ne vous reste plus qu'à cliquer dessus 😉

## Les suggestions

En dessous de chaque transaction **À catégoriser**, Indy tente de suggérer les deux catégories les plus pertinentes, en fonction du libellé de la transaction et de vos habitudes de catégorisation.

[Image]

💡 Si aucune ne convient, cliquez sur **À catégoriser** afin d'attribuer la bonne catégorie depuis le tableau des catégories.

## Rechercher une catégorie

Si vous n'arrivez pas à trouver la catégorie que vous voulez, utilisez la barre de recherche et Indy **surlignera le mot** que vous avez saisi en dessous de la bonne catégorie 👇

[Image]

Vous pouvez également passer votre curseur sur chacune des catégories pour obtenir une description plus précise de son contenu en info-bulle 👇

[Image]

## Exemples d’opérations fréquentes

|     |     |
| --- | --- |
| **Exemple d'opérations** | **Catégorie comptable à appliquer** |
| Frais de création d'entreprise ou frais de Greffe | **Frais d'acte et de contentieux** |
| Abonnement Indy | **Abonnement logiciel** |
| Honoraires d'un expert-comptable | **Honoraires divers** |
| Frais de repas dans le cadre d'un déplacement, d'une formation, d'un déjeuner client | **Réception et congrès** |
| Frais de repas quotidien (si vous déjeunez seul) | **Restaurant et repas d'affaires** |
| Loyer au titre de la domiciliation ou d'une quote-part | **Loyers et charges locative** |
| Frais de formation | **Honoraires divers** |
| Réalisation d'une tâche que vous ne pouvez pas faire en interne, confiée à une autre entreprise | **Prestation de service** |
| Réalisation d'une tâche que vous pourrez faire en interne, confiée à une autre entreprise | **Sous-traitance** |
| Location Longue Durée (LLD) | **Location de matériel** |
| Location avec Option d'Achat (LOA) | **Crédit-bail véhicule** |
| Consommables alimentaires type **boissons**, **café, viennoiseries** | **Frais divers** |
| Petites fournitures de bureau et consommables type **mouchoirs**, ramettes de papier pour l’**imprimante** | **Frais divers** |
| Consommations d’énergie type eau, gaz, **électricité** | **Eau, gaz, électricité** |
| Abonnement à des logiciels ou plateformes type **Youtube**, Indy, **OVH** | **Abonnement logiciel** |

## Les dépenses depuis un compte bancaire personnel

Les dépenses que vous effectuez depuis un compte bancaire personnel sont à ajouter manuellement sur Indy et à catégoriser **selon la nature de la dépense**.

Par exemple, vous payez une facture d'un logiciel de gestion depuis votre compte bancaire personnel, vous devez alors ajouter une note de frais sur Indy.

👉 Par nature, et de manière automatique, la note de frais créée vient impacter votre compte courant d'associé. Il ne faut donc pas faire de doublon en catégorisant cette transaction en **Apport/Remboursement d'associé**, mais bien sélectionner **Abonnement logiciel**.
