# J'ai effectué un retrait mais je n'ai pas pu récupérer les billets ?

Si vous avez **effectué un retrait d'argent en distributeur automatique de billet** (DAB), que celui a fonctionné mais que vous n'avez finalement **pas pu récupérer la somme**, il convient de réaliser une contestation de l'opération 👇

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite ! 😊
