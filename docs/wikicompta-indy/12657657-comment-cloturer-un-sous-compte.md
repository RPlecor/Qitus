# Comment clôturer un sous-compte ?

**Le solde de votre sous-compte doit être à 0** avant de pouvoir être clôturé.

👉 S'il reste des fonds sur votre sous-compte, il vous faudra donc d'abord les transférer vers votre compte principal.

Une fois le solde du sous-compte à 0, vous pouvez le clôturer depuis **[la version Web](https://app.indy.fr/)** \- la clôture n'est pas possible depuis l'application mobile. Il vous suffit :

[Image]

Vous recevrez ensuite un lien par SMS pour confirmer votre demande.

Une fois confirmée, la clôture du sous-compte sera **effective sous un délai maximum de 31 jours**. Pendant ce délai, votre sous-compte restera visible avec le statut "**En fermeture**".

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite 😉

À très vite !
