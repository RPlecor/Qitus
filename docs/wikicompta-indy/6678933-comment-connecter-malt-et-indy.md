# Comment connecter Malt et Indy ?

Vous souhaitez bénéficier du partage de données entre Indy et Malt ? Cet article vous explique comment faire pour en profiter.

16 février 2026

Pour connaître les avantages liés à l'intégration Malt sur Indy, vous pouvez vous rendre sur **[cet article](https://wikicompta.indy.fr/fr/articles/6671602-integration-automatique-avec-malt)**.

Afin d'activer la connexion entre vos deux outils, il faut vous rendre dans l'onglet **Paramètres**, dans la rubrique **Options** puis **Intégration avec Malt** 👇

[Image]

Ensuite, vous avez simplement à suivre les instructions qui sont mentionnées 🤓

En cliquant sur le lien **[Votre compte Malt](https://www.malt.fr/signin)**, depuis Indy, vous arriverez directement sur la page vous permettant de récupérer les informations nécessaires. Voici le chemin à suivre ensuite :

💡 Depuis peu, vous devez maintenant également ajouter les autorisations suivantes à vos tokens :

[Image]

Via cette procédure, vous avez donc récupéré un Token.

C'est ce dernier qui va permettre d'intégrer vos données, de Malt vers Indy.

👉 Il faut par conséquent, copier/coller ce code sur Indy pour activer le lien.

La validation peut prendre quelques minutes, soyez patients !

Une fois que tout est en ordre, vous verrez ce message dans l'onglet 👇
