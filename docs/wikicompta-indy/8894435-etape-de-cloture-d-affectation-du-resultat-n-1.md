# Étape de clôture d'Affectation du résultat N-1

Cette étape de clôture va permettre de préciser comment le résultat de l'exercice précédent a été affecté.

Vous allez rencontrer plusieurs possibilités d'affichage dans cette étape !

## 🥇 C'est le premier exercice

Bonne nouvelle, vous n'avez rien à faire dans cette étape puisque vous n'avez pas de résultat précédent à affecter ! 🎉

Vous pouvez **Valider** l'étape 👇

[Image]

## 🥈 Ce n'est pas le premier exercice mais le premier sur Indy

Lors de la création de votre compte Indy, vous avez dû réaliser une initialisation de comptabilité si vous aviez un historique comptable.

_Pour avoir un rappel sur cette étape, c'est par [ici](https://wikicompta.indy.fr/fr/articles/8820390-l-initialisation-de-votre-comptabilite-reprise-de-votre-historique-comptable) !_

Indy récupère donc le résultat de votre exercice précédent de cette manière 🙂

Si vous êtes dans une situation de bénéfice, Indy vous propose les réserves suivantes 👇

## 🥉 Ce n'est pas votre premier exercice sur Indy

Dans ce cas, Indy reprendra automatiquement les montants que vous avez affectés lors de la rédaction de votre PV d'AG 🎉

Vous avez juste à **Valider** l'étape.

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite 😉
