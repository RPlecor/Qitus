# Où trouver mon code PIN ?

Vous pouvez afficher le code PIN de votre carte physique depuis **l'application mobile** ou **[l'interface Web](https://app.indy.fr/)** d'Indy.

## Depuis votre app Indy

1️⃣ Rendez-vous sur l'onglet "**Compte pro**" de votre application et sélectionnez votre carte physique :

[Image]

  
2️⃣ Cliquez sur le bouton **⚙️ Paramètres** sous la carte correspondante :

[Image]

3️⃣ Cliquez sur "**Afficher le code PIN**" (_une authentification vous sera demandée pour sécuriser l'accès au code PIN_) 👇

[Image]

## Depuis la [version web d'Indy](https://app.indy.fr/)

1️⃣ Sous l'onglet "**Compte pro"** dans la section "**Cartes"**, cliquez sur votre carte physique :  
​

[Image]

2️⃣ Cliquez ensuite sur "**Afficher le code PIN**" 👇

[Image]

3️⃣ Vous recevrez ensuite un lien par SMS pour réaliser l'authentification forte depuis votre smartphone 👇  
​

[Image]

4️⃣ Après avoir cliqué sur le lien reçu par SMS, un écran vous demandera de confirmer que vous souhaitez afficher votre code PIN. Cliquez sur "**Confirmer**" 👇

[Image]

**5️⃣ Procédez à l'authentification forte** via le mode installé sur votre smartphone par défaut (Face ID, Touch ID, empreinte, code).  
​  
​**Le code PIN de votre carte sera révélé uniquement sur l'écran de votre smartphone** 👇

[Image]

  
​
