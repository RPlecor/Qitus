# Récupérer mes données bancaires sur la version gratuite

Lorsque vous synchronisez votre compte sur Indy, nous récupérons automatiquement les transactions bancaires associées.

Généralement la récupération de ces données est limitée à quelques mois dans le passé : c'est une limite de votre banque, qui n'autorise pas toujours qu'un historique complet soit récupéré.

💡 **À noter**

Chez certaines banques cette reprise est complète, dans ce cas vous aurez tout votre historique bancaire sur Indy.

Dans certains cas, vous avez besoin de faire ce qu'on appelle **une reprise d'historique** : c'est tout simplement une reprise de vos données bancaires jusqu'au début de votre exercice fiscal 😊

## Comment faire pour reprendre mon historique ?

Pensez à envoyer vos relevés bancaires au format PDF pour que nous puissions faire le nécessaire 🙏

⚠️ Sachez que la reprise d'historique est **limitée à l'année en cours**.

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite 😉

À très vite ! 👋
