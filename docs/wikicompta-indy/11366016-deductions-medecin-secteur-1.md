## **La déduction complémentaire de 3%**

Il est également possible de cumuler une déduction forfaitaire complémentaire de **3%**.

⚠️ Celle-ci est calculée uniquement sur vos **honoraires conventionnés.**

C'est pour cette raison que l'étape de clôture vous demandera de préciser le montant de vos honoraires conventionnés (s'il est différent du total de vos recettes) ⬇️
