# Comment programmer l'émission d'une facture ?

Vous avez besoin d'émettre une facture à une date future - pour anticiper une prestation, respecter un échéancier ou préparer votre facturation à l'avance ? Sur Indy, les factures avec une date d'émission dans le futur sont automatiquement **programmées** : elles ne sont pas émises immédiatement, mais à la date que vous avez choisie 📆

## Retrouver vos factures programmées

Vos factures planifiées en attente d'émission sont regroupées dans la section dépliable **"Planifiées"**, en haut de votre liste de factures. Le compteur indique combien de factures sont en attente.

## Obtenir un document avant l'émission : la facture proforma

Avant la date d'émission, votre facture n'est pas encore officielle. Vous pouvez cependant télécharger un **proforma** : un aperçu non numéroté portant un filigrane "PROFORMA".
