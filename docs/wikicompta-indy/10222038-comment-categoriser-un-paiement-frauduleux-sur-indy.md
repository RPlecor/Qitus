# Comment catégoriser un paiement frauduleux sur Indy ?

## 1️⃣ Faire opposition à votre carte bancaire

Si vous soupçonnez un paiement frauduleux sur votre compte bancaire professionnel, le premier réflexe à avoir est de **faire opposition à votre carte bancaire**.

👉 Pour cela, rapprochez-vous directement de votre banque.

## 2️⃣ Porter plainte

Pour qu’une dépense frauduleuse puisse être considérée comme déductible, il est impératif de **porter plainte auprès d’un commissariat de police** 🚓

Sans dépôt de plainte, la dépense ne pourra pas être déduite de votre comptabilité professionnelle ❌

## 3️⃣ Demander le remboursement du paiement frauduleux

Une fois la plainte déposée, vous pouvez **demander le remboursement du paiement frauduleux** à votre banque 🏦
