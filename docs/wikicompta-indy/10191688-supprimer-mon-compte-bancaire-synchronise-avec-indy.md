# Supprimer mon compte bancaire synchronisé avec Indy

La suppression d'un compte bancaire consiste à **supprimer** **définitivement** les les transactions, des justificatifs ainsi que de l’ensemble des données liées à ce compte. Cette action est **irréversible !**
