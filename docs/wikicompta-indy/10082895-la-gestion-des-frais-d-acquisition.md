# La gestion des frais d'acquisition

Souvent appelés frais de notaire, les frais d'acquisition font partie intégrante des coûts globaux nécessaire pour acquérir le bien 🤓

Il existe alors deux méthodes de retraitement de ces frais d'acquisition dans votre comptabilité :

Choisissez l'option qui vous convient le mieux !

Suivez ensuite ce guide pour les comptabiliser correctement selon si :

## Les frais d'acquisition sont payés directement depuis le compte bancaire de la SCI

Dans ce cas, la transaction bancaire est visible sur le compte en banque 🤓

## 1️⃣ Si vous souhaitez déduire les frais d'acquisition

👉 Catégorisez la transaction en **Achat d'un bien** _(colonne Fonctionnement)_.

👉 Comme vous venez d'acquérir un bien immobilier, il sera également nécessaire de l'ajouter sur Indy dans [l'étape de clôture](https://wikicompta.indy.fr/fr/articles/8688417-etape-de-cloture-des-biens-immobiliers) **[Biens immobiliers](https://wikicompta.indy.fr/fr/articles/8688417-etape-de-cloture-des-biens-immobiliers)**.

À l'ajout du bien dans cette étape, une fenêtre apparait pour ajouter les montants relatifs à l'acquisition du bien, dont une case **Frais d'acquisition** 👇

[Image]

## 2️⃣ Si vous souhaitez amortir les frais d'acquisition

👉 Catégorisez la transaction en **Achat d'un bien** _(colonne Fonctionnement)_.

À l'ajout du bien dans cette étape, une fenêtre apparait pour ajouter les montants relatifs à l'acquisition du bien, dont une case **Frais d'acquisition** 👇

[Image]

## Les frais d'acquisition ne sont pas payés depuis le compte bancaire de la SCI

C'est le cas le plus courant _(les frais sont payés directement via le notaire ou la banque par exemple)_ 🤓

Dans cette configuration, aucune transaction bancaire n'est alors visible sur le compte en banque de la SCI !

## 1️⃣ Si vous souhaitez déduire les frais d'acquisition

De nouveau, comme vous venez d'acquérir un bien immobilier, il sera également nécessaire de l'ajouter sur Indy dans [l'étape de clôture](https://wikicompta.indy.fr/fr/articles/8688417-etape-de-cloture-des-biens-immobiliers) **[Biens immobiliers](https://wikicompta.indy.fr/fr/articles/8688417-etape-de-cloture-des-biens-immobiliers)**.

À l'ajout du bien dans cette étape, une fenêtre apparait pour ajouter les montants relatifs à l'acquisition du bien, dont une case **Frais d'acquisition** 👇

[Image]

## 2️⃣ Si vous souhaitez amortir les frais d'acquisition

De nouveau, comme vous venez d'acquérir un bien immobilier, il sera également nécessaire de l'ajouter sur Indy dans [l'étape de clôture](https://wikicompta.indy.fr/fr/articles/8688417-etape-de-cloture-des-biens-immobiliers) **[Biens immobiliers](https://wikicompta.indy.fr/fr/articles/8688417-etape-de-cloture-des-biens-immobiliers)**.

À l'ajout du bien dans cette étape, une fenêtre apparait pour ajouter les montants relatifs à l'acquisition du bien, dont une case **Frais d'acquisition** 👇

[Image]

**ℹ️ Vous avez payé les frais d'acquisition directement depuis votre compte bancaire personnel ?**

Dans ce cas, n'oubliez pas d'ajouter une note de frais, à catégoriser en **Achat d'un bien**, puis à ajouter ce montant dans l'étape de clôture des Biens immobiliers selon votre méthode de comptabilisation.

_Pour en savoir plus sur les notes de frais, c'est par [ici](https://wikicompta.indy.fr/fr/articles/6611638-comment-ajouter-une-transaction-qui-n-apparait-pas-sur-votre-compte) !_

## Amortir ou déduire : que choisir ?

Vous pouvez estimer que le paiement des frais d'acquisition engage la SCI sur le long terme, et qu'à ce titre il est préférable d'amortir.

Ou bien, que c'est une dépense liée à l'activité courante de la SCI, et qu'à ce titre il est préférable de la passer en charge.
