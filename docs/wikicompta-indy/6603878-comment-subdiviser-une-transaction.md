# Comment subdiviser une transaction ?

C'est très simple vous allez voir 🤗

## **Ajouter une ventilation**

En cliquant sur la transaction concernée, la page de subdivision suivante va s'afficher👇

[Image]

Dans la colonne de droite, vous pouvez cliquer sur **\+ Ajouter une ventilation** pour subdiviser la transaction.

Par défaut, la ventilation 2 n'est pas catégorisée c'est à vous de choisir la seconde catégorie.

[Image]

Une fois la ventilation renseignée, il ne vous reste plus qu'à cliquer sur **Appliquer** en haut à droite de la page 🎉  
​

_Dans l'exemple ci-dessus, le loyer se décompose en 750 € de loyer hors taxes, et de 250 € de provisions pour charges locatives._

_Le locataire a payé directement un montant total de 1 000 €. Il est donc nécessaire de subdiviser la transaction._

## **Supprimer une ventilation**

Si vous souhaitez annuler/supprimer une subdivision, vous devez le faire manuellement. Pour cela, retrouvez la transaction concernée dans l'onglet **Transactions** et cliquez de nouveau sur celle-ci afin d'ouvrir la fenêtre de ventilation 👇

[Image]

Cliquez ensuite sur l'icône corbeille pour supprimer la ventilation souhaitée 👇

[Image]

Enfin, pour que votre transaction retrouve son montant global, cliquez sur la baguette magique à droite du montant de la ventilation restante.  
La subdivision est désormais annulée 🎉

[Image]

  
Il ne vous reste plus qu'à cliquer sur **Appliquer** pour enregistrer cette annulation 👍
