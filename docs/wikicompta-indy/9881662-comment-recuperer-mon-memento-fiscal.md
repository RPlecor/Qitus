# Comment récupérer mon mémento fiscal ?

C'est un document récapitulatif où figurent les informations générales de votre entreprise, sa situation juridique ou ses obligations fiscales 🤓

Il est très utile pour vous assurer que votre compte Indy est bien paramétré !

Si vous n'avez plus votre mémento fiscal sous la main, voici un tutoriel vidéo pour le récupérer auprès de votre SIE 👇

⚙️ **Quelle est la marche à suivre pour récupérer votre mémento fiscal ?**

Rendez-vous sur **impots.gouv.fr/contacts** ;

Cliquez sur **Professionnel** ;

Puis sur **Une question fiscale d'ordre général** ;

Saisissez **l'adresse de votre société** et validez ;

Envoyez un **email à votre SIE** à l'adresse indiquée ✅

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite 😉

À très vite !
