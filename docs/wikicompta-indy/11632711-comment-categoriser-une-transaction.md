# Comment catégoriser une transaction ?

Comment catégoriser les transactions bancaires de votre LMNP ?

2 février 2026

Table des matières

Avec Indy, **90% de vos transactions sont catégorisées automatiquement** 💃

Certaines transactions parmi les plus délicates peuvent échapper au premier niveau de catégorisation c'est alors à vous de trouver la bonne catégorie ✨

## **Transactions à catégoriser**

Vous pouvez facilement voir le nombre de transactions qui n'ont pas encore de catégorie attribuée sur Indy ; depuis votre onglet **[Transactions](https://app.indy.fr/transactions)**, il vous suffit de cliquer sur **Catégoriser ces transactions** pour accéder à la liste des écritures **À catégoriser 👇**

[Image]

## **Rechercher une catégorie**

Si vous ne savez pas quelle catégorie appliquer à une transaction, vous pouvez utiliser la **barre de recherche** en tapant le **mot-clé** auquel se réfère votre transaction et vous devriez trouver la catégorie adaptée👇

[Image]

Si vous avez un doute sur une catégorie, vous pouvez passer votre curseur sur chacune des catégories pour obtenir une description plus précise de son contenu.

## **Les dépenses depuis un compte bancaire personnel**

_Par exemple :_

_Je paie les honoraires de l'agence immobilière que j'ai engagée pour la gestion de mon bien depuis mon compte bancaire personnel, je dois alors créer **une dépense sur compte personnel.**_

**👉 _Par nature_** _la dépense créée vient impacter automatiquement **[le compte de l'exploitant](https://www.indy.fr/guide/tenue-comptable/plan-comptable/compte-classe-un/compte-108/)** pour la comptabiliser correctement. Il ne faut donc pas faire de doublon en catégorisant cela en **Apport/Prélèvement personnel**, je viens plutôt sélectionner **Honoraires divers.**_
