# Comment supprimer un bénéficiaire ?

Vous pouvez désormais supprimer un bénéficiaire déjà enregistré lors de précédents virements.

## **Depuis la version web d'Indy**

Pour supprimer un bénéficiaire, rendez-vous sur la version web d'Indy dans l'onglet **Compte pro**, puis cliquez sur "**Virements**" 👇

[Image]

Ensuite vous pouvez cliquer sur l'onglet "**Bénéficiaires**", en cliquant sur les trois petits points suivant le bénéficiaire en question, vous pourrez cliquer sur "**Supprimer**"👇

[Image]

## **Depuis l'app d'Indy**

**Depuis votre app Indy**

Pour supprimer un nouveau bénéficiaire, rendez-vous sur l'application Indy à l'onglet **Compte pro**, puis cliquez sur "**Virements**" 👇

[Image]

Ensuite vous pouvez cliquer sur l'onglet "**Bénéficiaires**" 👇

[Image]

Il suffit de faire **glisser le bénéficiaire vers la gauche** pour voir apparaître le bouton de suppression 👇  
​

[Image]

👉 Confirmez ensuite la suppression en cliquant sur "**Supprimer**"

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite 😉

À très vite ! 👋
