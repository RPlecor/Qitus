# Comment modifier mon RIB pour la télétransmission de ma déclaration de TVA ?

Vous avez reçu un mail de refus de votre déclaration de TVA à cause de l'IBAN renseigné, ou vous souhaitez tout simplement changer le compte bancaire utilisé pour le prélèvement de votre TVA ?

Pas d'inquiétude, vous pouvez modifier votre RIB directement via l'activité **Déclaration de TVA** sur Indy 🤓

## Pourquoi l'IBAN renseigné sur Indy doit-il être identique à celui des impôts ?

Lors de la télétransmission de votre déclaration de TVA, Indy envoie un **ordre de paiement** au Service des Impôts des Entreprises (SIE). Pour que cet ordre soit validé, l'IBAN renseigné dans votre activité de **Déclaration de TVA** doit être **strictement identique** à celui enregistré sur votre espace impots.gouv 🪞

💡 Un seul caractère qui change (espace, majuscule, code BIC) suffit à entraîner un rejet de la déclaration. Pensez à bien recopier l'IBAN caractère par caractère.

## Comment modifier mon RIB sur Indy ?

Une fois l'IBAN des impôts en main, vous pouvez corriger celui renseigné dans votre déclaration de TVA sur Indy.

1️⃣ Rendez-vous dans l'onglet **[À faire](https://app.indy.fr/activites)**.

2️⃣ Si vous n'avez pas encore validé votre activité **Déclaration de TVA**, celle-ci est normalement ouverte. Si vous avez eu un refus de votre déclaration de TVA, contactez-nous via le live-chat afin que nous puissions rouvrir votre activité **Déclaration de TVA**.

3️⃣ Dans l'étape **Télétransmission de votre déclaration** rendez-vous dans la section **Coordonnées bancaires** 👇

[Image]

4️⃣ Modifiez sur le RIB en saisissant l'IBAN ainsi que le BIC dans les champs prévus à cet effet.

5️⃣ Cochez la case _Je confirme que les informations ci-dessus sont celles enregistrées sur mon espace professionnel impots.gouv._ puis validez la modification.

Une fois le RIB mis à jour, vous pouvez **télétransmettre votre déclaration de TVA** directement depuis votre activité ✅

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite 😉
