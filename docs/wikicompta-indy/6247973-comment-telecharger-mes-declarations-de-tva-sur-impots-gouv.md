# Comment télécharger mes déclarations de TVA sur impots.gouv ?

Vos déclarations de TVA peuvent vous être demandées. Vous retrouverez dans cet article la marche à suivre pour télécharger vos déclarations de TVA depuis votre espace professionnel impots.gouv 💪

2️⃣ Ensuite, saisissez votre **adresse électronique** et **votre mot de passe**.  
​

[Image]

3️⃣ Puis, cliquez sur **Compte Fiscal**.

[Image]

4️⃣ Si vous avez plusieurs dossiers, choisissez le bon et cliquez sur **Consulter**.

[Image]

5️⃣ Pour afficher vos déclarations, vous pourrez alors cliquer sur le menu déroulant **Accès par impôts > TVA et taxes assimilées > Liste des références des obligations fiscales et secteurs d'activité**.

[Image]

6️⃣ Si vous avez plusieurs références d'obligations fiscales ou activités, il faudra sélectionner la bonne ligne dans la liste et cliquer sur **Déclarations.**

[Image]

7️⃣ Enfin, vous pourrez télécharger les déclarations qui vous intéressent dans la colonne **Télécharger formulaires**.

[Image]

Si besoin, pour les envoyer à un conseiller Indy, il faudra cliquer sur le petit trombone 📎 dans la fenêtre de conversation.

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite 😊

À très vite ! 👋
