# Guide pas à pas : compléter la 2044

Le formulaire 2044 sert à déclarer les revenus fonciers des associés d’une SCI relevant de l’impôt sur le revenu. Il s’adresse aux propriétaires de biens immobiliers mis en location ayant opté pour le régime réel d’imposition, par opposition à ceux relevant du régime micro-foncier.

Ce formulaire doit être rempli chaque année, en même temps que la déclaration de revenus. Il accompagne la déclaration d’impôt sur le revenu personnelle, à savoir le formulaire 2042.

Voici les étapes, pas à pas, pour vous accompagner dans le remplissage du formulaire 2044 lors de votre déclaration de revenus.

## Étape 1 : activer l'annexe 2044

Compléter les deux premières étapes de la déclaration d'impôt sur le revenu :

Vous arriverez ensuite à l'étape 3 : **Revenus et charges**.

[Image]

Sur cette étape, vous pouvez cocher toutes les parties que vous souhaitez faire apparaître dans votre déclaration.

Pour déclarer des revenus fonciers issus de votre SCI, il faudra cocher : **Revenus fonciers** _Location non meublée_ 👇

[Image]

Lorsque vous cochez cette case, une autre fenêtre apparait et vous pourrez y cocher : **Déclaration des revenus fonciers 20XX N°2044** puis **Valider.**

[Image]

Cochez toutes les autres cases dont vous avez besoin à titre personnel pour votre déclaration d'impôt puis validez ce premier panneau.

## Étape 2 : accéder à l'annexe 2044

Une fois ce panneau validé, vous accédez à la première source de revenus à renseigner pour votre déclaration.

  
Cliquez directement dans la colonne de gauche pour compléter l'annexe 2044 👇

### C. 410 Intérêts d'emprunt

Cette case concerne uniquement les biens détenus en nom propre et pas les SCI. Vous pouvez simplement cliquer sur **Suivant** sans compléter les éléments.

### E. 450 Déficits antérieurs restants à imputer

Si vous n'avez pas de déficit reportable, vous pouvez simplement cliquer que **Suivant**.

## Étape 3 : Revenus fonciers _location non meublée_

Une fois que vous aurez complété tous les éléments de la 2044, vous rebasculerez sur la déclaration d'impôt principale.

Vous y trouverez notamment la partie **Revenus fonciers** _location non meublée_ 👇

[Image]

En ayant complété l'annexe 2044 en premier lieu, **cette partie est pré-remplie** et vous pouvez simplement cliquer sur **Suivant** en bas de la page.
