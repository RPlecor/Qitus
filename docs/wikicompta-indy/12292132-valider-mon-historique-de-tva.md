# Valider mon historique de TVA

Dès la création de votre compte, Indy vous demande d’importer vos déclarations de TVA déjà effectuées avant votre inscription sur Indy 👇

[Image]

👉 Grâce à cette étape, Indy peut vérifier la cohérence de vos déclarations passées et identifier rapidement d’éventuelles anomalies.

_💡 Pour l'instant, les déclarations importées via cette activité **ne s'affichent pas dans l'onglet Documents**._

## Comment ça fonctionne ?

Lors de cette étape, vous allez importer vos déclarations de TVA déjà réalisées sur l’exercice en cours.

Pour cela, vous avez le choix entre deux méthodes :

[Image]

## Récupération automatique

Il faudra renseigner **les identifiants de votre compte professionnel impôts.gouv**. Indy pourra alors visualiser les déclarations de TVA déjà déposées sur le site des impôts, puis les télécharger en lecture seule 👇

[Image]

Une fois que toutes les déclarations sont récupérées automatiquement, **valider l'étape afin de passer à l'analyse de votre historique.**

## Ajout manuel

Dans cette option, c'est à vous de joindre manuellement les déclarations de TVA déjà déposées sur le site des impôts :

[Image]

_Indy ne pré-remplit pas la liste des déclarations de TVA attendues → cela permet de vérifier que votre paramétrage TVA est correct._

⚠️ Si des déclarations sont manquantes, elles apparaissent en **orange**.  
Ajoutez simplement le ou les fichiers manquants via le bouton **\+ Ajouter** 👇

[Image]

Une fois que vous avez téléchargé toutes les déclarations déjà déposées sur le site des impôts sur l'exercice en cours, **valider l'étape afin de passer à l'analyse de votre historique.**

## Et si je n’ai aucune déclaration à importer ?

Si vous n’avez effectivement aucune déclaration passée (par exemple si vous êtes en TVA annuelle **CA12** et que vous n’avez pas encore déclaré), vous pouvez cliquer sur **“Je n’ai aucune déclaration à importer”** 👇

[Image]

Une modale de confirmation s'affiche ensuite. Cela n'empêchera toutefois pas Indy d'analyser ensuite votre historique de TVA 😉

## Analyse de votre historique

Une fois que toutes les déclarations de TVA sont récupérées, Indy les analyse automatiquement pour vérifier :

Deux résultats possibles :

Vous pouvez avancer sereinement vers votre première déclaration avec Indy. **Terminer** alors l'activité.

Dans ce cas, Indy vous liste et vous explique les anomalies détectées. Il faudra alors les corriger avant de pouvoir faire votre première déclaration.

Cliquez alors sur **Demander la résolution** 👇

[Image]

Une modale s'affiche vous demandant de nous envoyer votre **mémento fiscal** et une capture d'écran de votre page du site des impôts **Déclarer → TVA** 👇

[Image]

Ces documents seront **essentiels** et **indispensables** pour résoudre l'anomalie.

_Si vous avez besoin d'aide pour retrouver votre mémento fiscal, [cliquez ici](https://wikicompta.indy.fr/fr/articles/9881662-comment-recuperer-son-memento-fiscal)._

_Si vous avez besoin d'aide pour retrouver la page du site des impôts Déclarer → TVA, [cliquez ici](https://wikicompta.indy.fr/fr/articles/10067493-acceder-aux-services-de-declaration-de-tva-sur-son-espace-des-impots)._

Cliquez ensuite sur **Valider**. Notre équipe experte analysera votre dossier, corrigera l’anomalie et mettra à jour votre activité.

Une fois l'anomalie résolue, vous serez prévenu par e-mail et vous pourrez alors reprendre votre parcours normalement 🙂

N'hésitez pas à revenir vers nous via le live-chat en bas à droite si cet article n'a pas répondu à toutes vos questions 😇

À très vite !
