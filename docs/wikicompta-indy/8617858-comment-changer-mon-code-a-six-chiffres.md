# Comment changer mon code à six chiffres ?

Si vous avez oublié ce code ou que vous souhaitez le modifier :

1️⃣ Lancez une action sur votre compte pro Indy nécessitant votre code.  
👉 Vous pouvez par exemple faire comme si vous [ajoutiez un nouveau bénéficiaire](https://wikicompta.indy.fr/fr/articles/11582853-comment-ajouter-un-beneficiaire).

2️⃣ Sur la fenêtre vous invitant à saisir votre code, cliquez sur "**J'ai oublié mon code d'accès**".

[Image]

3️⃣ Cliquez sur le bandeau fuchsia "**Réinitialiser**".

4️⃣ Choisissez un **nouveau code de validation à six chiffres** (différent de votre date de naissance).

5️⃣ Vous allez maintenant être invité à **confirmer votre identité au moyen de l'enregistrement d'une vidéo** de votre visage - _il vous suffit de suivre les instructions à l'écran._

6️⃣ Une étape de vérification par e-mail peut aussi intervenir.

7️⃣ Si votre identité est **validée, vous recevrez un SMS de confirmation**.

_Si vous n'avez pas pu valider votre identité, notre partenaire Swan vous contactera sous 72H._

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite ! 😊
