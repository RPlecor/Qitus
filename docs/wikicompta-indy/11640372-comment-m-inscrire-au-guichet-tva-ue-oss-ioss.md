# Comment m'inscrire au guichet TVA UE (OSS/IOSS) ?

## Qu'est-ce que le guichet TVA UE ?

Le guichet TVA UE (OSS/IOSS) permet aux entreprises de déclarer et payer la TVA sur les ventes à distance dans l'Union Européenne, sans avoir besoin de déclarer la TVA dans chaque pays.

## Suis-je concerné par le guichet TVA UE (OSS/IOSS) ?

Vous devez vous inscrire au **guichet OSS** si vous remplissez les trois conditions suivantes :

Vous devez vous inscrire au **guichet IOSS** si vous réalisez du **dropshipping**, c'est-à-dire si vous **importez des biens depuis un pays hors UE pour une valeur inférieure ou égale à 150€**.

## Les étapes pour s'inscrire au guichet TVA UE

## 1️⃣ Adhérer au service

Connectez-vous à votre espace professionnel impots.gouv puis rendez-vous dans la rubrique **Mon Espace > Gérer les services** depuis l'espace d'accueil.

[Image]

Renseignez votre **numéro SIREN** puis cocher le service `Guichet de TVA UE`.

[Image]

Un code d'activation vous est immédiatement envoyé via la messagerie sécurisée impots.gouv. Vous recevrez ce même code par voie postale sous un délai de 2 semaines.

## 2️⃣ Activer le code

Rendez-vous dans votre messagerie impots.gouv : sur l'écran d'accueil, dans la rubrique `Messagerie`, tout en haut.

[Image]

Notez le code que vous avez reçu puis déconnectez-vous de votre compte impots.gouv.

Sur la page d'accueil, cliquez sur `Activer mon espace/mes services`.

[Image]

Inscrivez le code d'activation que vous avez reçu par mail ainsi que les caractères du Captcha demandés.

Les impôts vous demande ensuite votre SIREN et votre adresse mail de connexion à impots.gouv.

Une fois ces étapes validées, le service `Guichet de TVA UE` est activé 🎉

Vous pouvez vous reconnecter à votre espace professionnel : vous y verrez le service activé dans la rubrique **Démarches**.

Cliquez dessus afin de finaliser l'inscription au guichet TVA UE.

[Image]

## 3️⃣ Demander l'inscription

### Identification

Deux rubriques sont visibles ici :

Choisissons l'option OSS. Les impôts vous demande ensuite :

Vous devez ensuite renseigner des informations personnelles tels que votre numéro de téléphone et votre adresse mail.

À la fin du formulaire, ne cochez la case que si vous vendez des biens entre pays de l'UE ou au sein même d'un même pays dans l'UE **pour le compte d'un autre vendeur non établis dans l'UE**.

### Compte bancaire

Cliquez sur le bouton `Continuer` puis renseignez votre IBAN : c'est cet IBAN qui sera utilisé pour le prélèvement de la TVA européenne que vous aurez à régler.

### Établissement ou stock

Un "**Établissement stable"** dans un autre pays de l'UE vous concerne dans le cas où vous possédez une **filiale ou des bureaux dans un autre pays de l'UE**.

La partie "**Stock dans un autre pays de l'UE**" vous concerne si vous avez un entrepôt ou un lieu physique hors UE où vous stockez vos marchandises.

**👉 Si vous n'avez pas de filiale, bureau ou de stock dans un autre pays de l'UE**, cliquez sur le bouton `Continuer` sans rien remplir à cette étape.

### Autre identification

Cette partie est à remplir uniquement si vous disposez d'un numéro de TVA intracommunautaire dans un autre pays membre de l'Union Européenne.

### Autres inscriptions

Cette partie est à remplir uniquement si vous avez déjà été inscrit au guichet unique de l'UE en France ou dans un autre pays membre de l'UE.

👉 Si c'est la première fois que vous vous inscrivez au guichet, cliquez sur le bouton `Continuer` sans rien remplir à cette étape.

Votre inscription au guichet TVA UE est désormais terminé, cliquez ensuite sur le bouton `Valider` ✅
