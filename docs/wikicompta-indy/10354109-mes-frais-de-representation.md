## Les frais de représentation, c'est quoi ?

Les frais de représentation regroupent les dépenses engagées pour représenter votre activité auprès de clients, partenaires ou fournisseurs. Ils servent à entretenir vos relations professionnelles.

## Et la TVA, est-elle déductible ?

La TVA sur les frais de représentation n’est **pas toujours récupérable**, même si la dépense elle-même est déductible comptablement.

Découvrez ci-dessous un tableau récapitulatif qui vous indique si la TVA est récupérable ou non selon les frais de représentation 👇

|     |     |     |
| --- | --- | --- |
| **Type de frais** | **TVA récupérable ✅** | **TVA non récupérable** ❌ |
| Frais de restaurant (seul ou avec un client) au nom de la société avec TVA indiquée | ✘   |     |
| Frais de vêtements professionnels spécifiques (tenues règlementées, EPI, etc.) | ✘   |     |
| Frais de déplacement (train, taxi, bus, métro, avion, VTC, etc.) |     | ✘   |
| Frais d’hébergement (hôtel, Airbnb…) pour vous ou vos salariés |     | ✘   |
| Cadeaux d'affaires aux clients, fournisseurs et salariés < 73€ TTC par an et par bénéficiaire | ✘   |     |
| Cadeaux d'affaires aux clients, fournisseurs et salariés > 73€ TTC par an et par bénéficiaire |     | ✘   |

**Si la TVA n'est pas déductible**, le taux de TVA à appliquer est **0%**.
