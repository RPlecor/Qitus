# Comment trouver l'IBAN renseigné sur mon espace impots.gouv ?

En cas d'échec de télétransmission de votre déclaration de TVA causé par une erreur de paiement, il est essentiel de vérifier que les informations détenues par l'administration fiscale **correspondent exactement à celles saisies sur Indy** et que le **mandat de prélèvement B2B soit actif** 😊

Nous allons voir la marche à suivre ensemble !

## 1️⃣ Trouver l'IBAN enregistré

Pour retrouver l'IBAN que l'administration utilise pour vos prélèvements, connectez vous à votre espace professionnel impots.gouv et rendez-vous dans la rubrique **Gérer les comptes bancaires** via la **barre de menu** ou l'encart **Mon espace** 👇

[Image]

Ensuite, recopiez les caractères du CAPTCHA puis saisissez le SIREN de votre entreprise ou sélectionnez celui concerné dans la liste déroulante puis cliquez sur le lien **Rechercher** correspondant 👇

[Image]

Enfin, vous aurez accès aux différents IBAN renseignés sur votre espace 🎉

[Image]

Ainsi, vous pourrez modifier l'IBAN renseigné en cliquant sur **Modifier** 👇

[Image]

## 2️⃣ Comment voir si le mandat B2B est valide ?

Si le mandat B2B est valide, cela sera notifié dans la colonne **Interentreprises/B2B 👇**

[Image]

Dans le cas contraire, vous verrez une inscription en rouge dans la colonne **Interentreprises/B2B** 👇

[Image]

Pour remédier à cela, vous pouvez vous aider de **[ce tutoriel](https://www.impots.gouv.fr/modele-de-mandat-sepa-de-prelevement-interentreprises-b2b)** de l'administration fiscale 💪

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite 😊

À très vite ! 👋
