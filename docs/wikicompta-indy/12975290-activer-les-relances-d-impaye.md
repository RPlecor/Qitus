# Activer les relances d'impayé | Le wikicompta d'Indy

## Activer les relances

Vous pouvez activer cette option si vous souhaitez qu'Indy envoie un mail de relance aux clients qui n'ont pas réglé leurs factures.

Vous pouvez activer l'option sur une facture Indy **en attente** :

## Configurer les relances

Dans les deux cas, un écran similaire à l’envoi de mail s’ouvre pour vous permettre de configurer les relances 👇

[Image]

### Email(s) du ou des destinataire(s)

Cet encart est prérempli avec l’email indiqué lors de la création de facture. Vous pouvez mettre jusqu'à 10 adresses mail.

### Premier envoi

Vous pourrez sélectionner la date du premier mail de relance envoyé. Elle peut être comprise entre J+1 et jusqu'à 1 an dans le futur : pour une activation des relances le 10/12/25, la date de la première relance peut être comprise entre le 11/12/25 et le 10/12/26.

### Répétition

Vous pouvez définir la fréquence des relances : une fois, toutes les semaines ou tous les mois.

### Nombre maximum de relances

Sauf si vous avez sélectionné "une fois" dans l'encart de répétition vous pouvez choisir le nombre de relances qui seront envoyées avec un maximum de 6 fois.

## Voir la prochaine relance prévue

Une fois l'option activée vous pourrez voir depuis la facture la date de la prochaine relance prévue.
