# Comment obtenir une carte physique ?

La création de votre carte physique est 100% gratuite. La carte vous sera livrée sous 2 à 5 jours ouvrés.

Voici comment procéder à la commande de votre carte physique Mastercard.

## Depuis l'app Indy

1️⃣ Rendez-vous dans l'onglet "**Compte pro**" > **Cartes** et cliquez sur "**Commander une carte physique"** 👇  
​

[Image]

2️⃣ Cliquez ensuite sur "**Ajouter une carte physique secondaire"** 👇  
​

[Image]

3️⃣ Si vous êtes certain d'avoir besoin d'une carte physique, cliquez sur **"J'ai besoin d'une carte physique" :**

[Image]

4️⃣ Puis **saisissez votre adresse de livraison** et cliquez sur "**Commander**".

[Image]

5️⃣ Vous serez ensuite **redirigé chez Swan pour confirmer votre identité** et valider la commande de carte. Commencez par entrer le code à 6 chiffres reçu par SMS et cliquez sur "**Suivant**" :  
​

[Image]

6️⃣ Vérifiez les informations de la commande et cliquez sur "**Confirmer**" :

[Image]

7️⃣ Validez votre identité avec le mode d'authentification forte lié à votre smartphone :  
​

[Image]

  
8️⃣ Votre carte physique est maintenant commandée !

Si vous avez coché la case "**Je veux choisir mon code PIN**" :​ vous recevrez un lien par SMS de notre partenaire Swan qui vous invitera à définir votre code PIN.

👉 Vous n'avez plus qu'à attendre la réception de votre carte. Elle vous sera livrée sous un délai d'environ **2 à 5 jours ouvrés** (pour la France métropolitaine). 🎉  
​

[Image]
