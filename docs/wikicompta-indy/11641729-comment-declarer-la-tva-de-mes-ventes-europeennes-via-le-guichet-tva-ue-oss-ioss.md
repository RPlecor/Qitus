# Comment déclarer la TVA de mes ventes européennes via le guichet TVA UE ? (OSS/IOSS)

## À quoi sert le guichet TVA UE ?

Le guichet TVA UE est un dispositif mis en place par l’Union européenne pour simplifier les démarches de TVA des entreprises qui vendent à des particuliers dans d’autres pays membres.

Pourquoi utiliser ce guichet ?

Grâce à ce dispositif, vous n’avez plus besoin de faire une déclaration de TVA dans chaque pays où vous vendez. Il vous suffit de vous inscrire au guichet TVA UE pour déclarer l’ensemble de vos ventes à distance dans les différents États membres.

## Inscrivez-vous au guichet TVA UE

Pour utiliser le guichet unique et déclarer votre TVA européenne, vous devez au préalable vous inscrire.  
Si vous n'êtes pas encore inscrit, consultez **[cet article de notre centre d’aide](https://wikicompta.indy.fr/fr/articles/11640372-comment-m-inscrire-au-guichet-unique-oss-ioss)** qui vous guidera dans cette démarche 🤓

## Déclarer ses ventes au guichet TVA UE

## 1️⃣ Accédez à votre espace professionnel

Rendez-vous sur impots.gouv > Votre espace professionnel 👇

[Image]

Une fois sur la page d’accueil, dans la rubrique "Démarches" située en bas à droite de l’écran, cliquez sur "**Guichet de TVA UE**" 👇

[Image]

Une nouvelle page s’ouvre. Cliquez ensuite sur “**Déposer vos déclarations de TVA et payer**” 👇

[Image]

Une fois sur la page des déclarations, sélectionnez la période concernée et cliquez sur “**Déposer**” 👇

[Image]

Vérifiez les informations affichées, puis cliquez sur “**Continuer**” pour démarrer votre déclaration 👇

[Image]

## 2️⃣ Saisissez vos opérations de vente

Dans cette étape, vous allez déclarer les opérations de vente réalisées vers d’autres pays de l’Union européenne.

Dans votre interface Indy, vous retrouvez automatiquement tous les montants à déclarer pour la période en cours :

La TVA due est calculée automatiquement. 

Cliquez sur “**Ajouter opération**” après chaque ligne.

[Image]

**👉 Il vous suffit de copier ces montants (via l’icône dédiée) et de les coller dans le guichet TVA UE.**

Dans l'exemple ci-dessous, vous avez réalisé des ventes en Belgique pour un montant total de 8 021,56€ sur le dernier trimestre.

Cliquez ainsi sur l'icône pour copier le montant de 8 021,56€.

[Image]

Puis ajoutez cette vente dans le guichet TVA UE en sélectionnant :

[Image]

## 3️⃣ Vérifiez et validez la déclaration

Une fois toutes les opérations saisies, vérifiez que les montants affichés sur le site des impôts correspondent bien à ceux fournis par Indy.

**👉 En cas d’écart, revenez aux étapes précédentes pour ajuster les données.**

[Image]

[Image]

#### ✅ **Validez la déclaration**

Cliquez sur “**Valider**”, puis “**Confirmer**” pour finaliser votre déclaration.  Une fois cette action effectuée, il faudra ensuite payer le montant déclaré.

[Image]

## 4️⃣ Payez la TVA

Vous pouvez payer votre TVA de deux façons : par Payfip ou par virement bancaire.  
Suivez les instructions affichées dans votre espace.

[Image]

## 5️⃣ **Finalisez le suivi dans Indy**

Dans un délai maximum de 72h après le paiement, la transaction apparaitra sur Indy.

👉 Il ne vous restera plus qu’à la catégoriser en “**TVA payée européenne**” pour finaliser la déclaration.
