# Activer la déclaration de chiffre d'affaires à l'Urssaf

En auto-entreprise, vous devez faire une déclaration chaque mois ou chaque trimestre à l'Urssaf, afin de payer vos cotisations sociales : c'est la déclaration de chiffre d'affaires.

Indy vous permet désormais de préparer puis télétransmettre cette déclaration directement depuis notre application 🎉

Vous n'aurez donc plus besoin de passer par autoentrepreneur.urssaf.fr.

Voyons comment activer ou désactiver cette option sur votre compte 😊

## Activer l'option 🧑‍🔧

Pour activer la déclaration de chiffre d'affaires sur votre compte, il faudra vous rendre dans les [Paramètres > Option](https://app.indy.fr/compte/options).

Vous trouverez en bas de page l'encart suivant 👇

[Image]

Celui-ci sert à activer la déclaration de chiffre d'affaires à l'Urssaf.

Une fois ces deux informations renseignées, vous pourrez cliquer sur **Enregistrer**.

Vous trouverez ensuite l'activité de déclaration dans votre onglet [À faire](https://app.indy.fr/activities) 🎉

## Désactiver l'option 👋

Vous pouvez également désactiver cette activité de votre compte Indy.

Par exemple, si vous préférez faire la déclaration de votre côté 😊

Pour cela, rendez-vous dans les [Paramètres > Option](https://app.indy.fr/compte/options), puis cliquez sur **Révoquer** 👇

[Image]

Si vous avez des questions supplémentaires, n'hésitez pas à nous solliciter via le live-chat, en bas à droite.
