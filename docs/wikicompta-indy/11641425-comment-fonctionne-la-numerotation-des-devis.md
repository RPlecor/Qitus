# Comment fonctionne la numérotation des devis ?

Bonne nouvelle : **les numéros de devis sont totalement libres sur Indy** 🎉  
Contrairement aux factures, la numérotation des devis n’est soumise à **aucune règle spécifique**.

## Qui choisit le numéro du devis ?

C’est vous ! Lors de la création d’un devis, vous remplissez les informations nécessaires, puis vous cliquez sur **Valider le devis**.

À ce moment-là, une petite fenêtre s’ouvre 👇

[Image]

  
→ Elle vous permet de saisir **le numéro de devis de votre choix**.

🔎 En dessous du champ, Indy vous affiche **le numéro de votre dernier devis validé**, pour vous aider si vous souhaitez suivre une suite logique (par exemple : D-001, D-002, etc.).

## Et si deux devis ont le même numéro ?

Pas d’inquiétude !  
Cela signifie simplement que vous avez indiqué **le même numéro** sur deux devis différents 👯‍♀️
