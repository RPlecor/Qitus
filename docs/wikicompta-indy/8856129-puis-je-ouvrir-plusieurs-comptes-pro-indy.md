# Puis-je ouvrir plusieurs comptes pro Indy ?

Si vous avez plusieurs entreprises **avec des SIREN différents**, Indy vous permet d'ouvrir **un compte pro pour chacune d'entre elles** 🚀

Pour ouvrir un second compte pro, il vous suffit de **créer un deuxième compte Indy** en utilisant une adresse e-mail différente, puis de suivre [la procédure d'ouverture](https://wikicompta.indy.fr/fr/articles/8922558-comment-ouvrir-un-compte-pro-indy).

Vous pourrez utiliser le même numéro de téléphone pour vos différents comptes pro Indy.

  
Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite 😉
