# Déposer la 2572 sur le site des impôts

## Qu'est-ce que la déclaration de solde d'IS (2572) ?

📚 La déclaration 2572 est **obligatoire** pour toutes les entreprises dont les bénéfices sont soumis à l’impôt sur les sociétés (IS).

Ce document permet à l’entreprise de signaler à l’administration fiscale :

## Comment transmettre la déclaration 2572 aux impôts ?

Indy vous permet de **générer** et de **télétransmettre** directement votre déclaration 2572 aux impôts 🎉

Cependant, si vous préférez ne pas utiliser la télétransmission, vous devrez déposer **manuellement** cette déclaration sur le site des impôts. Voici les étapes à suivre :

## 1️⃣ Connexion sur impots.gouv

Tout d'abord, connectez-vous à votre compte professionnel sur le site impots.gouv à l’aide de votre identifiant et de votre mot de passe 👇

[Image]

## **2️⃣ Accéder à Déclarer > Impôts sur les sociétés**

Une fois connecté, cliquez sur **"Impôts sur les sociétés"** dans le menu **Déclarer** 👇

[Image]

Sélectionnez ensuite la société pour laquelle vous souhaitez déposer votre déclaration 2572, puis cliquez sur le bouton **Déclarer** 👇

[Image]

Ensuite, choisissez l'année d'imposition pour laquelle vous devez soumettre votre solde d'IS.  
Par exemple, pour déclarer le solde d'IS de l'exercice 2023, cliquez sur **Déclarer** en face de **Année 2023** 👇

[Image]

## 3️⃣ Remplir le formulaire

Un formulaire s'ouvrira alors. Ce formulaire correspond à celui déjà rempli par Indy 🤓

Pour le remplir, téléchargez d'abord la déclaration 2572 depuis l'onglet **Documents** de votre espace Indy. Ensuite, **reportez** les montants inscrits sur votre déclaration téléchargée exactement de la même manière sur le formulaire impots.gouv.
