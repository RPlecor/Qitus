# J'ai un compte Stripe ou Paypal

Comme vous tenez une comptabilité de trésorerie, pour les transactions bancaires, c'est la date du **jour où les montants apparaissent sur votre compte en banque qui fait foi,** donc ici c'est la date où les transactions apparaissent sur votre compte Stripe ou Paypal.

## Comment faire sur Indy ?

Vous pouvez, depuis l'onglet [Transactions](https://app.indy.fr/transactions), **ajouter manuellement** ces recettes à la date où vous les avez perçu et les catégoriser en **Recette**

[Image]

_Si l'ajout manuel de transactions ne vous est pas familier, vous pouvez suivre [cet article](https://wikicompta.indy.fr/fr/articles/2028081-comment-ajouter-une-transaction-manuellement)_ 👈

Il se peut que vous versiez ces recettes sur votre compte professionnel. Si c'est le cas, voici comment procéder 👇

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite 😉
