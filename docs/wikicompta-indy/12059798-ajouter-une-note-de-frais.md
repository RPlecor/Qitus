# Ajouter une note de frais

_._

Parce qu'il arrive que vous ayez à réaliser des dépenses professionnelles depuis votre compte bancaire personnel, voici la marche à suivre pour intégrer celles-ci à votre comptabilité et si besoin, matérialiser leur remboursement 🤓

## **1️⃣** Intégrer la transaction à votre comptabilité

À l'aide du bouton **\+ Ajouter**, choisissez _Note de frais_ et créez la transaction à intégrer 👇

[Image]

  
Lors de la création de l'opération, nul besoin d'utiliser le signe "-" pour indiquer que c'est une transaction au débit, Indy l'a déjà compris lorsque vous avez sélectionné **Note de frais** 😉

Choisissez simplement la catégorie correspondant à la dépense et le taux de TVA indiqué sur la facture.

​

## 2️⃣ Visualiser les notes de frais ajoutées

Une fois les notes de frais ajoutées, vous avez la possibilité de les visualiser dans l'onglet **Transactions.**

Pour cela, cliquez sur le bouton **Filtrer** depuis l'onglet Transactions puis rendez-vous dans la section **Types de transaction** et cochez **Notes de frais** 👇

[Image]

Seules les notes de frais que vous avez ajoutées sont visibles dans l'onglet **Transactions** grâce à ce filtre 🎉

## 3️⃣ Se rembourser les frais avancés

Le virement de remboursement que vous allez réaliser de votre compte professionnel vers votre compte personnel devra tout simplement être catégorisé en **Prélèvement personnel** (colonne _Fonctionnement_).

Si vous le souhaitez, vous avez la possibilité de vous regrouper plusieurs remboursements en un seul virement, cela n'a pas d'impact sur la comptabilité 💡

## Et sur l'app mobile ?

Si vous utilisez l'application mobile, voici comment ajouter une note de frais :

1️⃣ Cliquez sur le bouton **+** en bas de l'onglet **Transactions** 👇

[Image]

2️⃣ Choisissez l'option **Ajouter une dépense** 👇

[Image]

Il s'agit de la même chose qu'une note de frais sur la version web.

3️⃣ Cliquez sur l'option **Moyen de paiement** 👇

[Image]

4️⃣ Puis choisissez l'option **Compte perso / Note de frais** 👇

[Image]

Suivez ensuite les mêmes consignes qu'au début de cet article !
