# Réduction d'impôt pour mes frais de comptabilité

## De quoi s'agit-il ? 🧐

Pour tous les exercices comptables ouverts depuis le 1er janvier 2016, **les frais de comptabilité peuvent être remboursés** sous forme de réduction d'impôt, **à hauteur de 2/3 des dépenses et dans la limite de 915€/an**, s'ils répondent à certaines conditions.

Tout vous sera expliqué 👇

[Image]

L'étape permet d'annuler fiscalement 2/3 des montants engagés pour la tenue de votre comptabilité, afin de les récupérer sous forme de réduction d'impôt.

👉 C'est plus avantageux pour vous.

  
Sur Indy, il vous suffit simplement de rentrer le **montant total de ces frais** dans les cases prévues à cet effet et de valider l'étape 🤓

Une transaction correspondant aux 2/3 des frais de comptabilité sera alors générée 👇

[Image]

  
Il faudra ensuite reporter ces frais sur la 2042 (C PRO), cadre **Réductions Crédits Impôts - ligne 7FF**.

[Image]

Ce montant sera déduit du montant dû aux impôts au titre de l'impôt sur le revenu 🥳
