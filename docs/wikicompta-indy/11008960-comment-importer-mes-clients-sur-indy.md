# Comment importer mes clients sur Indy ?

## 1️⃣ Exporter un fichier au format .CSV

Voici la marche à suivre classique, à adapter selon le logiciel que vous utilisez :

## 2️⃣ Préparer le fichier pour Indy

Depuis la section **Clients** sur Indy, cliquez sur **Importer des clients**. Une modale s’ouvre et vous propose :

[Image]

Vous devez ajouter les colonnes suivantes, dans cet ordre précis 👇  
​`Type | Nom de famille | Prénom | Nom d'entreprise | N° SIRET | N° de TVA | Adresse facturation | Ville facturation | Code postal | Pays (Alpha-2) | Adresse livraison | Ville livraison | Code postal | Pays | Téléphone | Email(s) | Informations supplémentaires`

## 3️⃣ Déposer le fichier dans Indy

Glissez-déposez votre fichier dans la modale ou cliquez sur **Ajouter** 👇

[Image]

Une fois sélectionné, le fichier s’affiche directement dans l’interface 👇

[Image]

## 4️⃣ Vérification du fichier

**Si des erreurs sont détectées**, Indy vous affiche la **liste des lignes concernées** ainsi que les **champs en erreur** 👇

[Image]

**Si tout est bon,** un **aperçu des clients prêts à être importés** s’affiche pour validation ✔️
