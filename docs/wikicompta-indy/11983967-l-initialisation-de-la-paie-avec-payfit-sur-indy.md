# L'initialisation de la paie avec PayFit sur Indy

Depuis août 2025, Indy vous propose de réaliser vos fiches de paie avec notre partenaire PayFit.

Ainsi, une nouvelle activité pour initialiser votre paie apparait dans votre onglet **À faire**. Celle-ci vous permet de :

Voyons ensemble le déroulé de cette activité 👀

##   
Partenariat Indy x PayFit

Dans cette première étape de l'activité **Initialisation de votre paie**, Indy vous demande si vous avez déjà eu un abonnement PayFit, qu'il soit encore actif ou qu'il soit terminé.

[Image]

##   
Vérifier votre éligibilité à PayFit

Cette deuxième étape permet de vérifier votre éligibilité à PayFit.

Indy vous demande l'adresse du siège de votre société et si vous disposez d'une **convention collective** :

[Image]

[Image]

## Création du compte PayFit

Il existe deux cas de figure dans cette étape.

## 1️⃣ Vous éditiez vos fiches de paie avec Indy

Indy vous permet ici de récupérer automatiquement **vos DSN et fiche de paie**.

[Image]

Une fois que vous avez récupéré vos documents de paie, vous pouvez passer à l'étape suivante et créer votre compte sur PayFit.

[Image]

Une fois votre compte PayFit créé, revenez dans cette étape pour le confirmer ✅

## 2️⃣ Vous n'éditiez pas vos fiches de paie avec Indy

Vous pouvez directement **créer votre compte PayFit** en cliquant sur le bouton 👇

[Image]

Une fois votre compte PayFit créé, revenez dans cette étape pour le confirmer ✅

[Image]

## Connexion PayFit x Indy

Cette dernière étape vous permet de synchroniser votre compte PayFit avec Indy 🔄

Cette synchronisation sera utile pour **récupérer les informations de votre paie**, notamment lors de votre clôture comptable et fiscale aux étapes concernant la rémunération et les cotisations sociales.

Cliquez d'abord sur le bouton d'autorisation puis sur **Connecter mon compte**.

[Image]

Renseignez ensuite vos identifiants PayFit puis sélectionnez le compte PayFit à rattacher.

Enfin, il suffit de cliquer sur `Valider l’initialisation de la paie` pour terminer l’activité 👇

[Image]
