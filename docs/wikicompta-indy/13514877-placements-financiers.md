# Placements financiers | Le wikicompta d'Indy

## Définition

Commençons d'abord par une présentation rapide !

Les placements financiers détenus peuvent se classer dans trois grandes familles :

Il s'agit d'un placement financier **long terme**, avec influence dans l'entreprise émettrice. En détenant un titre de participation. vous pouvez peser sur les décisions ou la gestion.

Il s'agit également d'un placement financier **long terme**, mais cette fois sans influence dans l'entreprise émettrice ou sans volonté de contrôle.

Il s'agit d'un placement financier **court terme**, où l'objectif est de générer une plus-value rapide après avoir placé la trésorerie.

ℹ️ Pour savoir à quelle famille appartient le placement financier que vous venez de réaliser, vous pouvez vous poser les deux questions suivantes :

S'il est court terme ou dans une optique de générer une plus-value rapide pour améliorer votre trésorerie, alors vous venez d'acquérir une **valeur mobilière de placement**.

Si l'objectif est long terme, posez-vous la question suivante 👇

Si vous allez avoir une influence dans les décisions prises ou dans la gestion de l'entreprise émettrice, alors il s'agit d'un **titre de participation**.

Si vous n'allez pas avoir d'influence, alors il s'agit d'un **autre titre immobilisé**.

## Comment les ajouter sur Indy ?

La bonne comptabilisation de l'achat de vos valeurs mobilières de placement va se faire en plusieurs étapes !

## Alimenter son compte-espèce

La première étape est d'alimenter votre compte-espèce en faisant un virement du compte bancaire professionnel vers le compte-espèce.

Ce virement doit être catégorisé en **Placement financier** _(colonne Fonctionnement)_.

Vous pouvez maintenant acheter des placements court terme avec l'argent disponible sur votre compte-espèce 🙂

Ces placements seront ensuite visible sur votre compte-titre.

## Enregistrer le placement sur Indy

La deuxième étape sera d'enregistrer le placement financier acquis sur Indy.

Pour cela, rendez-vous dans votre [Clôture comptable et fiscale](https://app.indy.fr/activities/bic-is-fiscal-declaration-activity), puis dans l'étape de clôture des **Placements financiers**. Cliquez sur `+Ajouter un placement` 👇

[Image]

### Qualifier le placement financier

Vous allez maintenant devoir passer un **funnel de qualification de l'actif financier**. Cela permettra de nous assurer qu'il s'agit bien d'un actif court terme et donc d'une valeur mobilière de placement 🤓

Pour cela, sélectionnez d'abord le **type de placement acheté** (actions, obligations, ETF, OPCVM, cryptomonnaie).

Puis répondez à la première question de qualification comme vu plus haut : **Avec quel objectif avez-vous acheté ce placement ? 👇**

[Image]

S'il s'agit d'un placement **court terme**, où l'objectif est de générer une **plus-value** rapide après avoir placé la trésorerie, vous allez pouvoir continuer dans l'étape !

Sinon, il s'agit plutôt d'un titre de participation ou d'un autre titre immobilisé, et il faudra alors plutôt le comptabiliser dans l'étape de clôture [Compléter le registre d'immobilisations](https://app.indy.fr/activities/bic-is-fiscal-declaration-activity/check-assets).

_📎 Pour en savoir plus, n'hésitez pas à consulter [cet article](https://wikicompta.indy.fr/fr/articles/12354646-immobilisations-financieres-ou-titres-de-participation-quelle-difference) de notre centre d'aide._

## À la clôture : calculer le solde théorique final du compte-espèce

Tout au long de l'exercice, plusieurs mouvements ont pu impacter votre compte-espèce : vous l'avez alimenté, vous avez acheté des titres, vous avez perçu des gains (cession de titres, dividendes, intérêts...etc).

Tous ces mouvements ont forcément eu un impact sur le solde de votre compte-espèces 🙂

👉 Il est donc nécessaire de faire un **contrôle de cohérence** au moment de la clôture, entre le montant final théorique calculé par Indy et le montant visible sur votre compte-espèce 👇

[Image]

Pour cela, Indy vous demande le solde final du compte-espèce en fin d'exercice.

ℹ️ Ces informations sont disponible auprès de votre banque (ou application bancaire), courtier, gestionnaire de patrimoine.

⚠️ Attention : si vous avez plusieurs comptes-espèces, il s'agit de l'addition de tous les comptes-espèces.
