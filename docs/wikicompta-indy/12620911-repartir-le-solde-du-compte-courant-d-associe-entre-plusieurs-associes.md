# Répartir le solde du compte courant d’associé entre plusieurs associés

Lorsque votre société a plusieurs associés, Indy permet désormais de suivre **un compte courant par associé** (et plus seulement un compte courant global) 🎉

Si vous aviez jusqu’ici **un seul compte courant d’associé**, il peut vous être demandé de **répartir ce solde historique** entre vos associés.

Cet article vous explique :

## Pourquoi Indy vous demande de répartir ce solde ?

Désormais :

👉 Si vous aviez un **solde historique global**, Indy vous demande donc de le répartir entre vos associés pour repartir sur des bases propres et cohérentes.

Une fois l’étape validée, Indy met simplement à jour vos comptes courants dans la comptabilité : l’ancien compte courant global est soldé et remplacé par un compte courant pour chaque associé, avec les montants que vous avez indiqués dans l’écran de répartition 🙌
