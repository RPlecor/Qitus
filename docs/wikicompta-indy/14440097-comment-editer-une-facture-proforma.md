# Comment éditer une facture proforma ?

##   
📓 **Qu'est-ce qu'une facture proforma ?**

Une facture proforma est une **facture provisoire**, utilisée pour présenter une proposition commerciale à un client.  
Contrairement aux devis, que vous utilisez si vous voulez **officialiser un accord** avec un client, vous pouvez utilisez une **facture proforma** si vous avez besoin de présenter un **aperçu de facture** à votre client, sans engagement.
