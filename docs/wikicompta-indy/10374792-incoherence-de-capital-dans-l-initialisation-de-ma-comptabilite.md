# Incohérence de capital dans l'initialisation de ma comptabilité

L'activité d'**initialisation de votre comptabilité** est une activité indispensable qui va permettre de récolter des informations sur votre SCI et vos associés 🙂

Lorsque vous arrivez à la dernière étape, celle de validation, vous pouvez être bloqué et un message s'affiche 👇

[Image]

Il s'agit ici d'une sécurité que nous avons mise en place afin de s'assurer que les informations renseignées sur le capital social de votre SCI soient cohérentes avec votre Kbis.

Voici les différentes étapes à suivre pour résoudre cette incohérence 👇

1️⃣ La première étape consiste à vous munir de vos **statuts.**

Vous devrez rechercher l'article concernant le _**Capital social**_ afin de trouver les éléments dont vous aurez besoin aux prochaines étapes 👇

[Image]

Vous allez maintenant vérifier les étapes précédentes pour trouver ce qui bloque🕵️

2️⃣ La première étape à vérifier est **Informations sur votre SCI.**

Il faut vérifier ici si le montant renseigné dans le champ valeur d'une part sociale... 👇

[Image]

... correspond bien à ce qui est indiqué sur vos statuts 👇

[Image]

Si le montant est identique, c'est parfait !

Si ce n'est pas le cas, il faut le modifier en indiquant le même montant que sur vos statuts.

3️⃣ La deuxième étape à vérifier est **Informations sur les associés.**

Il faut vérifier ici si le nombre de parts renseignées pour chaque associé... 👇

[Image]

... correspond bien à ce qui est indiqué sur vos statuts 👇

[Image]

Pour modifier les parts, il vous suffit de cliquer sur l'icône ✏️ en haut à droite de chaque associé 👇

[Image]

4️⃣ Après avoir fait ces vérifications et corrections, le problème est résolu et vous pouvez valider votre initialisation 🎉

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via la bulle rose en bas à droite 😉

À très vite !
