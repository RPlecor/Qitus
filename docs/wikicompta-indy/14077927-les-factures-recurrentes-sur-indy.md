# Les factures récurrentes sur Indy

Les factures récurrentes vous permettent de créer des gabarits de facturation automatique : vous configurez une fois le contenu de votre facture, définissez une fréquence et Indy génère vos factures aux échéances prévues automatiquement.

17 mars 2026

Table des matières

## Accéder aux factures récurrentes

Les factures récurrentes sont accessibles depuis le module de facturation, via l'onglet **Factures récurrentes**.

## Créer un gabarit de facture récurrente

Pour créer votre première facture récurrente, cliquez sur le bouton de création depuis l'onglet dédié. Le formulaire qui s'ouvre est très similaire à celui d'une facture classique - vous retrouvez les mêmes champs habituels (client, lignes de prestation, montants, TVA le cas échéant).
