# Comment demander un remboursement de crédit de TVA ?

Vous avez un crédit de TVA et vous souhaitez son remboursement par les impôts ? Voici comment faire sur Indy.

Mis à jour cette semaine

Lorsque vous remplissez l'activité de TVA pour faire votre CA3 ou C12 dans l'onglet **À faire**, vous avez une étape **Crédit de TVA**.

Dans un premier temps vous devrez indiquer si vous souhaitez que ce crédit de TVA se reporte sur la prochaine déclaration ou si vous souhaitez un remboursement de la part des impôts.

[Image]

Si vous optez pour le remboursement de TVA, voici ce que vous devrez faire ensuite 👇

[Image]

Ensuite, comme à votre habitude, vous pourrez choisir ou non de télétransmettre votre déclaration de TVA avec Indy 🎉

Pour assurer, le suivi de ce remboursement, nous vous demanderons le statut de votre demande lors de vos déclarations de TVA suivantes.

Vous aurez le choix entre 3 possibilités 👇

Si vous avez des questions supplémentaires, n'hésitez pas à nous solliciter via le live-chat, en bas à droite.

À bientôt ! 👋
