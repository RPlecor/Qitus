# Taxes sur les véhicules de tourisme : fonctionnement et déclaration sur Indy

Depuis 2023, la TVS a été remplacée par deux taxes environnementales sur les véhicules de tourisme. Découvrez qui est concerné, comment les calculer et comment les déclarer via Indy.

23 avril 2026

Table des matières

Depuis le 1er janvier 2023, la TVS (Taxe sur les Véhicules de Société) n'existe plus. Elle a été remplacée par **deux nouvelles taxes environnementales** 👇

1️⃣ La **taxe annuelle sur les émissions de CO₂**,

2️⃣ La **taxe annuelle sur les émissions de polluants atmosphériques, aussi appelée taxe d'ancienneté**.

## Qui est concerné ?

Votre entreprise est redevable de ces taxes si elle possède ou utilise un véhicule de tourisme, que ce soit :

👉 Seuls les véhicules utilisés en France sont concernés.

La période d'imposition s'étend **sur l'année civile**, et non plus par trimestre 📆

## Comment se calcule la taxe ?

Indy s'occupe du calcul pour vous 🎉 Mais pour les curieux, voici comment ça fonctionne 👇

1️⃣ **Taxe sur les émissions de CO₂** → calculée selon le taux de CO₂ émis (normes NEDC ou WLTP) ou, à défaut, la puissance fiscale du véhicule. Le calcul se fait au prorata du nombre de jours d'utilisation dans l'année.

2️⃣ **Taxe sur les émissions de polluants atmosphériques (taxe d'ancienneté)** → basée sur la date de première mise en circulation du véhicule et le type de carburant utilisé.

👉 Le montant total correspond à la **somme des deux taxes**. Des simulateurs en ligne existent si vous souhaitez vérifier les montants.

## Quand et comment déclarer ?

La déclaration **dépend de votre régime de TVA** 👇

## Comment déclarer sur Indy ?

Dès le mois de janvier, vous retrouverez dans votre onglet **[À faire](https://app.indy.fr/activities)** une activité intitulée _Taxes sur les véhicules de tourisme_. Elle automatise le calcul des taxes et vous guide pour la déclaration 💪
