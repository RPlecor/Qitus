# Intégration automatique avec Malt | Le wikicompta d'Indy

Lorsque vous passez par un intermédiaire comme Malt, l'édition de vos factures est simplifiée. Cela se traduit par le prélèvement d'une commission par la plateforme.

  
Que ce soit les factures générées ou les commissions, ces données sont indispensables à votre comptabilité 💎

Nous avons donc travaillé en partenariat avec Malt pour vous proposer une **intégration automatique de vos données comptables** 🔥

## Quelle est l'utilité de cette fonctionnalité ?

Les opérations bancaires venant de la plateforme sont identifiées dans l'onglet **Transactions**.

Une ligne apparaîtra en dessous des transactions en question, vous suggérant de visualiser les deux propositions d'Indy 👇

[Image]

Lorsque votre client vous règle via Malt, la plateforme déduit automatiquement ses commissions avant de vous reverser votre recette.

  
​_Voici un exemple :_

_Vous facturez 1000€ à votre client. Ce même client va verser les 1000€ sur la plateforme. Malt vous reversera ensuite seulement 950€ sur votre compte bancaire, car ils vont prélever 5% de commissions._

Dans votre comptabilité, il faudra bien faire apparaitre les 1 000 € de recette perçue, et les 50 € de commissions payées.

Indy va donc se charger de subdiviser la transaction, pour affecter les bonnes sommes dans les bons comptes comptables 💪

Pour chaque mission, vous avez donc deux factures :

\- celle que vous envoyez à votre client

\- celle qui vous est destinée, de la part de Malt.  
​  
​_En reprenant notre exemple précédent :_

_\- une facture de 1000€ (envoyée à votre client, justifiant votre chiffre d'affaires)_

_\- une facture de 50€ (justifiant la commission prélevée)_

Ainsi, les deux documents vont bien être récupérés automatiquement par Indy via l'intégration 👌

Une fois validée, le tour est joué ! 🎉
