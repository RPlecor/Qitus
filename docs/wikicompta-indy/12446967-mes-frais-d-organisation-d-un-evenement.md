# Mes frais d'organisation d'un évènement

Votre entreprise peut être amenée à organiser différents types d’événements. Cet article vous explique comment traiter ces dépenses sur Indy.

29 septembre 2025

Table des matières

Votre entreprise peut être amenée à organiser différents types d’événements : une rencontre avec des clients autour d’un cocktail, une soirée d’entreprise, un séminaire destiné à vos salariés, ou encore un repas dans un cadre professionnel ou social (par exemple, l’organisation d’une soirée pour promouvoir un nouveau produit).
