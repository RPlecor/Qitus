# Que faire lorsque la synchronisation avec le compte pro Indy est interrompue ?

Il peut arriver dans de rare cas que la synchronisation de votre compte pro Indy soit interrompue.  
​  
La procédure suivante permet de rétablir le lien entre Indy et votre compte très facilement.  
​

👉 Rendez-vous sur **[Paramètres > Banques](https://app.indy.fr/compte/banques)** puis cliquez en bas sur "**Ajouter un compte ou une carte"** puis en bas de nouveau sur **"Ajouter une nouvelle banque"**.  
​

Choisissez **Compte Pro Indy** et suivez les instructions :

[Image]

Une authentification forte vous sera demandée pour valider la synchronisation, une fois celle-ci réalisée, vous retrouverez l'accès à l'ensemble des fonctionnalités.

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite 😉

À très vite ! 👋
