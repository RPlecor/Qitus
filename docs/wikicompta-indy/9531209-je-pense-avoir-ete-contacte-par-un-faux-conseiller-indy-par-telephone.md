# Je pense avoir été contacté par un faux conseiller Indy par téléphone

## Vos échanges avec Indy

Des personnes se faisant passer pour des conseillers Indy vous ont peut-être contacté.  
​

⚠️ **Chez Indy tout le service client passe par le live-chat depuis le site ou l'app, nous n'utilisons jamais le téléphone.**

Le live-chat est accessible par cette petite bulle rose avec un sourire, en bas à droite, lorsque vous naviguez sur Indy.

[Image]

💁 _plus d'information sur le service client sur [cet article](https://wikicompta.indy.fr/fr/articles/2589510-le-service-client)_

## Si vous avez été contacté par une personne prétendant être un conseiller Indy

👉 **Ne faites aucune action spécifique sur votre compte**, une personne malveillante **vous fera miroiter un problème urgent qui nécessite une action de votre part,** cela doit vous mettre la puce à l'oreille sur le fait que c'est une fraude.

🛟 Le réflexe à avoir est de vous connecter sur votre app ou sur le web et de **cliquer sur la bulle rose avec un sourire pour nous contacter et nous le signaler.**

Si une personne vous demande d’approuver une transaction par authentification forte ou encore d'en réaliser une pour annuler une opération frauduleuse, **il s’agit d’une arnaque.** Indy n'a a aucun moment besoin de vos informations de sécurité personnelles pour bloquer une transaction.

Enfin, si une personne vous propose de **virer une transaction sur un autre IBAN, prétextant un problème de plafonds, c'est une arnaque.**

## Si vous avez un doute sur un email, un message, un appel

👉 Contactez nous sur le live-chat pour en discuter, nous pourrons vous aider à identifier la source.
