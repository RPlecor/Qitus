# Comment supprimer mon compte Indy ?

Si vous souhaitez conserver l'accès à vos données et documents comptables sans supprimer votre compte, vous pouvez 👉 **[annuler votre abonnement.](https://wikicompta.indy.fr/fr/articles/5402858-annuler-votre-abonnement)**

## Comment procéder pour supprimer mon compte ?

Pour supprimer votre compte Indy, rendez-vous dans vos **Paramètres > Profil**, puis cliquez sur **"Supprimer mon compte"** 👇

[Image]

## Pourquoi je ne peux pas supprimer mon compte ?

## 1️⃣ J'ai un compte pro Indy

Si vous avez un Compte pro, la suppression de votre compte ne peut pas être effectuée en autonomie ❌  
👉 Veuillez **contacter notre service client via le live-chat** afin que nous puissions traiter votre demande.

## 2️⃣ Je suis encore en cours d'inscription

Si votre inscription sur Indy n'est pas finalisée, vous ne pourrez pas supprimer votre compte sans intervention de notre part.  
👉 N'hésitez pas à **nous contacter via le live-chat** pour que nous effectuions cette action de notre côté.

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite 😉

À très vite ! 👋
