# Frais d’agence locative : comment bien les comptabiliser ?

Si certaines de vos recettes apparaissent sur votre compte bancaire minorées par des frais, voici la solution pour les comptabiliser.

27 mars 2025

Table des matières

Lorsque vous faites appel à une agence locative pour la gestion de vos biens, celle-ci encaisse tous les versements des locataires avant de vous les reverser, après avoir déduit différents frais.

Pas d'inquiétude, voici comment saisir l'intégralité de vos recettes et des frais associés sur Indy 😊

## Étape 1 : obtenir le relevé de gestion

Dans un premier temps, vous aurez besoin du relevé de gestion mensuel de votre agence. Ce document détaille l'ensemble des revenus et des dépenses liés aux biens gérés pour la SCI.

## Étape 3 : subdiviser la transaction

Maintenant que vous avez identifié les montants de chaque catégorie, vous pouvez les répartir au sein de la transaction correspondant au solde versé par l'agence.

Pour cela, cliquez sur la transaction bancaire, puis sur **\+ Ajouter une ventilation** 👇
