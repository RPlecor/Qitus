# Quels sont les chiffres derrière les graphiques ?

Dans votre onglet **[Pilotage](https://app.indy.fr/)**, vous avez accès à plusieurs graphiques :

## 📊 Évolution de vos recettes et dépenses

[Image]

**📆 À savoir**

L'affichage est mensuel, sauf s'il y a moins de 3 mois de données.

Dans ce cas-là vous aurez un affichage hebdomadaire.

**Dans ce graphique vous avez :**

Si vous êtes assujetti·e à la TVA 👉 sachez que **tous les montants affichés sont HT**.

## 💸 Les dépenses

[Image]

Sur le deuxième graphique, vous trouverez le classement de vos dépenses dans l'ordre (du plus gros poste de dépense vers le plus faible). Nous laissons 5 postes de dépense visibles, le reste est regroupé en **Autres** **dépenses**.  
​

## 💰 Les prélèvements personnels

[Image]

Sur ce graphique, nous listons uniquement **les prélèvements personnels** (en bleu clair) et **les dépenses personnelles** (en bleu foncé).

**💡 À noter**

Il est conseillé de catégoriser :

## 🏦 Trésorerie

[Image]

Ce graphique permet de suivre l'évolution de votre trésorerie sur les 3 derniers mois.

💡 Votre trésorerie correspond à l'argent disponible sur votre compte bancaire.

Si vous avez plusieurs comptes bancaires synchronisés sur Indy, le graphique affichera le solde cumulé de ces différents comptes 😊

## 🧐 Quid des opérations comptables ?

Ces graphiques ne prennent pas en compte les retraitements comptables ❌

Seules les transactions passant par un compte financier sont comptabilisées.  
C'est-à-dire :

Ainsi cela vous permet d'avoir un oeil sur la trésorerie de votre entreprise, mais pas nécessairement sur votre résultat fiscal.

### Par exemple

_L'achat de matériel avec votre compte professionnel apparaît dans le graphique._  
​_Mais les amortissements de vos immobilisations en fin d'année n’apparaissent pas !_ 

_En effet, cela viendrait augmenter grandement vos dépenses de décembre et ne serait pas révélateur de votre comptabilité au quotidien._  
​  
Tous les retraitements comptables sont pris en compte dans la 2035 en revanche 🤓
