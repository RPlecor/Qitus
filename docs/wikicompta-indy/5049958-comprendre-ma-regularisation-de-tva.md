# Comprendre ma régularisation de TVA

L'étape **Régulariser la TVA** permet de comparer les différents montants de TVA que vous avez déclarés aux impôts avec ceux qui auraient dû être déclarés.

👉 Pour cela, Indy va contrôler les comptes de TVA sur l'ensemble de l'année puis vérifier que ces montants ont bien été déclarés auprès des impôts.

## Comment sont calculés les montants à déclarer ?

Indy utilise les informations de vos transactions, c'est-à-dire les taux de TVA que vous appliquez toute l'année sur vos recettes et vos dépenses 🤓

Vous retrouverez toutes ces informations directement dans le **[Grand Livre](https://app.indy.fr/document/grand-livre)** 👇

[Image]

Ici nous voyons un compte de TVA déductible 👉 chaque ligne détaille le montant de TVA déductible qui est associé à chaque transaction faite sur le mois de mars.

💡 La dernière ligne "**Déclaration de TVA - Mars 2025**" n'est pas prise en compte dans le calcul de TVA déductible. C'est une écriture comptable qui reprend le montant de TVA déductible déclaré aux impôts.

Indy va ainsi récupérer les **informations de chaque compte de TVA sur toute l'année** pour avoir le **montant total dans vos transactions**. C'est le **montant à déclarer 😊**

## Comment sont trouvés les montants que j'ai déclarés aux impôts ?

Ces montants sont calculés grâce aux déclarations que nous récupérons via l'étape.

Indy va analyser toutes ces déclarations afin de trouver les montants que vous avez déclarés toute l'année.

🚨 Attention, ces montants ne sont pas toujours ceux qui sont simplement affichés dans votre déclaration de TVA. Indy utilise toutes les lignes de votre déclaration de TVA pour les trouver.

_Par exemple :_

Indy vous indique trouver 1 251,93 € sur l'exercice 👇

[Image]

Pourtant vous avez déclaré un peu moins : 1 036 € 🤔

[Image]

La différence (= 215,93 €) sera alors reportée sur la déclaration de TVA du prochain exercice ✅

## J'ai une régularisation

Rien de grave, ne vous en faites pas 😃

Lorsqu'Indy trouve une différence entre ce qui a été déclaré aux impôts et ce que vous avez dans vos transactions, il crée une régularisation.

💡 La régularisation sera appliquée sur la prochaine déclaration de TVA que vous allez faire une fois votre liasse fiscale clôturée.

Cela vous permet de vous assurer que tout a bien été déclaré de votre côté et sans aucune incohérence.

Vous aurez plus d'infos directement dans l'étape en cliquant sur **Afficher le détail** 👇

[Image]

Vous pourrez ainsi retrouver les différents montants trouvés par Indy 👇

[Image]

Ainsi que l'écart détecté, qui sera donc régularisé dès que possible.

Enfin il se peut que vous ayez des **régulations de TVA liées à un autre exercice**.

☝️ Dans ce cas il est important de nous le signaler, afin que nous puissions intégrer ces transactions correctement sur votre déclaration.

## Pour résumer

Pensez à bien vérifier vos transactions dans le **[Grand Livre](https://app.indy.fr/document/grand-livre)**.

👉 Si les taux de TVA appliqués sont incorrects cela peut être source d'écart.

Vérifiez qu'aucune régularisation ne soit liée à un exercice précédent

👉 Vous pouvez solliciter votre comptable précédent, si vous n'utilisiez pas Indy.

Si vous ne voyez aucune anomalie dans vos transactions

👉 La régularisation proposée est donc nécessaire, vous pouvez valider.
