# Télétransmettre ma 2035 aux impôts

Sur Indy, nous envoyons votre 2035 directement aux impôts que vous soyez adhérent à une AGA ou non. C'est ce qu'on appelle la télétransmission 😊

Nous vous détaillons la procédure dans cet article.

## Comment faire votre télétransmission ?

La télétransmission n'est possible qu'une fois la 2035 terminée sur Indy ✅

Ainsi, vous devez d'abord avoir rempli toutes vos étapes de clôture dans l'onglet **[À faire](https://app.indy.fr/activities/d-2035-activity)**.

Vous pourrez cliquer sur Télétransmettre pour envoyer votre 2035 aux impôts 👇

[Image]

À ce moment-là, Indy fait le lien avec les impôts directement 💪

Le document envoyé est un fichier au format EDI : celui-ci contient la déclaration 2035.

Vous recevrez un mail de confirmation une fois la 2035 réceptionnée par les impôts.

ℹ️ Si vous êtes adhérent d'une AGA, nous vous envoyons en parallèle votre liasse fiscale ainsi qu'à votre AGA.

  
Si vous n'avez pas d'AGA, vous recevrez une confirmation de télétransmission par mail.  
Si vous souhaitez vos documents en PDF, il suffit de télécharger votre 2035 dans vos **Documents.**

## Vous ne souhaitez pas que l'on télétransmette votre 2035 ?

Vous pouvez nous contacter sur le support afin qu'on annule la télétransmission.  
⚠️ Dans ce cas c'est à vous de déclarer votre 2035 auprès des impôts.

Vous avez la possibilité de télécharger votre liasse fiscale depuis vos **Documents.**

[Image]

Si vous souhaitez que votre AGA vérifie votre comptabilité avant la télétransmission aux impôts, il vous suffit de télécharger vos documents et leur transmettre par mail.

## Et si j'ai fait une erreur ?

Pas de panique, vous pourrez rouvrir la 2035 via l'onglet [Documents](https://app.indy.fr/documents) 👉 [article ici](https://wikicompta.indy.fr/fr/articles/4507700-reouverture-et-changement-d-exercice-fiscal-annee-comment-faire).

Vous pourrez la corriger puis la renvoyer, toujours via le bouton **Télétransmettre**

ℹ️ Vous aurez aussi la possibilité de **Valider sans télétransmettre**, si les modifications ne nécessitent pas un renvoi à l’administration fiscale.

[Image]

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite 😊
