# Comment subdiviser une transaction ?

Il se peut que certaines transactions correspondent à deux catégories différentes.

Voyons comment subdiviser vos transactions afin d'appliquer ces 2 catégories.

## ✂️ Attribuer une part personnelle

En profession libérale, la frontière entre personnel et professionnel est souvent étroite.

Prenons en exemple votre facture de téléphone : la plupart du temps, vous n’avez qu’un seul abonnement qui vous sert pour le personnel et le professionnel. 

Il est donc courant d’attribuer une part personnelle à une transaction en fonction de l’usage que vous en faites. 

Dans Indy, c’est très simple. Il vous suffit de vous rendre dans l'onglet **[Transactions](https://app.indy.fr/transactions)** et de **cliquer sur la transaction** que vous souhaitez subdiviser.

Vous pourrez avoir accès aux informations de votre transaction : le libellé, la date et le montant total (en TTC) 👇 

[Image]

💡 **À noter**

Sur la colonne de droite, vous pouvez **changer la catégorie de la transaction** ainsi que le **taux de TVA** si vous y êtes assujetti·e.

C'est aussi ici que vous pourrez **subdiviser la transaction**.

  
Cliquez sur **\+ Ajouter une ventilation** 👇 

[Image]

Il vous suffit d'appliquer la catégorie adéquate à votre ventilation.

Dans notre exemple, vous pouvez laisser **Télécom, fournitures, docum.** pour la ventilation 1 et **Dépense personnelle** pour la ventilation 2.

🚨 Il est important d'ajouter le signe "-" devant le montant à subdiviser.

Vous pouvez également ajouter une 3e subdivision, voire plus si nécessaire 😊

Pour valider, il ne vous reste plus qu'à cliquer sur **Appliquer** en haut à droite.

**💡 Petite astuce**

Si vous voulez attribuer un % pour une dépense non personnelle : choisissez dans un premier temps la catégorie **Dépense personnelle** pour pouvoir saisir un pourcentage et ensuite, changez de catégorie 🤓

## Supprimer une ventilation ❌

Si vous souhaitez annuler/supprimer une subdivision, vous devez le faire **manuellement.**

Pour cela, retrouvez la transaction concernée dans l'onglet **[Transactions](https://app.indy.fr/transactions)** et cliquez dessus.

Cliquez ensuite sur **l'icône poubelle** 🗑 pour supprimer la ventilation souhaitée. 

[Image]

Pour finir, afin que votre transaction retrouve son montant global, cliquez sur la **baguette magique** à droite du montant TTC de la ventilation restante.

[Image]

  
La subdivision est désormais annulée 🎉

  
Il ne vous reste plus qu'à cliquer sur **Appliquer** pour enregistrer cette annulation.
