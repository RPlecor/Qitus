# Comment acquitter mes factures sur Indy ?

Si vous éditez vos factures sur Indy, vous avez la possibilité de les indiquer comme Acquitté. Voici la marche à suivre !

19 janvier 2026

Table des matières

Depuis août 2024, vous pouvez ajouter la **mention “acquittée”** sur vos factures créées dans Indy. Il s’agit d’une pièce facultative, mais parfois utile pour prouver qu’une facture a bien été réglée dans son intégralité.
