# Synchroniser un nouveau compte bancaire professionnel sur Indy

Sur Indy, vous pouvez connecter jusqu'à 99 comptes bancaires différents 🎉  
​

## Ajouter un compte depuis une banque déjà connectée

Pour ajouter un compte bancaire sur Indy, il faut vous rendre dans l'onglet **Paramètres > [Banques](https://app.indy.fr/compte/banques).**  
​  
Ensuite, il vous suffit de cliquer sur **\+ Ajouter un compte ou une carte** puis de sélectionner le compte concerné 👇

[Image]

Vous pouvez reconnaitre le compte par son nom ou par le montant de son solde par exemple.

[Image]

##   
Ajouter une nouvelle banque

Cela se passe aussi depuis l'onglet **Paramètres > [Banques](https://app.indy.fr/compte/banques).**  
​  
Votre compte est dans une banque différente du premier compte synchronisé, il faut alors **connecter cette nouvelle banque.**

[Image]

Comme lors de votre première connexion, vous devez saisir vos identifiants pour que Indy puisse synchroniser cette nouvelle banque.

[Image]

Ensuite vous n'aurez plus qu'à sélectionner le compte que vous souhaitez synchroniser avec Indy 🎉

[Image]

## Éviter de multiplier les comptes bancaires

Plus vous ajouterez de comptes bancaires sur Indy et moins votre comptabilité sera claire.  
​  
Nous vous conseillons vivement d'avoir **uniquement votre compte courant** professionnel synchronisé avec Indy : le compte sur lequel apparaissent vos recettes et vos dépenses professionnelles.
