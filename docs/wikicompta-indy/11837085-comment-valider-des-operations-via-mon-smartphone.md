# Comment valider des opérations via mon smartphone ?

Pour valider les opérations sensibles sur votre compte pro Indy, il est nécessaire de procéder à une authentification forte. Cette procédure passe par défaut par l'envoi d'un SMS contenant un lien.

Vous pouvez désormais valider vos opérations sensibles directement depuis une **notification in‑app** sur votre smartphone !
