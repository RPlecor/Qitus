# Synchroniser mon compte bancaire sur Indy

## **Ajouter un autre compte depuis une banque déjà synchronisée sur Indy**

Depuis la page **[Paramètres > Banques](https://app.indy.fr/compte/banques)**, il vous suffit de cliquer sur **\+ Ajouter un compte ou une carte** puis de sélectionner le compte bancaire concerné dans la fenêtre qui s'ouvre.

[Image]

Sélectionnez alors le compte bancaire à synchroniser dans la fenêtre qui s'ouvre 👇

[Image]

## **Je change de banque, comment faire ?**

Lorsque vous changez de banque vous avez souvent une période de transition où vous avez l'ancien et le nouveau compte bancaire.

Sur Indy ce n'est pas un problème vous pouvez **ajouter les deux comptes bancaires** dans l'onglet **[Banques](https://app.indy.fr/compte/banques)**.

Toujours depuis l'onglet **Paramètres > Banques > + Ajouter un compte ou une carte**, cliquez sur le bouton **\+ Ajouter une nouvelle banque 👇**

[Image]

Comme lors de votre arrivée chez Indy, vous devez saisir vos identifiants pour que Indy puisse synchroniser cette nouvelle banque 👇

[Image]

Ensuite vous n'aurez plus qu'à sélectionner le compte bancaire que vous souhaitez synchroniser avec Indy 🎉

[Image]

## Quid des comptes épargnes ?

En tant que SCI à l'IS, les comptes épargne relatifs à votre société doivent également figurer dans votre bilan en fin d'exercice.

Si vous **éditez un bilan**, les comptes épargne doivent également apparaitre dans votre bilan en fin d'exercice.

Si vous **n'éditez pas de bilan pour votre SCI à l'IR**, vous pouvez vous passer des comptes épargne.

Toutefois, si un compte épargne génère des intérêts imposables, ils doivent être intégrés dans votre comptabilité. Contactez le service client pour en savoir plus !

## **Ajouter un compte en devises étrangères**

Indy n’est pas adapté pour synchroniser **les comptes bancaires en devises étrangères** 👉 seuls **les** **comptes en euros** peuvent être synchronisés.
