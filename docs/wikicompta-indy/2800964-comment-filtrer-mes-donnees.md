# Comment filtrer mes données ?

La filtres sont disponibles dans l'onglet **[Transactions](https://app.indy.fr/transactions)** 👇

[Image]

  
​

Ils vous permettent de retrouver rapidement certaines de vos transactions.

Balayons ensemble les différents filtres existants 😊

## 📆 Plage de dates

Vous pouvez filtrer par dates afin de retrouver les transactions sur une période spécifique.

### Sélectionner des dates spécifiques

En cliquant sur **Début - Fin** sous **Plage de dates**, un calendrier s'ouvrira et vous permettra de choisir la période que vous souhaitez mettre en lumière 💡

  
Il suffit de cliquer :

[Image]

### Sélectionner des mois

Comme pour la sélection par mois, en cliquant sur **Début - Fin**, vous avez la possibilité de choisir le mois souhaité en cliquant sur celui qui vous intéresse.

[Image]

Indy vous affichera toutes les transactions du mois sélectionné 🤓

### Sélectionner un exercice

Vous pouvez encore élargir votre recherche par exercice fiscal.  
Pour cela, il suffit de cliquer sur **Exercice** puis sélectionner celui qui vous intéresse 👇

[Image]

## 🤑 Plage de montants

Vous pouvez utiliser ce filtre pour retrouver une transaction spécifique (ou plusieurs !) par son montant 👌    
​

[Image]

## ↔️ Entrées / sorties

Cette fonction vous permet de filtrer vos transactions par **Recette** ou **Dépense** 🤓

[Image]

## 🧩 Catégories

Ici vous pouvez filtrer vos transactions par **Catégorie**.  
Il est possible de cumuler plusieurs catégories 👍

[Image]

## 📝 Justificatifs

Par défaut réglé sur **Tous**, vous pouvez choisir d'afficher les transactions auxquelles un justificatif a été lié ou bien toutes celles qui n'en n'ont pas 🧾  
​

[Image]

## 🧩 Taux de TVA

Vous pouvez filtrer vos transactions par taux de TVA ou par la nature de la TVA : Française ou étrangère.  
​  
Vous pouvez afficher vos transactions qui n'ont pas encore de TVA définie.

[Image]
