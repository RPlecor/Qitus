# Quelle TVA puis-je déduire ?

Vous souhaitez récupérer la TVA d’une facture ? Découvrez sur Indy comment identifier une TVA déductible.

23 avril 2026

Table des matières

Lorsque vous êtes assujetti.e à la TVA, vous :

C'est la différence entre la TVA collectée et la TVA déductible qui détermine si vous devez reverser de la TVA à l'Etat ou si c'est l'Etat qui vous doit en doit !

→ Il est donc naturel de chercher à maximiser le montant de TVA déductible.

En d’autres termes, si votre facture concerne une exception, vous ne pouvez pas récupérer la TVA et vous devez **indiquer 0%** sur la transaction correspondante, comme illustré ci-dessous.

## Mais alors, comment s'y retrouver ?

Découvrez ci-dessous un tableau récapitulatif qui vous indique si la TVA est récupérable ou non selon le type de dépenses.

|     |     |     |
| --- | --- | --- |
| **Types de dépenses** | **TVA récupérable ✅** | **TVA non récupérable** ❌ |
| Frais d’hébergement (hôtel, Airbnb…) pour vous ou vos salariés |     | ✘   |
| Frais d’hébergement d’un tiers (client, fournisseur…) | ✘   |     |
| Frais de restaurant (en déplacement professionnel) au nom de la société avec TVA indiquée | ✘   |     |
| Frais de restaurant (en déplacement professionnel) sans nom de la société (avec tampon au nom de la société à la main) et inférieurs à 150€ TTC | ✘   |     |
| Frais de restaurant dans le cadre d’une formation, d’un séminaire, etc. | ✘   |     |
| Petit déjeuner et/ou repas à l’hôtel facturés séparément des nuits d’hôtel | ✘   |     |
| Achat et entretien d'un véhicule de tourisme (avec des places à l'arrière) |     | ✘   |
| Location ou crédit-bail d'un véhicule de tourisme |     | ✘   |
| Achat, location et entretien d'un véhicule utilitaire | ✘   |     |
| Frais de transport de personnes (train, taxi, bus, métro, avion, VTC, etc.) quel que soit le tiers |     | ✘   |
| Refacturation de frais de transport de personnes (train, taxi, bus, métro, avion, VTC, etc.) par un prestataire en plus de ses honoraires | ✘   |     |
| Frais de péage et de parking pour un véhicule de tourisme ou pour un véhicule utilitaire | ✘   |     |
| Cadeaux aux clients, fournisseurs et salariés < 73€ TTC par an et par bénéficiaire | ✘   |     |
| Frais de carburant (gazole, essence, électricité, GPL) pour un véhicule utilitaire | ✘ (100%) |     |
| Frais de carburant (gazole, essence) pour un véhicule de tourisme | ✘ (80%) | (20%) |
| Frais de carburant (électricité, GPL) pour un véhicule de tourisme | ✘ (100%) |     |

💡 Si la TVA est récupérable, vous pouvez indiquez le taux de la facture directement sur la transaction, et Indy se charge de reporter le montant en ligne 20 de votre déclaration.
