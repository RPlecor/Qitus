# Distinguer les sources de revenus

Il est courant qu'une entreprise ait plusieurs activité. Par exemple :

Sur Indy il n'est pas possible de créer une catégorie de recette pour chacune. Il existe toutefois deux manières de distinguer vos revenus.

## 1️⃣ Utiliser les catégories proposées par Indy

Les catégories Vente de service, Vente de services annexes et Autre ains divers sont parfaites pour ce rôle. Nous vous conseillons la répartition suivante :

Vous trouverez ces catégories dans la modales de catégorisation en colonne Revenus 👇

[Image]

Vous pourrez ensuite filtrer vos transactions par catégorie pour retrouver facilement les revenus liés à chacune de vos activités.

## 2️⃣ Utiliser les annotations

Pour encore plus de souplesse, vous pouvez annoter vos transactions. Vous retrouverez comment faire par en suivant [ce lien](https://wikicompta.indy.fr/fr/articles/1954201-comment-annoter-les-transactions-pour-une-comptabilite-personnalisee).

Vous pouvez ensuite rechercher vos annotations directement depuis la barre de recherche de l'onglet Transactions pour retrouver rapidement les revenus liés à vos différents activités.
