## Comment se rembourser ses indemnités kilométriques ?

Pour vous rembourser, il vous suffit de faire un virement du **compte professionnel vers votre compte personnel**.

Sur Indy, cette transaction doit être catégorisée en **Prélèvement personnel** ✅

## Quelques règles à connaître quand on opte pour les indemnités kilométriques

## Dépenses de véhicule payées avec le compte professionnel

Si vous avez payé des dépenses liées au véhicule avec votre **compte professionnel** (essence, assurance, entretien, etc.), elles doivent être catégorisées en **Prélèvement personnel ✅**

👉 Sinon, elles risquent d’être déduites deux fois (au réel + via l'indemnité kilométrique).

## Péages et parking

Comme ils ne sont pas inclus dans le barème kilométrique, vous pouvez les catégoriser en **Frais de déplacement ✅**
