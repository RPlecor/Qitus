# Modifier les paramètres de l'onglet Profil

L’onglet **Profil** dans vos paramètres vous permet de renseigner :

L'accès à l'onglet **Profil** se fait depuis l'onglet Paramètres comme ci-dessous 👇

Voyons ensemble les différentes rubriques et à quoi elles servent 🔍

## Informations personnelles

[Image]

Ce sont les informations renseignées dans votre profil (prénom, nom, adresse) qui vont figurer sur **vos factures d’abonnement Indy** 🧾

L’adresse postale renseignée doit être votre **adresse professionnelle.** En cas de changement d’adresse de local, il faudra bien la mettre à jour dans votre profil Indy 📍

L’adresse email indiquée dans votre profil Indy vous permet de recevoir les **réponses du chat et les différentes communications** Indy 💌

### ✍️ Comment modifier votre adresse email sur Indy ?

Vous pouvez modifier votre adresse email directement depuis la page Profil dans vos paramètres.

Une fois la modification effectuée, vous recevrez un **email de confirmation** à votre nouvelle adresse. Vous devrez cliquer sur le lien reçu pour valider le changement.

👉 Ce lien est valable pendant **15 minutes**.

## Informations comptables

[Image]

→ **Le numéro SIRET**

Vous pouvez modifier votre SIRET dans votre profil Indy.

Pour cela, vous devez bien supprimer toute la ligne et re-saisir les numéros ⌨️

→ **La raison sociale**

Si la raison sociale de votre entreprise change, vous pouvez également la modifier dans votre profil Indy. Lorsque vous l’avez modifié, il faut cliquer sur "**Enregistrer**" 💾

→ **Votre profession**

Vous ne **pouvez pas modifier votre profession** dans votre profil Indy. Celle-ci a été indiquée au moment de votre inscription. Si toutefois vous souhaitez la modifier, contactez-nous via la bulle de live-chat en bas à droite de votre écran 💬

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite 😉
