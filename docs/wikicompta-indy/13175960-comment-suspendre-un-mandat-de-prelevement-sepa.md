# Comment suspendre un mandat de prélèvement SEPA ?

## Suspendre un mandat _Core_

Si vous souhaitez suspendre un mandat de prélèvement _Core_ (standard), il vous suffit :

[Image]

_Capture d'écran d'exemple sur la version Web_

[Image]

_Capture d'écran d'exemple sur l'app mobile_

Une fenêtre vous demandera ensuite de confirmer votre demande.

[Image]

Une fois votre mandat suspendu, tous les **futurs prélèvements du créancier liés à ce mandat** seront automatiquement rejetés.

Si besoin, vous aurez toujours la possibilité de le réactiver en cliquant sur **Réactiver** dans la liste de vos mandats.

## Suspendre un mandat DGFIP

La suspension du mandat DGFIP doit passer par une intervention de notre partenaire Swan.

Afin de suspendre ce type de mandat, nous vous invitons à **nous transmettre votre demande via le live-chat** pour que nous puissions faire la démarche auprès de Swan.

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite 😉

À très vite !
