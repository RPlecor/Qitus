# Comment supprimer une carte virtuelle ?

Veuillez noter que cette action est irréversible, il ne vous sera pas possible de réactiver cette carte par la suite.

**Pour supprimer votre carte virtuelle :**

## Sur l'app mobile

1️⃣ Allez dans l'onglet "**Compte Pro**" et cliquez sur votre carte virtuelle dans la section "_Cartes_"

2️⃣ Cliquez sur le bouton "**Paramètres"** puis sur **"Supprimer la carte"**

[Image] [Image]

## Sur le web

1️⃣ Allez dans l'onglet "**Compte pro**" et cliquez sur votre carte virtuelle dans la section "_Cartes_"

2️⃣ Cliquez sur **"Supprimer ma carte"**
