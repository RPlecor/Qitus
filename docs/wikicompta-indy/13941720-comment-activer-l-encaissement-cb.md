# Comment activer l'encaissement CB ?

Vous pouvez demander l'activation de l'encaissement CB depuis l'app mobile ou la version web.

## Depuis votre app mobile Indy

5️⃣ Validez votre demande.

Celle-ci va ensuite être vérifiée par notre partenaire Swan sous un délai de **2 jours ouvrés**.

Vous recevrez une notification **par e-mail** et **par notification push** pour vous informer du changement de statut de votre demande.

Vous pouvez aussi vérifier le statut depuis l'onglet **Compte pro** > **Encaissements.**

## Depuis la version web d'Indy

3️⃣ Validez votre demande.

Celle-ci va ensuite être vérifiée par notre partenaire Swan sous un délai de **2 jours ouvrés**.

Vous recevrez une notification **par e-mail** (et **par notification push** sur votre téléphone si vous avez l'app mobile Indy) pour vous informer du changement de statut de votre demande.

Vous pouvez aussi vérifier le statut depuis l'onglet **Compte pro** > **Encaissements.**
