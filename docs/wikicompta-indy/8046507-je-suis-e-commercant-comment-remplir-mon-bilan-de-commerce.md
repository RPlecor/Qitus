# Je suis e-commerçant : comment remplir mon bilan de commerce ?

En tant qu'entreprise vendant des biens en ligne, vous êtes concerné par le bilan de commerce sur Indy ✅

Pour aller plus loin, découvrez les autres situations possibles :

Dans cet article, on vous explique étape par étape comment remplir correctement votre **bilan mensuel de commerce** 📝

## À quoi sert le bilan de commerce ?

Le bilan de commerce est une activité à compléter **chaque mois** dans l'onglet **À faire** sur Indy.

  
Il permet de récupérer toutes les informations liées à votre activité afin de tenir une comptabilité juste et complète.

Le bilan de commerce doit être activé manuellement dans **Paramètres > Options 👇**

[Image]

💬 Si vous ne voyez pas cette option, il est possible que votre compte ne soit pas correctement paramétré. N'hésitez pas à nous contacter via le **live-chat**, accessible depuis la bulle rose en bas à droite de votre écran.
