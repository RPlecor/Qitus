# Ma régularisation de TVA est supérieure à 4000€, que faire ?

Lorsque vous oubliez de déclarer des opérations dans une déclaration de TVA, la plupart du temps une régularisation est possible 😊

_Par exemple : vous avez oublié de déclarer des revenus le mois où ils ont été perçus, vous pouvez toujours les déclarer sur une déclaration de TVA postérieure, tant que cette régularisation est inférieure à 4000€ de TVA._

En revanche, lorsque le montant de votre régularisation est supérieur à 4000€ les impôts exigent une ou des déclarations de TVA rectificatives. Vous ne pouvez donc pas reporter la régularisation sur votre déclaration suivante.

C'est pour cette raison que nous bloquons l'étape **Régulariser la TVA** 👇

[Image]

## Comment faire pour débloquer l'étape ?

Pour débloquer votre étape il faut vérifier que cette régularisation est bien correcte dans un premier temps. Ensuite, si celle-ci est justifiée, il faudra obligatoirement faire une, voire, des déclarations de TVA rectificatives.

Voyons comment faire 👇

## 1️⃣ Vérifier le montant de la régularisation

Avant tout chose, pensez-bien à vérifier si le montant de régularisation proposé est cohérent avec votre comptabilité.

Vous pouvez également nous contacter afin qu'on vérifie cela avec vous 😊

## 2️⃣ Re-faire vos déclarations

Si le montant de régularisation proposé par Indy est correct, vous êtes dans l'obligation de re-faire vos déclarations de TVA.

👉 Vous pouvez nous contacter afin qu'on vous rouvre vos déclarations sur Indy.

Ensuite, vous aurez simplement à les re-déposer sur le site des impôts - comme si vous faisiez la déclaration pour la première fois.

💡 Les impôts vous feront payer la différence immédiatement.

Une fois cette étape complétée, vous pourrez patienter 3 jours avant de recommencer l'étape **Régulariser la TVA** 👇

[Image]

Nous pourrons récupérer les nouvelles déclarations de TVA, et donc débloquer votre étape 💪

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite 😊
