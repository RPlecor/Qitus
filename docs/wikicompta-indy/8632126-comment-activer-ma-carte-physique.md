# Comment activer ma carte physique ?

Vous pouvez activer votre nouvelle carte de 3 façons différentes :

👉 En effectuant un **paiement avec saisie du code PIN chez un commerçant,**

👉 En effectuant un **retrait d'espèces** à un distributeur automatique**,**

**👉 Directement depuis votre compte pro Indy** :

1️⃣ Dans l'app mobile ou la version web, rendez-vous sur l'onglet **Compte pro** et **sélectionnez la carte physique** dans la section _Cartes._

2️⃣ Cliquez sur **Activer** dans l'app mobile ou **Activer ma carte** dans la version web.

3️⃣ Saisissez **l'identifiant à dix chiffres situé au dos** de votre carte physique et cliquez sur **Activer**.

4️⃣ **Validez une authentification forte** sur la fenêtre Swan qui s'ouvre pour finaliser l'activation.

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite 😉

À très vite ! 👋
