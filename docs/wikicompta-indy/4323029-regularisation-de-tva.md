# Régularisation de TVA | Le wikicompta d'Indy

Lors de la clôture de votre exercice, il est essentiel de vérifier que toutes vos opérations ont bien été déclarées auprès des impôts grâce à une déclaration de TVA.

Vos déclarations de TVA doivent aussi être cohérentes avec votre bilan. 🤓

Chez Indy, nous vous proposons de vérifier tout cela via une étape de clôture.

## **📝 Comment récupérer vos déclarations de TVA ?**

La première étape consiste à récupérer les déclarations que vous avez déposées auprès des impôts pour les intégrer à votre bilan.

Pour cela, saisissez **vos identifiants professionnels** de votre accès à **impots.gouv** dans l'étape.

💡 Vos identifiants sont cryptés, **aucun membre de notre équipe n'y a accès.**

Une autre possibilité est d'ajouter vous même et manuellement vos déclarations de TVA en cliquant sur le bouton "**J'ajoute mes déclarations manuellement**".

[Image]

Une fois vos déclarations de TVA téléchargées et ajoutées dans l'étape, il est nécessaire de cliquer sur le bouton "**Valider mes déclarations**" pour accéder à la seconde phase de l’étape : le calcul et l’affichage des montants de régularisation.

[Image]

## **🚨 L'étape est bloquée, que faire ?**

Il existe plusieurs raisons pour lesquelles vous pourriez rencontrer le message d'erreur ci-dessous, mais dans tous les cas il n'y aura aucune inquiétude à avoir 👇

[Image]

1/ _Vous avez fait votre déclaration il y a moins de 3 jours sur le site des impôts_ 👉 dans ce cas, le seul remède sera l'attente. C'est le délai nécessaire pour que vos déclarations remontent sur l'étape ⌛

2/ _Vous effectuez vos déclarations sur le site de votre AGA_ 👉 envoyez-nous vos déclarations de TVA **au format PDF** et nous nous chargerons du reste 💪

3/ _Vous êtes devenu.e assujetti.e en cours d’année_ 👉 indiquez-nous votre date d'assujettissement sur le live-chat pour que nous débloquions l'étape.

⛔ Attention, si vous aviez ajouté de la TVA sur vos transactions avant la date d'assujettissement, celle-ci sera supprimée ❌

4/ _Vous n’avez pas de compte pro DGFIP ou n’avez pas adhéré au service “**Payer votre TVA**”_ 👉 suivez-nos indications sur l'article **[juste ici](https://wikicompta.indy.fr/fr/articles/5984674-adherer-aux-services-des-impots-pour-payer-votre-tva-en-ligne)** 👩‍🏫

Une fois ces déclarations récupérées de notre côté, nous pourrons les **comparer avec toutes les transactions de votre exercice** afin d'en vérifier la cohérence 🎉

Si vous avez suivi Indy toute l'année, c'est parfait : vous n'aurez aucune régularisation et vous pourrez valider votre bilan grâce à cette étape.

## **🤕** **J’ai une régularisation ?!**

L'étape de **Régularisation de TVA** peut mettre en lumière des écarts entre ce que vous avez déclaré et ce que qui aurait dû être déclaré.

Si vous avez rejoint Indy en cours d'année, ou si vous avez modifié des transactions après avoir déposé vos déclarations par exemple :

[Image]

👉 Les montants de cette régularisation seront détaillés sous l'étape.

En cliquant sur **Afficher le détail** vous aurez plus d'informations.

Cette **régularisation sera donc appliquée sur votre prochaine déclaration de TVA.**

Elle vous sera signalée par un encart 👇

[Image]

Si les montants régularisés vous semblent incohérents, pensez à **vérifier les taux de TVA** appliqués sur vos transactions. Cela peut être une source d'écart. 🤗

Enfin il se peut que vous ayez des **régulations de TVA liées à un autre exercice**.

☝️ Dans ce cas il est important de nous le signaler, afin que nous puissions intégrer ces transactions correctement sur votre déclaration.

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite ! 😊

À très vite 👋
