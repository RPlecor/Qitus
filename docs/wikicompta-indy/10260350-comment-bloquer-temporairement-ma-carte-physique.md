# Comment bloquer temporairement ma carte physique ?

Il est désormais possible de bloquer votre carte physique de façon temporaire sur le compte pro Indy.

Le blocage temporaire de la carte est une opération immédiate. Vous pourrez la réactiver ultérieurement en quelques clics, la supprimer ou faire opposition si nécessaire.

Le blocage entraîne la suspension de toutes les opérations de paiement de la carte, y compris les prélèvements d'abonnements éventuels.

_\*le blocage temporaire d'une carte n'est disponible que pour la carte physique_
