# Comment faire sa cessation d'activité sur Indy ?

En cessation d'activité, vous disposez de **60 jours pour transmettre votre déclaration 2035** aux impôts. Nous allons voir quelles sont les différentes étapes à suivre pour réaliser votre dernière 2035 sur Indy 🤓

## 👀 Pré-requis

Dans un premier temps, assurez-vous d'avoir déclaré (si ce n'est pas déjà fait) votre cessation d'activité sur le site de l'INPI.

⚠️ Vous avez 30 jours pour le faire.

Ils se chargent ensuite de prévenir tous les organismes obligatoires (INSEE, caisses de protections sociales...), afin que ceux-ci calculent vos dernières cotisations.

## **💻 Comment faire sur Indy ?**

Sur Indy, il faudra suivre différentes étapes afin de préparer votre 2035 de cessation.

**1ère étape : contactez-nous sur le live-chat**

Si les étapes de clôture ne sont pas ouvertes, contactez-nous sur le live-chat afin que nous puissions les ouvrir pour vous de façon anticipée 🤗  
​  
​**2ème étape : cédez toutes vos immobilisations**

Lorsque vous mettez fin à votre activité, votre patrimoine professionnel n’a plus de raison d’exister. **Toutes les immobilisations détenues doivent donc être cédées**.

Vous devez **enregistrer une cession pour chaque immobilisation.**  
Elles peuvent être **vendues, jetées (mise au rebut) ou récupérées dans le patrimoine privé.**  
Rendez-vous 👉 **[ici](https://wikicompta.indy.fr/fr/articles/5520021-ajouter-une-cession-d-immobilisation-sur-indy)** 👈 pour savoir comment enregistrer une cession d’immobilisation.

**3ème étape : suppression des transactions & validation du solde final**

Toute transaction après la date de votre cessation devra être supprimée.

En revanche, certaines de ces transactions doivent bien apparaitre dans votre comptabilité (prélèvement Urssaf, abonnement Indy...).

Il faut donc **[ajouter manuellement](https://wikicompta.indy.fr/fr/articles/2028081-comment-ajouter-une-transaction-manuellement)** ces transactions à la date de cessation d’activité pour qu’elles soient bien prises en compte dans votre comptabilité ✅

➡️ Prévenez-nous dès que ce sera fait, **afin que l'on supprime les transactions après votre date de cessation** (celles-ci seront prises en compte via les ajouts manuels que vous aurez fait au préalable ☝️).

➡️ Envoyez-nous également **vos derniers relevés de banque** au format PDF, afin que nous validions votre solde final dans l'étape **[10 - Vérifier les soldes bancaires](https://app.indy.fr/activities/d-2035-activity/initialize-financial-accounts)** 📮  
​  
​_​Il vous restera à indiquer le solde final de votre caisse dans l'étape 11 - Caisse._

**4ème étape : renseignez votre date de cessation d'activité**

À savoir : la date officielle de votre cessation d’activité (qui correspond bien à la date de fin d’exercice, et non la date à laquelle vous faites votre déclaration) et la raison.

[Image]

Pour le reste, il vous suffira de faire toutes vos étapes de clôture de façon habituelle.

Vous pourrez ensuite envoyer la 2035 aux impôts ou à votre AGA si vous en avez une 😊

## 👋 **Désabonnement Indy**

À noter : en cessation, il se peut que certains organismes vous contraignent d’établir une **déclaration 2035 rectificative.** C'est le cas, par exemple, lorsque vous avez des régularisations auprès de l'Urssaf.

  
Nous vous recommandons de **rester abonné jusqu’à la fin du processus de cessation d’activité**, afin de pouvoir **modifier votre déclaration si besoin** 🙏

  
Pour vous simplifier la vie :

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite ! 😊
