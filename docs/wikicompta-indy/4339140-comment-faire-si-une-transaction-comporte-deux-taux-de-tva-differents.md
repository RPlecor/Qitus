# Comment faire si une transaction comporte deux taux de TVA différents ?

Lorsqu'une transaction comporte **plusieurs taux de TVA**, il faut **subdiviser la transaction** pour appliquer ces différents taux. 🤓

  
​_Par exemple, votre dépense en télécom comporte une part dont le taux de TVA est le taux normal de 20% et une part au taux réduit de 2,10%._

Il faut donc subdiviser en deux lignes votre transaction.
