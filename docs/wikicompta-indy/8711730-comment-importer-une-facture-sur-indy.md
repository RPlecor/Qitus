# Comment importer une facture sur Indy ?

Vous avez déjà émis une facture en dehors d’Indy ?  
Pas de souci : vous pouvez facilement l’importer via un formulaire simplifié, qu’elle soit liée à l’exercice en cours ou à un exercice antérieur.

## Accéder au formulaire d'import

Depuis l’onglet **Facturation**, cliquez sur **\+ Nouveau**, puis sélectionnez **Importer une facture** 👇

[Image]

Un formulaire s’ouvre.  
Vous pouvez glisser-déposer votre fichier PDF en haut de page pour activer l’extraction automatique des informations 👇

[Image]

Une fois tous les champs complétés, cliquez sur **Importer la facture**.
