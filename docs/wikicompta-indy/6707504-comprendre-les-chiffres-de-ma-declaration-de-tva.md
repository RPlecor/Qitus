# Comprendre les chiffres de ma déclaration de TVA

Cet article vous aidera à comprendre les données utilisées pour éditer votre déclaration de TVA sur Indy, et comment modifier la déclaration de TVA si besoin.

Pour mieux comprendre les impacts, regardez votre déclaration de TVA disponible dans l'onglet **[Documents](https://app.indy.fr/documents)** 👇

## Modifier la déclaration de TVA

Une fois que la déclaration de TVA est clôturée, il est toujours possible de modifier les caractéristiques des opérations comptabilisées dans cette même déclaration.

Dans ce cas, la modification effectuée ne viendra pas impacter la déclaration de TVA concernée puisqu'elle est déjà clôturée.

## Les chiffres en gras dans la déclaration de TVA

Depuis l'onglet **[Documents](https://app.indy.fr/documents)**, vous avez la possibilité de visualiser vos déclarations de TVA.

Il est possible de voir comment chaque montant est calculé par Indy **en cliquant sur les montants en gras** 👇

[Image]

Une fenêtre s'ouvre alors avec la liste des transactions comptabilisées sur la ligne en question.

Pour aller encore plus loin dans l'analyse des chiffres, vous avez la possibilité de **cliquer maintenant sur le petit oeil rose** présent à côté de chaque opération qui impacte la déclaration de TVA 👇

[Image]
