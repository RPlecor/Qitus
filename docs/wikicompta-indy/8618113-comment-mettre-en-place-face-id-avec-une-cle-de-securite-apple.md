# Comment mettre en place Face ID avec une clé de sécurité Apple ?

Si vous avez choisi d'activer les clés de sécurité sous iOS 17, il est possible lors de votre onboarding que vous n'arriviez pas à activer Face ID immédiatement.

[Image]

🔐Dans le cas où vous utiliseriez une clé de sécurité externe, reportez-vous à [cette documentation Apple.](https://support.apple.com/fr-fr/HT213154)  
​  
Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite 😉
