# Comment masquer mon solde dans l'app mobile ?

Pour masquer les montants affichés dans votre app mobile Indy :

1️⃣ Aller dans l'onglet **Compte pro** ou l'onglet **Transactions,**

2️⃣ Appuyer sur **l'icône 👁️** à côté du solde.

Les montants suivants seront alors masqués :

👉 **Pour les réafficher**, appuyez à nouveau sur l'icône 👁️.

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite 😉

À très vite !
