# Comment mettre en place un prélèvement SEPA standard ?

Pour mettre en place un mandat de prélèvement SEPA Core, il vous suffit de **fournir votre RIB Indy au créancier pour être prélevé** par ce dernier.

Vous n'avez pas besoin de nous envoyer le mandat ou d'effectuer une action sur votre compte pro pour autoriser les prélèvements. **Le mandat s'ajoutera automatiquement** sur votre compte lorsque le premier prélèvement du créancier se présentera.

​​

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite 😉
