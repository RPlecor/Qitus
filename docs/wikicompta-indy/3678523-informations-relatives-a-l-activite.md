# Informations relatives à l'activité | Le wikicompta d'Indy

Cette étape vous permet de saisir les informations générales en lien avec votre activité.

Ces informations seront reprises dans votre 2035 et permettront à l'administration fiscale d'identifier votre entreprise lorsqu'elle recevra votre liasse.

Il est donc **essentiel** de vérifier les informations saisies, autrement la 2035 pourrait être refusée par les impôts ❌

Voyons en détail quelles sont les informations demandées 😊

[Image]

Par défaut, nous pré-remplissons le champ avec l'entreprise individuelle.

Si vous êtes en EIRL, il suffira de mettre à jour la forme juridique 🤓

Il s'agit de la date à laquelle vous avez **commencé votre activité libérale**.

Si vous êtes immatriculé·e depuis le 15/01/2005, c'est cette date qu'il faudra renseigner.

[Image]

La ROF est un numéro d'identification définissant les obligations fiscales à souscrire par une entreprise. Si vous faites une 2035, votre ROF est probablement BNC1.

  
En cas de doute, vous pouvez demander votre memento fiscal aux impôts 😊

Il s'agit d'un code à 5 caractères qui vous est attribué lors de la création de votre activité. Il existe un code NAF/APE par branche d'activité.

Si vous avez oublié le vôtre, n'hésitez pas à vous rendre 👉 **[ici](https://avis-situation-sirene.insee.fr/)**.

[Image]

Cette information est à saisir uniquement si vous êtes PAMC.

Autrement, vous pouvez laisser le champ rempli ainsi : "Je ne suis pas médecin"

Si vous travaillez avec votre conjoint, c'est le moment de le renseigner 😉

[Image]

Pensez bien à indiquer l'AGA dont vous êtes adhérent·e pour l'exercice fiscal en cours de clôture. Si vous n'aviez pas d'AGA, indiquez **AUCUNE**.

Indiquez ici la 1ère année d'adhésion à l'AGA renseignée précédemment.

[Image]

Indiquez ici le type de local que vous avez et comment vous l'occupez.

⚠️ Il s'agit bien du local professionnel, à distinguer de l'habitation personnelle.

Quelques cas d'usage :

👉 En collaboration, indiquez **usage gratuit**.

👉 Associé·e d'une SCM ou SCI dont vous occupez le local, indiquez **usage gratuit**.

👉 En remplacement, indiquez **absence de local professionnel**.

Si vous cochez oui, nous vous demanderons l'adresse du second cabinet.
