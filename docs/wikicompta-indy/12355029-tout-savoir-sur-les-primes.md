# Tout savoir sur les primes

## Qu’est-ce qu’une prime ?

Une prime est une somme versée par l’entreprise en complément du salaire de base.  
Elle peut récompenser un effort particulier (prime d’assiduité, de rendement, etc.), marquer un événement (prime de fin d’année, de vacances, etc.) ou encore encourager certaines pratiques (prime transport, télétravail, etc.) 💰

👉 Selon leur nature, certaines primes sont **soumises aux cotisations sociales** (comme le salaire), tandis que d’autres en sont **exonérées**.

## Exemples de primes courantes

|     |     |
| --- | --- |
| **Type de prime** | **Cotisations sociales** |
| Prime de 13ème mois | ✅ Soumise |
| Prime d’assiduité / ancienneté / rendement | ✅ Soumise |
| Prime de vacances | ✅ Soumise |
| Prime de partage de la valeur (ex-Macron) | ❌ Exonérée |
| Bons d’achat ou chèques cadeaux | ❌ Exonérée |
| Forfait mobilité durable | ❌ Exonérée |
| Indemnité repas (prime panier) | ❌ Exonérée |
| Prime de télétravail (≤ 2,70 €/jour) | ❌ Exonérée |

## Mandataires sociaux

Les dirigeants de sociétés (président de SAS/SASU, gérant de SARL) ne peuvent pas percevoir de primes… sauf s’ils ont aussi un **contrat de travail salarié**.

## Accords d’intéressement

Un accord d’intéressement ne peut être mis en place que si l’entreprise emploie au moins un salarié autre que le dirigeant.  
👉 Un gérant de SASU sans salarié ne peut donc pas mettre en place ce type de dispositif.
