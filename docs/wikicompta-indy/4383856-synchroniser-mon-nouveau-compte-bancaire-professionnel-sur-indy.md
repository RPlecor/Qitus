# Synchroniser mon nouveau compte bancaire professionnel sur Indy

Voici le lien pour **[accéder à la page Banques](https://app.georges.tech/compte/banques)** 👈 depuis laquelle vous allez pouvoir ajouter jusqu'à 99 comptes bancaires différents à votre espace Indy.

## Ajouter un compte depuis une banque déjà connectée

Depuis la page **Paramètres > Banques**, il vous suffit de cliquer sur **\+ Ajouter un compte ou une carte** puis de sélectionner le compte concerné dans la fenêtre qui s'ouvre 👇

[Image]

[Image]

##   
Ajouter un compte depuis une autre banque

Si le compte que vous souhaitez ajouter est dans une banque différente du premier compte, il faut alors **connecter cette nouvelle banque** 🤓

Toujours depuis l'onglet **Paramètres > Banques > + Ajouter un compte ou une carte**, cliquez sur le bouton **\+ Ajouter une nouvelle banque 👇**

[Image]

Comme lors de votre arrivée chez Indy, vous devez saisir vos identifiants pour que Indy puisse synchroniser cette nouvelle banque 👇

[Image]

Ensuite vous n'aurez plus qu'à sélectionner le compte que vous souhaitez synchroniser avec Indy 🎉

[Image]

## Je change de banque, comment faire ?

Lorsque vous changez de banque vous avez souvent une période de transition où vous avez l'ancien et le nouveau compte.

Sur Indy ce n'est pas un problème vous pouvez **ajouter les deux comptes** dans l'onglet **[Banques](https://app.indy.fr/compte/banques)**, en suivant les deux processus indiqués ci-dessus 🤓

## Ajouter une carte à débit différée

Pour ajouter une carte à débit différé, le fonctionnement est le même.

Si elle est associée à la banque déjà synchronisée, la carte vous sera proposée au même titre qu'un autre compte bancaire (cf. première étape de cet article)

## Ajouter un compte non-professionnel, est-ce possible ?

Si vous souhaitez ajouter les dépenses ponctuellement effectuées pour votre entreprise depuis un compte personnel, **[cet article](http://wikicompta.georges.tech/fr/articles/4291280-integrer-une-note-de-frais)** vous explique la marche à suivre 🐾

## Ajouter un compte épargne

En tant qu'entreprise soumise à l'IS, les comptes épargne relatifs à votre entreprise doivent également figurer dans votre bilan en fin d'exercice.

## **Ajouter un compte en devises étrangères**

Indy n’est pas adapté pour synchroniser **les comptes bancaires en devises étrangères** 👉 seuls **les** **comptes en euros** peuvent être synchronisés.

Enfin, si le compte que vous souhaitez synchroniser n'apparaît pas ou pour toute question à ce sujet, n'hésitez pas à revenir vers nous via le live-chat en bas à droite 😉
