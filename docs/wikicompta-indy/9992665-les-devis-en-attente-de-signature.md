# Les devis en attente de signature

Dès lors que vous éditez un devis sur Indy, il passe automatiquement **En attente de signature** dans la rubrique **Devis** de votre onglet **[Facturation](https://app.indy.fr/facturation/factures)** 👇

[Image]

En cliquant sur les trois petits points verticaux, plusieurs options sont proposées 👇

[Image]

Faisons le tour ensemble des différentes options 🤓

Votre devis s'affiche alors.

Vous retrouverez toutes les autres options en haut à droite, ainsi que la possibilité de le marquer directement comme signé 👇

[Image]

Cette fonctionnalité vous permet de demander à vos clients de signer électroniquement les devis édités.

En cliquant sur ce bouton, vous allez pouvoir :

Dès que vous aurez cliqué sur le bouton `Envoyer l’email,` votre client signataire du devis recevra un email lui permettant de consulter puis signer électroniquement le devis 🎊

Une fois le devis signé par votre client, vous recevrez vous-même un email vous confirmant que son devis a été signé !

Votre client accepte le devis.

Vous pouvez alors le marquer comme signé. Automatiquement, une fenêtre s'ouvre pour vous proposer de créer la facture associée 👇

[Image]

Retrouvez ensuite ce devis dans la partie Signés en bas de la rubrique **Devis** 👇

[Image]

Votre client refuse le devis.

Vous pouvez alors le marquer comme décliné. Retrouvez ensuite ce devis dans la partie **Déclinés** en bas de la rubrique **Devis** 👇

[Image]

Tant que votre devis est toujours en attente de signature, vous avez la possibilité de le modifier.

La fenêtre de création de devis s'ouvrira de nouveau. Apportez vos modifications et re-valider le devis 🤓

Cette option vous permet de télécharger votre devis en format **.pdf**.

Pour gagner du temps, vous avez la possibilité de dupliquer votre devis.

La fenêtre de création de devis s'ouvrira alors de nouveau. Apportez vos modifications et valider le nouveau devis.

Contrairement aux factures, il est possible de supprimer un devis en attente de paiement.

Pour cela, cliquez sur **Supprimer**. Une fenêtre s'ouvre pour valider la suppression du devis 👇

[Image]

Cette option vous permet soit de visualiser les informations de votre client, soit d'apporter des modifications sur la fiche de votre client.
