# Comment gérer le mélange compte professionnel/compte personnel ?

Pour les professions libérales, nous savons que la séparation entre compte personnel et professionnel n'est pas toujours très bien marquée 😊

Voyons comment retranscrire ça dans votre comptabilité.

## **💳 Transaction personnelle effectuée avec le compte professionnel**

Vous pouvez sortir certaines transactions de votre comptabilité, si elles n'ont pas de rapport avec votre activité professionnelle. Pour cela, vous pouvez les mettre dans l'une de ces catégories 👇

##   
👩‍🏫 **Transaction professionnelle effectuée avec le compte personnel**

Bien qu'à caractère professionnel, la transaction n'a pas été effectuée sur votre compte synchronisé sur Indy ; peut-être n'aviez-vous pas votre carte professionnelle avec vous lors du paiement.

Il vous faut donc ajouter manuellement une transaction depuis l'onglet **[Transactions](https://app.indy.fr/transactions)**.

Je vous invite à prendre connaissance de la procédure juste 👉 **[ici](https://wikicompta.indy.fr/fr/articles/2028081-comment-ajouter-une-transaction-manuellement)**.  
💸 Pour les paiements en espèces, lisez **[notre article](https://wikicompta.indy.fr/fr/articles/2162255-comment-gerer-ma-caisse)**.

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite ! 😊
