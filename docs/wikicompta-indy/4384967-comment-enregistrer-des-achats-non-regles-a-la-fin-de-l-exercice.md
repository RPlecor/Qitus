# Comment enregistrer des achats non réglés à la fin de l'exercice ?

Il peut arriver qu’au moment de votre clôture, vous n’ayez pas encore réglé certains achats effectués ; on parle alors de **dettes fournisseurs** 🤓

Comme Indy se base sur vos transactions bancaires, il n’a pas connaissance de ces dettes.

Pas d’inquiétude, une étape de clôture est dédiée pour vous permettre de lister les achats non-réglés en fin d’exercice et ainsi les prendre en compte dans vos documents comptables 📑

## Enregistrer une dette fournisseur

Pour enregistrer une dette fournisseur, rendez-vous dans l'onglet **À faire** **\> Clôture comptable et fiscale👇**

[Image]

Sélectionnez **l'étape : Recenser les dettes fournisseurs 👇**

[Image]

Cliquez sur le symbole ➕ **Ajouter** et ajoutez une ligne pour chaque dette fournisseur.

Une fenêtre s'ouvre alors, vous permettant de détailler pour chaque prestation :

Une fois que tous les champs sont remplis, cliquez sur le bouton Valider ✅

Votre dette apparaît bien dans le tableau _Détails des dettes fournisseurs_.

💡 Il faut ajouter une ligne pour chaque dette.

## Les dettes antérieures

Si votre bilan précédent a été initialisé sur Indy ou si vous avez déjà fait une clôture à nos côtés, vos précédentes dettes fournisseurs ont été gardées en mémoire et sont reportées automatiquement par Indy.

Dans cette étape, il est alors nécessaire lors de chaque clôture, de mettre à jour le statut de ces dettes fournisseurs avant d'en ajouter de nouvelles pour votre exercice en cours de clôture.

Pour cela, cliquez sur le ✏️ de la dette en cours que vous souhaitez mettre à jour 👇

[Image]

Une fenêtre s'ouvre pour vous permettre d'indiquer le montant réglé auprès de votre fournisseur au cours de l’exercice en cours de clôture 😀

[Image]

N'oubliez pas de cliquer sur **Valider** et le tour est joué **🎉**

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite 🤓

À très vite ! ☀️
