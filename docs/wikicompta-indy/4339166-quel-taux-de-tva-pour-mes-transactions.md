# Quel taux de TVA pour mes transactions ?

Il est **essentiel de vérifier les taux de TVA** appliqués sur vos transactions.

Grâce à cela, Indy peut préparer vos déclarations de TVA avec les **bonnes informations**.

Pour connaître le taux de TVA à appliquer, il faut toujours **se baser sur la facture** d'achat concernée.

## 1️⃣ Si la facture a un taux de TVA français

Il faut alors appliquer **le même taux que sur la facture sur Indy ✅**

## 2️⃣ S'il n'y a pas de taux de TVA français sur la facture

Le traitement dépend des éléments présents sur la facture. Pour appliquer l’autoliquidation de TVA sur une facture **éditée par une entreprise en UE**, plusieurs conditions doivent être réunies :

👉 Si tous ces éléments sont présents, il faut appliquer un **taux UE** sur la transaction ✅

👉 Si non, plusieurs possibilités :

Dans ce cas, il faut appliquer le **taux de 0%** sur la transaction ✅

Vous pouvez ajouter et modifier le taux de TVA directement sur vos opérations, depuis l'onglet **[Transactions](https://app.indy.fr/transactions)** en cliquant sur le **%** 👇

[Image]

Indy utilisera cette information pour préparer votre déclaration 🤓

## Taux spécifique à 8,5%

Si votre activité est immatriculée dans un Département d'Outre-Mer (DOM), le taux normal est de 8,5%.

Vous pouvez nous contacter pour mettre à jour ce taux sur votre compte.

## Plusieurs taux de TVA sur une facture

Si une facture comporte plusieurs taux de TVA, il vous faut subdiviser la transaction concernée.
