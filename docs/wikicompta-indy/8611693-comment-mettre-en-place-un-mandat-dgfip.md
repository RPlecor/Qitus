# Comment mettre en place un mandat DGFIP ?

Le mandat de la DGFIP est un **mandat de prélèvement B2B** (également appelé interentreprises). À la différence d'un mandat Core (standard), le mandat B2B **doit être ajouté manuellement sur votre compte pro Indy pour autoriser les futurs prélèvements**.

Une fois que vous avez généré et signé votre mandat DGFIP pour être prélevé sur votre compte pro Indy, il n'est **pas nécessaire de nous envoyer le document** pour autoriser les prélèvements. Il vous suffit de configurer le mandat sur votre compte en suivant les instructions ci-dessous.

## Sur la version web

4️⃣ **Validez** **l'authentification forte** pour finaliser l'ajout du mandat DGFIP.

🎉 Votre mandat est **désormais actif** ! Vous pouvez le retrouver dans l'onglet **Compte pro** > **Mandats de prélèvement**.

## Sur l'app mobile Indy

4️⃣ **Validez** **l'authentification forte** pour finaliser l'ajout du mandat DGFIP.

🎉 Votre mandat est **désormais actif** ! Vous pouvez le retrouver dans l'onglet **Compte Pro** > **Mandats**.
