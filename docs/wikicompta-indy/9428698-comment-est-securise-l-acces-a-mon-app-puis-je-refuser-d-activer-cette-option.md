# Comment est sécurisé l'accès à mon app, puis-je refuser d'activer cette option ?

🔏 Nous vous recommandons vivement d'activer l'authentification au démarrage de l'application Indy en utilisant votre empreinte digitale, la reconnaissance faciale ou bien un code de verrouillage.

## Si vous souhaitez activer l'authentification

👉 Cliquez sur **J'active l'authentification**

[Image]

###   
Pour activer le verrouillage de l'écran sur votre smartphone 👇

## Si vous ne souhaitez pas activer l'authentification

⚠️ En cas de vol de votre appareil, votre app Indy sera visible.

👉 Libre à vous de faire le choix de ne pas activer l'authentification  
👉 Si c'est le cas, un message vous proposera régulièrement d'activer la fonctionnalité

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite 😉

À très vite ! 👋
