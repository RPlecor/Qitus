# Pourquoi mon paiement par carte est "en cours de traitement" ?

Les cartes Mastercard proposées avec votre compte pro Indy sont des **cartes à autorisation systématique**. Cela signifie que pour chaque paiement par carte, le système vérifie que vous disposez des fonds nécessaires dans votre solde.

👉 Si c'est le cas, une empreinte de carte bloque temporairement les fonds correspondants dans votre solde,

👉 Sinon la transaction est refusée.

L'empreinte de carte s'affiche sur votre compte Indy avec le statut "**En cours de traitement**" 👇

[Image]

Les fonds sont alors **bloqués dans votre solde disponible** : vous ne pouvez plus les utiliser pour d'autres paiements, mais ils n'ont **pas encore été débités** par le commerçant.

## Empreinte bloquée alors que le paiement a échoué ou a été annulé

Il arrive qu'une empreinte reste temporairement bloquée sur votre compte malgré un échec ou une annulation. Cela se produit lorsque Swan n'en a pas encore été informé par la banque du commerçant.

Elle sera libérée une fois que Swan aura été notifié de l'abandon, ou après que le délai maximum de validité soit écoulé.

Pour que Swan puisse débloquer manuellement l'empreinte d'un paiement annulé, vous devrez nous transmettre impérativement **un ou des documents du commerçant attestant que l'empreinte a été libérée 👇**
