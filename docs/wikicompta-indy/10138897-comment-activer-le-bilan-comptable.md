# Comment activer le bilan comptable ?

## Qu’est-ce qu’un bilan comptable et un compte de résultat ?

Un bilan comptable est un document qui indique la situation financière d’une entreprise à un moment précis, généralement à la fin de l’exercice comptable. Ce document a deux grandes parties :

Le compte de résultat est un document qui présente les revenus et dépenses d'une entreprise sur un exercice fiscal. Contrairement au bilan, le compte de résultat se concentre sur les flux financiers pour mesurer le résultat (bénéfice ou perte) de l'activité de l’entreprise.

## C’est mon premier exercice fiscal

Au moment de vous inscrire sur Indy, vous avez indiqué que vous souhaitez réaliser avec l’outil le premier exercice fiscal de la SCI. Vous n’avez rien de plus à faire 👌

[Image]

👉 L’option pour le bilan est activée automatiquement pour un premier exercice.

Cette option vous permet de **tenir une comptabilité au plus juste avec tout l’historique comptable de votre SCI**, même ce qui n’impacte pas votre déclaration fiscale comme le montant d’achat d’un bien.

Si vous ne souhaitez pas faire de bilan, vous avez la possibilité de contacter le support pour demander la désactivation de cette option.

## Ce n’est pas mon premier exercice fiscal

**Il sera nécessaire d’avoir déjà établi des bilans sur vos exercices antérieurs.** En effet, les bilans reprennent certaines informations d’exercice en exercice depuis la création de votre SCI : il est donc indispensable d’avoir déjà édité un bilan sur l’exercice précédent.

Cela nous permettra d’importer ces données dans l’outil et de les combiner avec les informations de votre exercice en cours pour en générer un nouveau lors de votre première clôture avec Indy 🤓

👉 Vous avez un bilan antérieur à intégrer ? Contactez-nous sur le support client via la bulle rose en bas à droite afin de nous envoyer le bilan de votre dernier exercice.

**Notre équipe dédiée aux SCI pourra l’étudier en détail afin de s’assurer qu’il est bien compatible avec Indy ou de vous proposer des ajustements !**

S’il est bien compatible, ils pourront activer l’option sur votre compte Indy et vous accompagner pour intégrer le nécessaire.

L’activité **Initialisation de comptabilité** aura des étapes supplémentaires : ce sont justement ces étapes qui permettront d’importer sur Indy ces données antérieures !

## **Comment vérifier si l’option est activée ?**

Pour cela, il vous suffit de vous rendre dans l’onglet **[Documents](https://app.indy.fr/documents/comptabilite)**. Si vous pouvez voir le bilan et le compte de résultat, c’est que l’option est activée 👇

[Image]

Et inversement, si vous ne les voyez pas, c’est que vous ne bénéficiez pas de cette option 👇

[Image]

En effet, les amortissements ne sont pas déductibles pour une SCI soumise à l’impôt sur le revenu. C’est un mécanisme obligatoire pour les SCI soumises à l’impôt sur les sociétés qui, elles, peuvent les déduire fiscalement.

Si vous avez un bilan à intégrer ou des questions supplémentaires, n'hésitez pas à revenir vers nous via le support client 😉
