# L'ajout de justificatifs sur Indy

👉 Archivez toutes vos pièces comptables directement sur Indy pour justifier vos dépenses en cas de contrôle.

🎉 Vous pouvez **ajouter** et **stocker** vos justificatifs sur Indy en attendant de les lier à vos transactions.

☝️ Sachez que l'ensemble des justificatifs que vous ajouterez sur Indy auront une valeur probante ✅. Pour en savoir plus à ce sujet, c'est par ici.

## **1️⃣ L'ajout d'un justificatif depuis le bouton Ajouter**

Une fois votre achat fait, rendez-vous sur votre compte Indy.

Dans l'onglet **Transactions**, cliquez sur **Ajouter > Justificatif** 👇

[Image]

📱 **Depuis votre smartphone** : vous pouvez **prendre immédiatement une photo** ou **récupérer un fichier** dans votre galerie.

💻 **Depuis votre ordinateur** : vous pouvez **Sélectionner un fichier** ou effectuer un **drag&drop**.

Étape 1 : Le fichier est analysé

Étape 2 : Dans le bloc à droite, Indy remonte toutes les informations qu'il a trouvé sur votre justificatif. Il faudra alors vérifier les données 👇

[Image]

Étape 3 : Vous validez le justificatif.

Ici il y aura plusieurs cas d'usage 👀

## **2️⃣ L'ajout d'un justificatif sur Indy depuis une transaction**

**Première possibilité : ajout d'un justificatif via le trombone**

Lorsque vous avez plusieurs justificatifs à ajouter sur une transaction, vous pouvez le faire directement depuis celle-ci.

Dans l'onglet **[Transactions](https://app.indy.fr/transactions)** de votre compte, à côté de chaque transaction, vous pouvez voir un trombone 📎 sur la gauche du libellé 👇

[Image]

**Cliquez** sur le trombone, et ajoutez les justificatifs.

Ce sont jusqu'à 10 justificatifs qui peuvent être intégrés au **format PNG, JPG ou PDF**.

📱 **Depuis votre smartphone** : vous pouvez **prendre immédiatement une photo** ou **récupérer un fichier** dans votre galerie.

💻 **Depuis votre ordinateur** : vous pouvez **Sélectionner un fichier** ou effectuer un **drag&drop**.

Une fois les documents ajoutés, vous pouvez cliquer sur la croix en haut à droite.

👉 Vos justificatifs ont été conservés sur Indy 👇

[Image]

**Deuxième possibilité : la suggestion automatique**

Si vous avez stocké votre justificatif sur Indy en attendant que la transaction bancaire soit présente sur votre compte, ou si vous avez édité votre facture depuis l'onglet **[Facturation](https://app.indy.fr/facturation/factures)**, alors Indy pourra vous proposer de lier le justificatif à la transaction automatiquement !

Pour cela, Indy vous fait une proposition directement sous la transaction. Cliquez alors sur le bouton **Associer** 👇

[Image]

  
💡 **Grâce à l'appairage automatique quotidien**, les suggestions manuelles d'association apparaissent peu souvent pour les transactions synchronisées : Indy s'en charge pour vous en arrière-plan.  
Vous verrez toutefois toujours une demande de validation au moment de l'ajout d'un nouveau justificatif 🤓

##   
4️⃣ **Télécharger tous mes justificatifs**

Si vous avez besoin de **télécharger tous vos justificatifs**, rendez-vous dans votre onglet **Documents > Comptabilité** et cliquez sur **Export Comptable**👇  
​  
​

[Image]

  
Vous devrez alors choisir la plage de date souhaitée, puis cocher le bouton **Intégrer les pièces justificatives** 👇  
​

[Image]

Ensuite, cliquez sur **Recevoir par email 📩**  
​  
Vous recevrez un fichier .**zip** avec l'ensemble de vos pièces justificatives dans les minutes qui suivent.
