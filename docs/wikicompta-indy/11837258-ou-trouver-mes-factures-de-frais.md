# Où trouver mes factures de frais ?

Vous pouvez retrouver vos **factures de frais Swan** depuis l'app mobile ou la version web d'Indy.

## Depuis l'app mobile d'Indy

## Depuis la version web d'Indy

## Lier une facture de frais à la transaction correspondante

1️⃣ Rendez-vous dans l'onglet **Transactions**.

2️⃣ Ouvrez la **transaction correspondant aux frais**.

3️⃣ Cliquez sur **Ajouter un justificatif** (ou l’icône 📎) et importez la facture téléchargée.

4️⃣ Enregistrez et le tour est joué 🎉

La facture sera alors visible dans la fiche de transaction et listée dans vos justificatifs !

Pour toutes questions supplémentaires, n'hésitez pas à revenir vers nous via le live-chat en bas à droite 😉
