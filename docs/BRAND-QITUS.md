# Qitus — Brand Brief

## 1. Le nom

**Qitus** — prononcé "ki-tus"
Néologisme dérivé de "quitus" (validation de bonne gestion), sans le U.
Domaine : **qitus.io**

### Pourquoi ce nom

- Filiation directe avec le concept de quitus : décharge, validation, conformité
- 5 lettres, 2 syllabes, mémorable
- Mot inventé = zéro collision marque, SEO immédiat
- Prononçable en FR, EN, et langues latines sans déformation
- Le Q initial est graphiquement distinctif (courbe + queue)

### Pitch d'épellation

> "Q comme Quitus, sans le U — Q, I, T, U, S."

## 3. Positionnement

### Ce que Qitus est

Un **notaire digital** de la comptabilité. Sobre, précis, fiable.
L'outil qui transforme des données bancaires brutes en comptes irréprochables.

### Ce que Qitus n'est pas

- Pas un outil "fun" ou gamifié (pas de confettis, pas de mascotte)
- Pas un concurrent de l'expert-comptable (un allié)
- Pas un outil généraliste (spécialiste compta TPE/PME)

### Contre-positionnement

| Concurrent | Qitus se différencie par... |
|------------|----------------------------|
| Pennylane | Plus sobre, plus expert, moins "startup cool" |
| Indy | Plus structuré, moins simpliste, segment TPE/PME vs micro |
| Tiime | Plus technologique (IA), moins artisanal |

## 4. Univers visuel

### Palette

|------|---------|-----|
| Primaire | Bleu marine profond | `#1B2A4A` |
| Secondaire | Blanc cassé / ivoire | `#F7F5F0` |
| Accent | Or mat / bronze | `#C9A84C` |
| Utilitaire | Gris ardoise | `#4A5568` |
| Succès | Vert profond | `#2D6A4F` |
| Alerte | Rouge brique | `#C53030` |

**Règle** : max 3 couleurs par écran. L'or mat est réservé aux états de validation.

### Typographie

| Usage | Direction | Exemples |
|-------|-----------|----------|
| Logo / Titres | Serif moderne, traits fins | Editorial New, Freight Display, Cormorant Garamond |
| Corps UI | Sans-serif géométrique | Inter, Satoshi, General Sans |
| Données / chiffres | Monospace / tabular | JetBrains Mono, IBM Plex Mono |

### Éléments graphiques signature

- **Le sceau** : cercle + symbole de validation — favicon, badge, watermark documents
- **Lignes fines horizontales** : évoquent les lignes d'un registre comptable
- **Espaces blancs généreux** : le vide = maîtrise
- **Or mat sur validation** : exercice clôturé, FEC conforme, signature expert
